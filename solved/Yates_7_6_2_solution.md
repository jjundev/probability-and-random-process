# 7.6.2 풀이

> 출처: Probability and Stochastic Processes (Yates & Goodman), Probability_and_Stochastic_Processes_Problems/problems_png/7.6.2.png (사용자 첨부 스크린샷), 1순위 근거 Lecture08(상관계수 ρ 정의 p.6); 결합정규 PDF·조건부평균 선형성은 정리노트·STT 미수록 → Yates §7.6 표준 결과로 보강, 풀이일 2026-06-12

## 문제 원문

**7.6.2** X and Y are jointly Gaussian random variables with E[X] = E[Y] = 0 and Var[X] = Var[Y] = 1. Furthermore, E[Y|X] = X/2. Find f_{X,Y}(x,y).

---

## 1. 문제 정리 (Setup)

**주어진 것**: X, Y는 **결합(jointly) 가우시안**이고
- E[X] = E[Y] = 0
- Var[X] = Var[Y] = 1
- E[Y|X] = X/2

**구할 것**: 결합 PDF f_{X,Y}(x,y).

## 2. 무엇을 묻고 왜 이 도구인가 (개념 동기)

결정적 사실 하나: **결합 가우시안은 (μ_X, μ_Y, σ_X², σ_Y², ρ) 다섯 모수로 완전히 결정된다.** 평균·분산 4개는 이미 주어졌으니(0,0,1,1), **딱 하나 ρ만 찾으면 PDF가 끝**난다.

그 ρ는 주어진 조건부평균 E[Y|X]=X/2에서 나온다. 결합 가우시안에서 조건부평균은 **x에 대한 직선**이고 기울기가 ρ·σ_Y/σ_X (Yates §7.6 표준 결과):

$$\mathbb{E}[Y\mid X=x] = \mu_Y + \rho\,\frac{\sigma_Y}{\sigma_X}\,(x-\mu_X)$$

표준화돼 있으니(μ=0, σ=1) 기울기 = ρ. ρ가 "정규화된 covariance, 선형 의존성의 표준 지표"라는 정의는 정리노트에 있다: ρ = cov(X,Y)/√(Var X·Var Y) (Lecture08, p.6).

> ⬆ **보강 명시**: 핵심 다리 ρ의 정의는 Lecture08 p.6. 그러나 "결합 가우시안의 조건부평균이 선형"이라는 성질과 아래 이변량 정규 PDF 공식 자체는 정리노트·STT(단변량 z-table만 언급)에 없어 **Yates §7.6 표준 정의로 보강**.

## 3. 핵심 통찰 (Aha)

> 표준화된 결합 가우시안에서 **E[Y|X=x]의 기울기가 곧 ρ**다. 그러니 E[Y|X]=X/2는 ρ=1/2를 직접 알려주고, 나머지 모수는 다 주어졌으니 PDF가 확정된다.

## 4. 풀이 (Worked solution)

**(1단계) 표준 결합 가우시안 PDF 틀.** μ_X=μ_Y=0, σ_X=σ_Y=1, 상관계수 ρ인 이변량 정규 (Yates §7.6):

$$f_{X,Y}(x,y) = \frac{1}{2\pi\sqrt{1-\rho^{2}}}\exp\!\left\{-\frac{x^{2}-2\rho xy + y^{2}}{2\,(1-\rho^{2})}\right\}$$

**(2단계) ρ 결정.** 조건부평균 공식에 표준화 값 대입 → E[Y|X=x] = ρx. 주어진 조건과 비교:

$$\rho\,x = \frac{x}{2}\ \ \text{(모든 x)}\ \Longrightarrow\ \rho = \frac{1}{2}$$

**(3단계) ρ=1/2를 PDF에 대입.** 부품 계산:
- 1 − ρ² = 1 − 1/4 = 3/4
- 2π√(1−ρ²) = 2π·(√3/2) = π√3
- 2(1−ρ²) = 3/2
- 교차항 −2ρxy = −xy → 분자 = x² − xy + y²

$$\boxed{\,f_{X,Y}(x,y) = \frac{1}{\pi\sqrt{3}}\exp\!\left\{-\frac{2}{3}\bigl(x^{2}-xy+y^{2}\bigr)\right\}\,}$$

**정의역은 ℝ² 전체** — 가우시안은 전 평면에서 양수이므로, 유계台 분포와 달리 otherwise = 0인 영역이 **따로 없다**(전 (x,y)에서 위 식). 동치로 1/(π√3) = √3/(3π).

## 5. 검산·직관 (Sanity check)

- **ρ 범위**: ρ = 1/2 ∈ [−1, 1] ✓ (Lecture08, p.6의 −1≤ρ≤1).
- **주변분포 복원**: y를 적분하면 X ~ N(0,1)이 나와야 함 — 표준 이변량 정규의 주변은 항상 표준정규 ✓ (E[X]=0, Var[X]=1과 일치).
- **조건부분산 유효성**: Var[Y|X=x] = σ_Y²(1−ρ²) = 1·(3/4) = 3/4 > 0 ✓ (양수라야 정상; ρ=±1이면 0이 돼 퇴화).
- **대칭성**: 분자 x²−xy+y²는 x↔y 교환 대칭 → E[X|Y]=Y/2도 성립해야 하는데 표준화·동일 ρ이라 실제로 그렇다 ✓.
- **정규화**: 1/(π√3) ≈ 0.184; 이변량 정규의 표준 정규화 상수 1/(2πσ_Xσ_Y√(1−ρ²))와 일치 ✓.

## 6. 한 줄 요약

> 결합 가우시안은 (평균·분산·ρ)로 완전 결정 — 평균·분산은 주어졌고 조건부평균의 기울기 1/2가 곧 ρ이므로, ρ=1/2를 표준 이변량 정규 PDF에 꽂으면 f = (1/π√3)·exp(−⅔(x²−xy+y²)).
