# Yates 5.7.3 풀이

> 출처: Probability and Stochastic Processes (Yates & Goodman), 1순위 근거 Lecture07 (Covariance), 풀이일 2026-06-09

## 문제 원문

A random ECE sophomore has height X (rounded to the nearest foot) and GPA Y (rounded to the nearest integer). These random variables have joint PMF:

| P_{X,Y}(x,y) | y=1  | y=2  | y=3  | y=4  |
|---------------|------|------|------|------|
| x=5           | 0.05 | 0.10 | 0.20 | 0.05 |
| x=6           | 0.10 | 0.10 | 0.30 | 0.10 |

(otherwise = 0)

Find E[X+Y] and Var[X+Y].

---

### 1. 문제 정리 (Setup)

구할 것: E[X+Y], Var[X+Y]

---

### 2. 무엇을 묻고 왜 이 도구인가

E[X+Y]는 선형성으로 분해, Var[X+Y]는 **공분산 항이 포함**된 공식을 써야 한다. X와 Y가 독립인지 먼저 확인. (Lecture07, p.23–25)

$$\text{Var}[X+Y] = \text{Var}[X] + \text{Var}[Y] + 2\,\text{cov}(X,Y)$$

---

### 3. 핵심 통찰 (Aha)

X와 Y가 독립이 아니므로 공분산 항을 빠뜨리면 안 된다 — cov(X,Y) = E[XY] − E[X]E[Y]를 별도로 계산해야 한다.

---

### 4. 풀이 (Worked solution)

**주변 PMF**

P_X(5) = 0.05+0.10+0.20+0.05 = 0.40, P_X(6) = 0.60

P_Y(1) = 0.15, P_Y(2) = 0.20, P_Y(3) = 0.50, P_Y(4) = 0.15 (otherwise = 0)

**독립 확인**

P(X=5, Y=1) = 0.05 ≠ P_X(5)·P_Y(1) = 0.40×0.15 = 0.06 → 독립 아님, 공분산 필요.

**E[X], E[Y]**

E[X] = 5(0.40) + 6(0.60) = 5.60

E[Y] = 1(0.15) + 2(0.20) + 3(0.50) + 4(0.15) = 2.65

$$E[X+Y] = 5.60 + 2.65 = 8.25$$

**Var[X], Var[Y]**

E[X²] = 25(0.40) + 36(0.60) = 31.60 → Var[X] = 31.60 − 31.36 = 0.24

E[Y²] = 1(0.15) + 4(0.20) + 9(0.50) + 16(0.15) = 7.85 → Var[Y] = 7.85 − 7.0225 = 0.8275

**Cov(X,Y)** (Lecture07, p.23)

$$E[XY] = 5(1)(0.05)+5(2)(0.10)+5(3)(0.20)+5(4)(0.05)+6(1)(0.10)+6(2)(0.10)+6(3)(0.30)+6(4)(0.10) = 14.85$$

cov(X,Y) = 14.85 − (5.60)(2.65) = 14.85 − 14.84 = 0.01

**Var[X+Y]**

$$\text{Var}[X+Y] = 0.24 + 0.8275 + 2(0.01) = 1.0875$$

---

### 5. 검산·직관 (Sanity check)

직접 계산: E[(X+Y)²] = 69.15, Var[X+Y] = 69.15 − 8.25² = 1.0875 ✓

---

### 6. 한 줄 요약

> E[X+Y]는 선형성으로 분해되지만, Var[X+Y]는 독립이 아닌 한 반드시 2·cov(X,Y) 항을 더해야 한다.
