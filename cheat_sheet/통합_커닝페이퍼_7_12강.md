# 확률·추론·추정 통합 커닝페이퍼 (7~12강)

> 구성: **Part I 확률(7~9강)** → **Part II 통계적 추론 입문(10강)** → **Part III 추정(11~12강)**
> 흐름: 확률 기초(공분산·조건부기대·점근) → 추론(베이즈 룰·분류) → 추정(MAP/LMS/LLMS·ML). 추론·추정은 Part I의 cov/var·조건부기대를 그대로 사용.

# Part I. 확률 (7~9강)

## A. Derived distribution (7강) — Y=g(X)의 분포
원칙: **항상 CDF 먼저 구하고 미분.**
```
1) F_Y(y) = P(g(X) ≤ y)
2) f_Y(y) = dF_Y/dy
```

이산:
```
p_Y(y) = Σ_{x: g(x)=y} p_X(x),   otherwise 0
```

선형 Y=aX+b (a≠0, 연속):
```
f_Y(y) = (1/|a|)·f_X((y−b)/a),   otherwise 0
```

자주 나오는 꼴:
```
Y = X²:    f_Y(y) = (1/(2√y))·[f_X(√y)+f_X(−√y)]   (y≥0), otherwise 0
정규 선형보존: X~N(μ,σ²) → aX+b ~ N(aμ+b, a²σ²)
```

## B. 두 변수 함수 Z=g(X,Y) (7강)
```
max:  P(max(X,Y) ≤ z) = P(X≤z)P(Y≤z)        (둘 다 ≤ z)
min:  CCDF가 편함  P(min > w) = P(X>w)P(Y>w)  (둘 다 > w)
```

## C. Convolution: Z=X+Y, X⊥Y (7강)
```
이산:   p_Z(z) = Σ_x p_X(x)·p_Y(z−x)
연속:   f_Z(z) = ∫ f_X(x)·f_Y(z−x) dx
정규합: N(μx,σx²)+N(μy,σy²) = N(μx+μy, σx²+σy²)   (독립)
```

## D. Covariance / Correlation (7~8강)
```
cov(X,Y) = E[XY] − E[X]E[Y]
ρ(X,Y) = cov(X,Y) / (σ_X·σ_Y),     −1 ≤ ρ ≤ 1

cov(X,X) = var(X)
cov(aX+b, Y) = a·cov(X,Y)            (상수 b 무관, a는 1제곱)
cov(X, Y+Z) = cov(X,Y) + cov(X,Z)
var(X+Y) = var(X)+var(Y)+2cov(X,Y)
var(ΣXi) = Σvar(Xi) + Σ_{i≠j} cov(Xi,Xj)
```
- 독립 ⇒ uncorrelated(cov=0). **역은 거짓.**
- |ρ|=1 ⇔ 완전 선형관계 (X−μx = c(Y−μy))

Cauchy–Schwarz:
```
(E[XY])² ≤ E[X²]·E[Y²]
```

## E. 조건부 기댓값/분산 (8강)
```
E[X|Y], var[X|Y] 는 "Y의 함수 = 랜덤변수"
E[X|Y=y], var[X|Y=y] 는 "숫자"

반복기대 (iterated):   E[ E[X|Y] ] = E[X]
전분산 (LTV):          var[X] = E[var(X|Y)] + var(E[X|Y])
```
- LTV 두 항 모두 ≥ 0. (그룹내 변동 + 그룹간 변동)

## F. Random sum (8강) — Y = X1+…+X_N, N도 랜덤
조건: Xi iid (평균 μ, 분산 var[Xi]), Xi ⊥ N
```
E[Y]   = E[N]·μ
var[Y] = E[N]·var[Xi] + μ²·var[N]    (둘째 항 빼먹지 말 것)
```

## G. 점근 거동 (9강) — Sn=X1+…+Xn, Xi iid (μ, σ²)
합과 표본평균:
```
E[Sn]=nμ,   var[Sn]=nσ²
Mn=Sn/n,   E[Mn]=μ,   var[Mn]=σ²/n
```

WLLN (약한 큰 수의 법칙):
```
n→∞ 면 var[Mn]→0  →  Mn → μ  (확률 수렴)
스케일 1/n  →  결정론적 세계
```

CLT (중심극한정리):
```
Zn = (Sn − nμ)/(σ√n)  →  N(0,1)  (분포 수렴)
Xi 분포가 무엇이든 성립.   스케일 1/√n  →  정규분포 세계
```
표준화 일반식: Y = (X − μ)/σ → 평균0, 분산1.

# Part II. 통계적 추론 입문 (10강)

## 0. 문제 분류 (먼저 판단)
```
unknown이 유한개 후보 중 하나   → Hypothesis testing (오결정 확률 최소화)
unknown이 연속(무한) 값 중 하나 → Estimation (참값에 가까운 값 찾기)

θ를 랜덤으로 봄 + prior 있음     → Bayesian
θ를 모르는 상수로 봄 (prior X)   → Classical
```
용어 대응(본질 동일): hypothesis testing/estimation = classification/regression = detection/estimation

## 1. 핵심 용어
```
Likelihood   p(x; θ) = P(data | θ)   ← θ 가정 하에 데이터가 나올 확률
Prior        p_Θ(θ) / f_Θ(θ)         ← 관측 전 Θ의 분포
Posterior    p(θ|x) / f(θ|x)          ← 관측 후 Θ의 분포 = Bayesian의 완전한 답
```
- **Likelihood ≠ 확률**: θ 바꿔가며 계산한 값들의 합은 1 아님 (정규화 안 됨)
- Bayesian의 답은 **분포 전체**(posterior). 숫자 하나 필요할 때만 point estimate(MAP/LMS, 11강)

## 2. 두 방식의 결정 규칙
```
Classical (ML):  likelihood 큰 쪽 선택   argmax_θ p(x; θ)
Bayesian:        posterior 큰 쪽 선택    argmax_θ p(θ|x)
```
- **prior가 uniform이면 두 결론 일치** (Bayesian ⊃ Classical)

## 3. 베이즈 룰 (★핵심) — 4가지 버전
```
p(θ|x) = prior · likelihood / evidence
```
규칙: Θ 이산→ p_Θ, 분모는 Σ_θ' / Θ 연속→ f_Θ, 분모는 ∫dθ'.  X 이산→ p, 연속→ f.
```
[Θ이산 X이산]  p(θ|x) = p_Θ(θ)·p(x|θ) / p_X(x),   p_X(x)=Σ_θ' p_Θ(θ')p(x|θ')
[Θ이산 X연속]  p(θ|x) = p_Θ(θ)·f(x|θ) / f_X(x),   f_X(x)=Σ_θ' p_Θ(θ')f(x|θ')
[Θ연속 X연속]  f(θ|x) = f_Θ(θ)·f(x|θ) / f_X(x),   f_X(x)=∫ f_Θ(θ')f(x|θ')dθ'
[Θ연속 X이산]  f(θ|x) = f_Θ(θ)·p(x|θ) / p_X(x),   p_X(x)=∫ f_Θ(θ')p(x|θ')dθ'
```
- 분모(evidence)는 정규화 상수 → argmax 비교엔 무시 가능, 실제 값 필요할 때만 계산(전확률 정리)

## 4. 대표 예제 — 편향 동전 HHH
설정: θ∈{1/4, 3/4}, 3번 던져 모두 앞면(HHH). Bernoulli: P(H)=θ, P(T)=1−θ, otherwise 0
```
Classical:  P(HHH|3/4)=(3/4)³=27/64,  P(HHH|1/4)=(1/4)³=1/64  → 27배 → θ=3/4
Bayesian (prior 1/2씩):
   evidence P(HHH)=(27/64)(1/2)+(1/64)(1/2)=7/32
   posterior P(3/4|HHH)=27/28,  P(1/4|HHH)=1/28  → θ=3/4
```

## 5. Beta prior (확률값 θ∈[0,1] 추론 시 자주 씀)
```
f_Θ(θ) = θ^(α−1)(1−θ)^(β−1) / B(α,β)   (0≤θ≤1), otherwise 0
평균 = α/(α+β),   Beta(1,1) = Uniform[0,1]
```
- **Conjugate prior**: m회 중 앞면 k회 관측 → posterior = Beta(α+k, β+(m−k))
- 사전·사후가 같은 꼴 → recursive(순차) 추론에 유리

# Part III. 추정 (11~12강)

## 0. 세계관 구분 (먼저 판단)
```
Θ가 랜덤 + prior 있음    → Bayesian:  MAP / LMS / LLMS
θ가 모르는 상수 (prior X) → Classical:  ML
```

## 1. Bayesian: posterior 먼저
```
posterior  p(θ|x) ∝ p(x|θ)·p(θ)        (분모 p(x)는 정규화 상수, argmax엔 무관)
```

| 추정 | 공식 | posterior에서 |
|---|---|---|
| MAP | θ̂ = argmax_θ p(θ\|x) | 봉우리(mode) |
| LMS | θ̂ = E[Θ\|X=x] | 평균(mean) |

- MAP 실전: p(x|θ)p(θ)를 θ로 미분=0 (또는 log 취하고 미분). 분모 무시.
- LMS 실전: posterior 구해서 평균. 관측 없으면 θ̂=E[Θ].

## 2. LLMS (선형 LMS) ★최빈출
```
θ̂_L = E[Θ] + (cov(Θ,X)/var(X))·(X − E[X])
     = E[Θ] + ρ·(σ_Θ/σ_X)·(X − E[X])

ρ = cov(Θ,X)/(σ_Θ·σ_X)
a = cov(Θ,X)/var(X),   b = E[Θ] − a·E[X]
```
- **필요한 것: 평균·분산·공분산만** (분포 전체 불필요)
- X = E[X]면 보정항 0 → θ̂_L = E[Θ]

## 3. MSE 값 (셋 비교)
```
관측 없음:  MSE = var[Θ]
LMS:       MSE = E[var(Θ|X)] = var[Θ] − var(E[Θ|X])   (전역 최소)
LLMS:      MSE = (1 − ρ²)·var[Θ]

항상:  MSE_LMS ≤ MSE_LLMS ≤ var[Θ]
ρ=0 → 데이터 쓸모X (MSE=var[Θ]),   |ρ|=1 → MSE=0
```

### ★ 평균 MSE vs 조건부 MSE — 혼동 주의 (바깥 E[·] 유무로 구분)
```
전체(평균) MSE :  E[var(Θ|X)] = Σ_x var(Θ|X=x)·p_X(x)   ← 모든 관측 평균
조건부 MSE     :  var(Θ|X=x₀) = E[(Θ−θ̂_M(x₀))² | X=x₀]   ← 특정 관측 하나
```
- "given X=x₀" 처럼 **조건이 붙고 바깥 평균이 없으면** → 그냥 var(Θ|X=x₀) (평균 X)
- 위 박스의 `E[var(Θ|X)]`는 그 조건부 분산을 **모든 x에 대해 평균낸 것** (= 전역 최소 MSE)
- 둘은 일반적으로 값이 다름. 문제에 "E[ · | X=x₀]" 만 있으면 조건부 한 값.
- 예 (PSP 12.2.1 (h)): e*(−3)=E[(X−x̂_M(−3))²|Y=−3]=var(X|Y=−3)=2/9 (특정 y, 평균 아님)

### LLMS MSE — 두 형태는 같은 식 (시험지 표지에 둘 다 등장)
```
e*_L = (1 − ρ²)·var[Θ] = var[Θ] − cov(Θ,X)²/var(X)
```
- 유도: (1−ρ²)var[Θ] = var[Θ] − ρ²var[Θ], 그리고 ρ² = cov²/(var[Θ]·var(X))
  → ρ²var[Θ] = cov²/var(X)  (var[Θ] 약분) → 두 번째 형태.
- **쓰는 법**: ρ를 안 구했으면 둘째 형태(var[Θ] − cov²/var(X))가 빠름. ρ를 이미 구했으면 첫 형태.
- 예 (PSP 12.2.1): var X=2/3, cov=7/6, var Y=5
  → e*_L = 2/3 − (7/6)²/5 = 71/180 ≈ 0.394
  → 검산: ρ²=cov²/(varX·varY)=49/120, (1−49/120)(2/3)=71/180 ✓

## 4. Classical: ML estimator
```
θ̂_ML = argmax_θ p_X(x₁,…,xₙ; θ)      (likelihood 최대화)

독립이면 log우도 합 최대화:
θ̂_ML = argmax_θ  Σ log p_Xi(xi; θ)    → θ로 미분=0
```
- 실전: 우도 곱 → log → θ 미분=0 → θ̂ 풀기
- **MAP ↔ ML**: prior가 uniform(균등)이면 MAP = ML

## 5. 자주 쓰는 결과/도구
```
LTV (반복기대):  E[X] = E[E[X|Θ]]
Total variance:  var(X) = E[var(X|Θ)] + var(E[X|Θ])
cov(Θ,X) = E[ΘX] − E[Θ]E[X]
E[(Θ−c)²] 최소 ⇔ c = E[Θ]
```

### 결합PDF 모멘트 — 주변밀도 안 만들고 직접 적분 (LOTUS)
```
E[X]  = ∬ x·f_{X,Y}(x,y) dx dy       (E[Y],E[XY] 도 동일 패턴)
E[XY] = ∬ x·y·f_{X,Y}(x,y) dx dy
```
- f_X 만들어 푸는 것 = 이중적분에서 **한 변수를 먼저 적분한 중간 단계**일 뿐 (Fubini). 별개 방법 아님, 결과 항상 같음.
  - x 먼저: ∫x(∫f dy)dx  ↔  y 먼저: ∫x(∫f dx)dy → 둘 다 같은 E[X]
- **적분 쉬운 순서를 골라라.** 삼각영역(0≤x≤y≤1)이면 보통 안쪽 x: 0→y 가 깔끔.
- 연속 LLMS는 결국 이 적분으로 E[X],E[Y],var,cov 4개만 뽑아 §2 공식에 대입.
- 예 (PSP 12.2.4, f=2(y+x) on 0≤x≤y≤1): E[X]=5/12, E[Y]=3/4, varY=3/80, cov=1/48
  → a*=cov/varY=5/9, b*=E[X]−a*E[Y]=0 → X̂_L(Y)=(5/9)Y
