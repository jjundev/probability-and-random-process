# 확률 (7~9강) 커닝페이퍼

## A. Derived distribution (7강) — Y=g(X)의 분포
원칙: **항상 CDF 먼저 구하고 미분.**
```
1) F_Y(y) = P(g(X) ≤ y)
2) f_Y(y) = dF_Y/dy
```

이산:
```
p_Y(y) = Σ_{x: g(x)=y} p_X(x),   otherwise 0
```

선형 Y=aX+b (a≠0, 연속):
```
f_Y(y) = (1/|a|)·f_X((y−b)/a),   otherwise 0
```

자주 나오는 꼴:
```
Y = X²:    f_Y(y) = (1/(2√y))·[f_X(√y)+f_X(−√y)]   (y≥0), otherwise 0
정규 선형보존: X~N(μ,σ²) → aX+b ~ N(aμ+b, a²σ²)
```

## B. 두 변수 함수 Z=g(X,Y) (7강)
```
max:  P(max(X,Y) ≤ z) = P(X≤z)P(Y≤z)        (둘 다 ≤ z)
min:  CCDF가 편함  P(min > w) = P(X>w)P(Y>w)  (둘 다 > w)
```

## C. Convolution: Z=X+Y, X⊥Y (7강)
```
이산:   p_Z(z) = Σ_x p_X(x)·p_Y(z−x)
연속:   f_Z(z) = ∫ f_X(x)·f_Y(z−x) dx
정규합: N(μx,σx²)+N(μy,σy²) = N(μx+μy, σx²+σy²)   (독립)
```

## D. Covariance / Correlation (7~8강)
```
cov(X,Y) = E[XY] − E[X]E[Y]
ρ(X,Y) = cov(X,Y) / (σ_X·σ_Y),     −1 ≤ ρ ≤ 1

cov(X,X) = var(X)
cov(aX+b, Y) = a·cov(X,Y)            (상수 b 무관, a는 1제곱)
cov(X, Y+Z) = cov(X,Y) + cov(X,Z)
var(X+Y) = var(X)+var(Y)+2cov(X,Y)
var(ΣXi) = Σvar(Xi) + Σ_{i≠j} cov(Xi,Xj)
```
- 독립 ⇒ uncorrelated(cov=0). **역은 거짓.**
- |ρ|=1 ⇔ 완전 선형관계 (X−μx = c(Y−μy))

Cauchy–Schwarz:
```
(E[XY])² ≤ E[X²]·E[Y²]
```

## E. 조건부 기댓값/분산 (8강)
```
E[X|Y], var[X|Y] 는 "Y의 함수 = 랜덤변수"
E[X|Y=y], var[X|Y=y] 는 "숫자"

반복기대 (iterated):   E[ E[X|Y] ] = E[X]
전분산 (LTV):          var[X] = E[var(X|Y)] + var(E[X|Y])
```
- LTV 두 항 모두 ≥ 0. (그룹내 변동 + 그룹간 변동)

## F. Random sum (8강) — Y = X1+…+X_N, N도 랜덤
조건: Xi iid (평균 μ, 분산 var[Xi]), Xi ⊥ N
```
E[Y]   = E[N]·μ
var[Y] = E[N]·var[Xi] + μ²·var[N]    (둘째 항 빼먹지 말 것)
```

## G. 점근 거동 (9강) — Sn=X1+…+Xn, Xi iid (μ, σ²)
합과 표본평균:
```
E[Sn]=nμ,   var[Sn]=nσ²
Mn=Sn/n,   E[Mn]=μ,   var[Mn]=σ²/n
```

WLLN (약한 큰 수의 법칙):
```
n→∞ 면 var[Mn]→0  →  Mn → μ  (확률 수렴)
스케일 1/n  →  결정론적 세계
```

CLT (중심극한정리):
```
Zn = (Sn − nμ)/(σ√n)  →  N(0,1)  (분포 수렴)
Xi 분포가 무엇이든 성립.   스케일 1/√n  →  정규분포 세계
```
표준화 일반식: Y = (X − μ)/σ → 평균0, 분산1.
