# 추정 (11~12강) 커닝페이퍼

## 0. 세계관 구분 (먼저 판단)
```
Θ가 랜덤 + prior 있음    → Bayesian:  MAP / LMS / LLMS
θ가 모르는 상수 (prior X) → Classical:  ML
```

## 1. Bayesian: posterior 먼저
```
posterior  p(θ|x) ∝ p(x|θ)·p(θ)        (분모 p(x)는 정규화 상수, argmax엔 무관)
```

| 추정 | 공식 | posterior에서 |
|---|---|---|
| MAP | θ̂ = argmax_θ p(θ|x) | 봉우리(mode) |
| LMS | θ̂ = E[Θ|X=x] | 평균(mean) |

- MAP 실전: p(x|θ)p(θ)를 θ로 미분=0 (또는 log 취하고 미분). 분모 무시.
- LMS 실전: posterior 구해서 평균. 관측 없으면 θ̂=E[Θ].

## 2. LLMS (선형 LMS) ★최빈출
```
θ̂_L = E[Θ] + (cov(Θ,X)/var(X))·(X − E[X])
     = E[Θ] + ρ·(σ_Θ/σ_X)·(X − E[X])

ρ = cov(Θ,X)/(σ_Θ·σ_X)
a = cov(Θ,X)/var(X),   b = E[Θ] − a·E[X]
```
- **필요한 것: 평균·분산·공분산만** (분포 전체 불필요)
- X = E[X]면 보정항 0 → θ̂_L = E[Θ]

## 3. MSE 값 (셋 비교)
```
관측 없음:  MSE = var[Θ]
LMS:       MSE = E[var(Θ|X)] = var[Θ] − var(E[Θ|X])   (전역 최소)
LLMS:      MSE = (1 − ρ²)·var[Θ]

항상:  MSE_LMS ≤ MSE_LLMS ≤ var[Θ]
ρ=0 → 데이터 쓸모X (MSE=var[Θ]),   |ρ|=1 → MSE=0
```

### ★ 평균 MSE vs 조건부 MSE — 혼동 주의 (바깥 E[·] 유무로 구분)
```
전체(평균) MSE :  E[var(Θ|X)] = Σ_x var(Θ|X=x)·p_X(x)   ← 모든 관측 평균
조건부 MSE     :  var(Θ|X=x₀) = E[(Θ−θ̂_M(x₀))² | X=x₀]   ← 특정 관측 하나
```
- "given X=x₀" 처럼 **조건이 붙고 바깥 평균이 없으면** → 그냥 var(Θ|X=x₀) (평균 X)
- 위 박스의 `E[var(Θ|X)]`는 그 조건부 분산을 **모든 x에 대해 평균낸 것** (= 전역 최소 MSE)
- 둘은 일반적으로 값이 다름. 문제에 "E[ · | X=x₀]" 만 있으면 조건부 한 값.
- 예 (PSP 12.2.1 (h)): e*(−3)=E[(X−x̂_M(−3))²|Y=−3]=var(X|Y=−3)=2/9 (특정 y, 평균 아님)

### LLMS MSE — 두 형태는 같은 식 (시험지 표지에 둘 다 등장)
```
e*_L = (1 − ρ²)·var[Θ] = var[Θ] − cov(Θ,X)²/var(X)
```
- 유도: (1−ρ²)var[Θ] = var[Θ] − ρ²var[Θ], 그리고 ρ² = cov²/(var[Θ]·var(X))
  → ρ²var[Θ] = cov²/var(X)  (var[Θ] 약분) → 두 번째 형태.
- **쓰는 법**: ρ를 안 구했으면 둘째 형태(var[Θ] − cov²/var(X))가 빠름. ρ를 이미 구했으면 첫 형태.
- 예 (PSP 12.2.1): var X=2/3, cov=7/6, var Y=5
  → e*_L = 2/3 − (7/6)²/5 = 71/180 ≈ 0.394
  → 검산: ρ²=cov²/(varX·varY)=49/120, (1−49/120)(2/3)=71/180 ✓

## 4. Classical: ML estimator
```
θ̂_ML = argmax_θ p_X(x₁,…,xₙ; θ)      (likelihood 최대화)

독립이면 log우도 합 최대화:
θ̂_ML = argmax_θ  Σ log p_Xi(xi; θ)    → θ로 미분=0
```
- 실전: 우도 곱 → log → θ 미분=0 → θ̂ 풀기
- **MAP ↔ ML**: prior가 uniform(균등)이면 MAP = ML

## 5. 자주 쓰는 결과/도구
```
LTV (반복기대):  E[X] = E[E[X|Θ]]
Total variance:  var(X) = E[var(X|Θ)] + var(E[X|Θ])
cov(Θ,X) = E[ΘX] − E[Θ]E[X]
E[(Θ−c)²] 최소 ⇔ c = E[Θ]
```

### 결합PDF 모멘트 — 주변밀도 안 만들고 직접 적분 (LOTUS)
```
E[X]  = ∬ x·f_{X,Y}(x,y) dx dy       (E[Y],E[XY] 도 동일 패턴)
E[XY] = ∬ x·y·f_{X,Y}(x,y) dx dy
```
- f_X 만들어 푸는 것 = 이중적분에서 **한 변수를 먼저 적분한 중간 단계**일 뿐 (Fubini). 별개 방법 아님, 결과 항상 같음.
  - x 먼저: ∫x(∫f dy)dx  ↔  y 먼저: ∫x(∫f dx)dy → 둘 다 같은 E[X]
- **적분 쉬운 순서를 골라라.** 삼각영역(0≤x≤y≤1)이면 보통 안쪽 x: 0→y 가 깔끔.
- 연속 LLMS는 결국 이 적분으로 E[X],E[Y],var,cov 4개만 뽑아 §2 공식에 대입.
- 예 (PSP 12.2.4, f=2(y+x) on 0≤x≤y≤1): E[X]=5/12, E[Y]=3/4, varY=3/80, cov=1/48
  → a*=cov/varY=5/9, b*=E[X]−a*E[Y]=0 → X̂_L(Y)=(5/9)Y
