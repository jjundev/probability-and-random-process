# 통계적 추론 입문 (10강) 커닝페이퍼

## 0. 문제 분류 (먼저 판단)
```
unknown이 유한개 후보 중 하나   → Hypothesis testing (오결정 확률 최소화)
unknown이 연속(무한) 값 중 하나 → Estimation (참값에 가까운 값 찾기)

θ를 랜덤으로 봄 + prior 있음     → Bayesian
θ를 모르는 상수로 봄 (prior X)   → Classical
```
용어 대응(본질 동일): hypothesis testing/estimation = classification/regression = detection/estimation

## 1. 핵심 용어
```
Likelihood   p(x; θ) = P(data | θ)   ← θ 가정 하에 데이터가 나올 확률
Prior        p_Θ(θ) / f_Θ(θ)         ← 관측 전 Θ의 분포
Posterior    p(θ|x) / f(θ|x)          ← 관측 후 Θ의 분포 = Bayesian의 완전한 답
```
- **Likelihood ≠ 확률**: θ 바꿔가며 계산한 값들의 합은 1 아님 (정규화 안 됨)
- Bayesian의 답은 **분포 전체**(posterior). 숫자 하나 필요할 때만 point estimate(MAP/LMS, 11강)

## 2. 두 방식의 결정 규칙
```
Classical (ML):  likelihood 큰 쪽 선택   argmax_θ p(x; θ)
Bayesian:        posterior 큰 쪽 선택    argmax_θ p(θ|x)
```
- **prior가 uniform이면 두 결론 일치** (Bayesian ⊃ Classical)

## 3. 베이즈 룰 (★핵심) — 4가지 버전
```
p(θ|x) = prior · likelihood / evidence
```
규칙: Θ 이산→ p_Θ, 분모는 Σ_θ' / Θ 연속→ f_Θ, 분모는 ∫dθ'.  X 이산→ p, 연속→ f.
```
[Θ이산 X이산]  p(θ|x) = p_Θ(θ)·p(x|θ) / p_X(x),   p_X(x)=Σ_θ' p_Θ(θ')p(x|θ')
[Θ이산 X연속]  p(θ|x) = p_Θ(θ)·f(x|θ) / f_X(x),   f_X(x)=Σ_θ' p_Θ(θ')f(x|θ')
[Θ연속 X연속]  f(θ|x) = f_Θ(θ)·f(x|θ) / f_X(x),   f_X(x)=∫ f_Θ(θ')f(x|θ')dθ'
[Θ연속 X이산]  f(θ|x) = f_Θ(θ)·p(x|θ) / p_X(x),   p_X(x)=∫ f_Θ(θ')p(x|θ')dθ'
```
- 분모(evidence)는 정규화 상수 → argmax 비교엔 무시 가능, 실제 값 필요할 때만 계산(전확률 정리)

## 4. 대표 예제 — 편향 동전 HHH
설정: θ∈{1/4, 3/4}, 3번 던져 모두 앞면(HHH). Bernoulli: P(H)=θ, P(T)=1−θ, otherwise 0
```
Classical:  P(HHH|3/4)=(3/4)³=27/64,  P(HHH|1/4)=(1/4)³=1/64  → 27배 → θ=3/4
Bayesian (prior 1/2씩):
   evidence P(HHH)=(27/64)(1/2)+(1/64)(1/2)=7/32
   posterior P(3/4|HHH)=27/28,  P(1/4|HHH)=1/28  → θ=3/4
```

## 5. Beta prior (확률값 θ∈[0,1] 추론 시 자주 씀)
```
f_Θ(θ) = θ^(α−1)(1−θ)^(β−1) / B(α,β)   (0≤θ≤1), otherwise 0
평균 = α/(α+β),   Beta(1,1) = Uniform[0,1]
```
- **Conjugate prior**: m회 중 앞면 k회 관측 → posterior = Beta(α+k, β+(m−k))
- 사전·사후가 같은 꼴 → recursive(순차) 추론에 유리
