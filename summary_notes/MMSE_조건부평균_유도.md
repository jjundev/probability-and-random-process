# MMSE 추정값 = (조건부) 평균 — 왜?

> 근거: Lecture11 (LMS 최적성, slide 13–14, "θ̂_LMS = E[Θ|X=x]"). 정리일 2026-06-14.
> 한 줄: MMSE는 제곱오차의 기댓값을 최소화하는 추정값이고, 그 답은 항상 (해당 분포의) 평균이다.

## 1. MMSE의 정의

상수 c로 X를 추정할 때 벌점을 제곱오차의 기댓값 E[(X−c)²]로 재고, 이를 최소화하는 c가 MMSE 추정값.

## 2. 무조건부 유도

m = E[X]로 두고, 더하고 빼서 전개:

```
E[(X−c)²] = E[(X − m + m − c)²]
          = E[(X−m)²]  +  2(m−c)·E[(X−m)]  +  (m−c)²
```

가운데 항: E[(X−m)] = E[X] − m = 0 (m이 평균이라). 따라서:

```
E[(X−c)²] = Var(X)  +  (m−c)²
            └ 고정 ┘   └ c로 조절 ┘
```

- 첫 항 Var(X): c와 무관 — 못 줄이는 바닥.
- 둘째 항 (m−c)²: c = m일 때 정확히 0 → 최소.

```
∴ 제곱오차를 최소화하는 c = m = E[X]
```

## 3. 조건이 붙으면 (사건/관측 조건부)

조건부 분포 위에서 같은 논리가 그대로 반복된다:

```
E[(X−c)² | X<0.5] = Var(X | X<0.5) + (E[X | X<0.5] − c)²
→ 최소화 c = E[X | X<0.5]
```

즉 "사건 조건 MMSE 추정 = 조건부 평균". blind estimate(관측 없음)는 조건이 없는 특수 케이스라 그냥 E[X].

## 4. 손실함수 관점 (MAP와 대비)

MMSE가 평균인 건 손실이 **제곱**이기 때문. 손실이 0-1(틀리면 1)이었다면 답은 평균이 아니라 사후분포의 **봉우리(MAP)**.

| 손실함수 | 최적 추정값 | 이름 |
|---|---|---|
| 제곱 (X−c)² | 평균 E[·] | MMSE = LMS |
| 0-1 (맞/틀) | 봉우리 mode | MAP |
| 절댓값 |X−c| | 중앙값 median | — |

## 5. 한 줄 요약

> MMSE는 항상 (해당 분포의) 평균 — 제곱손실을 전개하면 Var + (평균−c)² 로 쪼개지고, c를 평균에 맞춰 둘째 항을 0으로 만들 때 최소가 되기 때문. 조건이 붙으면 그 조건부 분포의 평균.
