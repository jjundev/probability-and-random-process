## 문제
X, Y는 결합 연속 확률변수이고 결합 PDF는

f_XY(x,y) = (1/4)x^2 + (1/6)y,   −1 ≤ x ≤ 1, 0 ≤ y ≤ 2
f_XY(x,y) = 0,   otherwise

(a) 주변 PDF f_X(x), f_Y(y)를 구하라.
(b) P(X > 0, Y < 1)을 구하라.
(c) P(X > 0 또는 Y < 1)을 구하라.
(d) P(X > 0 | Y < 1)을 구하라.
(e) P(X + Y > 0)을 구하라.

## 1. 문제 정리
주어진 것: 직사각형 영역 [−1,1]×[0,2]에서 정의된 결합 PDF f_XY(x,y) = (1/4)x^2 + (1/6)y.
구할 것: (a) 두 주변 PDF, (b)~(e) 네 가지 확률.

## 2. 무엇을 묻고 왜 이 도구인가
결합 PDF 한 개로부터 주변·결합·조건부 확률을 모두 끌어내는 종합 문제다.
- 주변 PDF는 한 변수를 적분해 소거(marginalize): f_X(x) = ∫ f_XY dy, f_Y(y) = ∫ f_XY dx. (Lecture06, p.11)
- 사각형 영역의 확률은 결합 PDF의 이중적분: P((X,Y)∈B) = ∬_B f_XY dx dy. (Lecture06, p.11)
- 조건부는 정의 그대로 P(A|B) = P(A∩B)/P(B). (Lecture02, p.7)

## 3. 핵심 통찰
적분 영역이 직사각형이라 x·y가 분리돼 적분 한계가 상수다 — (b)(c)(d)는 단순 이중적분, (e)만 x+y>0이 영역을 비스듬히 잘라 여확률(complement)로 가는 게 빠르다.

## 4. 풀이

먼저 정규화 점검(검산 겸용):
∫_{-1}^1 ∫_0^2 [(1/4)x^2 + (1/6)y] dy dx
 = ∫_{-1}^1 (1/2)x^2 dx + ∫_{-1}^1 (1/3) dx   [∫_0^2 (1/4)x^2 dy = (1/2)x^2, ∫_0^2 (1/6)y dy = 1/3]
 = (1/2)(2/3) + (1/3)(2) = 1/3 + 2/3 = 1 ✓

**(a) 주변 PDF**

f_X(x) = ∫_0^2 [(1/4)x^2 + (1/6)y] dy = (1/4)x^2·2 + (1/6)·(2^2/2) = (1/2)x^2 + 1/3

f_X(x) = (1/2)x^2 + 1/3,   −1 ≤ x ≤ 1
f_X(x) = 0,   otherwise

f_Y(y) = ∫_{-1}^1 [(1/4)x^2 + (1/6)y] dx = (1/4)·(2/3) + (1/6)y·2 = 1/6 + (1/3)y

f_Y(y) = 1/6 + (1/3)y,   0 ≤ y ≤ 2
f_Y(y) = 0,   otherwise

**(b) P(X > 0, Y < 1)**

P(X>0, Y<1) = ∫_0^1 ∫_0^1 [(1/4)x^2 + (1/6)y] dy dx
 = ∫_0^1 [(1/4)x^2 + (1/6)(1/2)] dx   [∫_0^1 (1/6)y dy = 1/12]
 = ∫_0^1 [(1/4)x^2 + 1/12] dx = (1/4)(1/3) + 1/12 = 1/12 + 1/12 = 1/6

**(c) P(X > 0 또는 Y < 1)** — 포함배제 사용:

P(X>0) = ∫_0^1 f_X(x) dx = ∫_0^1 [(1/2)x^2 + 1/3] dx = (1/2)(1/3) + 1/3 = 1/6 + 1/3 = 1/2
P(Y<1) = ∫_0^1 f_Y(y) dy = ∫_0^1 [1/6 + (1/3)y] dy = 1/6 + (1/3)(1/2) = 1/6 + 1/6 = 1/3

P(X>0 또는 Y<1) = P(X>0) + P(Y<1) − P(X>0, Y<1) = 1/2 + 1/3 − 1/6 = 3/6 + 2/6 − 1/6 = 2/3

**(d) P(X > 0 | Y < 1)**

P(X>0 | Y<1) = P(X>0, Y<1) / P(Y<1) = (1/6) / (1/3) = 1/2

(참고: P(X>0 | Y<1) = 1/2 = P(X>0)이지만, 이는 우연이 아니라 PDF의 x-부분과 y-부분 구조 때문에 대칭에서 나온 값이며, 일반적으로 X, Y는 독립이 아니다 — f_XY ≠ f_X·f_Y.)

**(e) P(X + Y > 0)** — 여확률이 빠르다.

여집합 {X+Y ≤ 0} 즉 y ≤ −x. y ≥ 0이어야 하므로 −x ≥ 0, 즉 x ≤ 0이고 0 ≤ y ≤ −x인 삼각 영역.

P(X+Y ≤ 0) = ∫_{-1}^0 ∫_0^{−x} [(1/4)x^2 + (1/6)y] dy dx
 = ∫_{-1}^0 [ (1/4)x^2·(−x) + (1/6)·((−x)^2/2) ] dx
 = ∫_{-1}^0 [ −(1/4)x^3 + (1/12)x^2 ] dx
 = −(1/4)·(x^4/4)|_{-1}^0 + (1/12)·(x^3/3)|_{-1}^0
 = −(1/16)(0 − 1) + (1/36)(0 − (−1))
 = 1/16 + 1/36 = 9/144 + 4/144 = 13/144

P(X+Y > 0) = 1 − 13/144 = 131/144 ≈ 0.9097

## 5. 검산·직관
- 정규화 ∫∫ f_XY = 1 위에서 확인 ✓.
- f_X(x) ≥ 0, f_Y(y) ≥ 0 전 구간에서 성립(둘 다 양항 합). ∫_{-1}^1 f_X = (1/2)(2/3)+(1/3)(2) = 1/3+2/3 = 1 ✓, ∫_0^2 f_Y = (1/6)(2)+(1/3)(2) = 1/3+2/3 = 1 ✓.
- (c)에서 합집합 확률 2/3은 각 사건 확률(1/2, 1/3)보다 크고 1 이하 ✓.
- (e)에서 X+Y>0 영역이 전체의 대부분(작은 좌하단 삼각형만 제외)이라 확률이 1에 가까운 게 직관과 일치 ✓.

## 6. 한 줄 요약
> 결합 PDF 하나면 주변(한 변수 적분)·결합·조건부 확률이 다 나오고, 비스듬히 잘리는 영역 P(X+Y>0)은 작은 여집합 삼각형을 적분해 1에서 빼는 게 최단 경로다.
