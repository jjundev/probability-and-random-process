## 문제
두 확률변수 X, Y의 결합 PMF가 Table 5.6으로 주어진다.

결합 PMF P_{X,Y}(x,y):

            Y=0      Y=1      Y=2
  X=0       1/6      1/4      1/8
  X=1       1/8      1/6      1/6

Cov(X,Y)와 ρ(X,Y)를 구하라.

## 1. 문제 정리
- 주어진 것: 위 6칸 결합 PMF (그 외 = 0).
- 구할 것: 공분산 Cov(X,Y), 상관계수 ρ(X,Y).
- X ∈ {0,1}, Y ∈ {0,1,2}인 이산 결합분포.

## 2. 무엇을 묻고 왜 이 도구인가
공분산은 계산식 Cov(X,Y) = E[XY] − E[X]E[Y] 로 구한다 (Lecture07, p.23). 상관계수는 공분산을 각 표준편차로 정규화한 값
ρ(X,Y) = Cov(X,Y) / √(Var(X)·Var(Y)) 로, 차원 없는 [−1,1] 유계 지표다 (Lecture08, p.6).
따라서 필요한 재료는 E[X], E[Y], Var(X), Var(Y), E[XY] 다섯 개뿐 — 모두 결합 PMF와 주변 PMF에서 직접 합으로 얻는다.

## 3. 핵심 통찰
E[XY]는 결합 PMF에서 xy가 0이 아닌 칸만 살아남는다. 여기서 X=0인 행은 전부 0, Y=0인 칸도 0이므로 E[XY]에 기여하는 칸은 (X=1,Y=1)과 (X=1,Y=2) **단 둘**뿐이다.

## 4. 풀이

먼저 합이 1인지 확인하며 결합 PMF를 24분모로 통일:
1/6 + 1/4 + 1/8 + 1/8 + 1/6 + 1/6 = (4+6+3+3+4+4)/24 = 24/24 = 1.  (otherwise P_{X,Y} = 0)

주변 PMF (행·열 합):
P_X(0) = 1/6+1/4+1/8 = 13/24,   P_X(1) = 1/8+1/6+1/6 = 11/24,   otherwise P_X = 0
P_Y(0) = 1/6+1/8 = 7/24,   P_Y(1) = 1/4+1/6 = 10/24,   P_Y(2) = 1/8+1/6 = 7/24,   otherwise P_Y = 0

1차·2차 모멘트:
E[X] = 0·(13/24) + 1·(11/24) = 11/24
E[Y] = 0·(7/24) + 1·(10/24) + 2·(7/24) = (10+14)/24 = 1
E[X^2] = 0 + 1·(11/24) = 11/24      (X ∈ {0,1}이라 X^2 = X)
E[Y^2] = 0 + 1·(10/24) + 4·(7/24) = (10+28)/24 = 38/24 = 19/12

분산:
Var(X) = E[X^2] − (E[X])^2 = 11/24 − (11/24)^2 = (11/24)(13/24) = 143/576
Var(Y) = E[Y^2] − (E[Y])^2 = 19/12 − 1 = 7/12

곱의 기댓값 (X=1 행의 Y=1, Y=2 칸만 기여):
E[XY] = (1·1)·(1/6) + (1·2)·(1/6) = 1/6 + 2/6 = 1/2

공분산:
Cov(X,Y) = E[XY] − E[X]E[Y] = 1/2 − (11/24)·1 = 12/24 − 11/24 = 1/24

상관계수:
Var(X)·Var(Y) = (143/576)(7/12) = 1001/6912
ρ(X,Y) = (1/24) / √(1001/6912) = (1/24)·√(6912/1001) = (1/24)·(48√3/√1001) = 2√3/√1001 ≈ 0.1095

## 5. 검산·직관
- 결합 PMF 합 = 1 ✓, 모든 주변확률 ∈ [0,1] ✓.
- Cov > 0: X가 1로 커질 때 Y도 큰 값(1,2) 쪽으로 약하게 쏠림 → 양의 방향성, 부호 직관 일치.
- ρ ≈ 0.11 로 0에 가까움 → 선형 의존성이 약함. |ρ| ≤ 1 유계 만족 ✓ (Lecture08, p.6).
- Cov = 1/24 ≈ 0.0417 로 작고, 표준편차 곱 √(1001/6912) ≈ 0.381 로 나눠 ρ ≈ 0.11 — 자릿수 타당.

## 6. 한 줄 요약
> Cov = E[XY] − E[X]E[Y]로 부호·크기를, ρ = Cov/√(VarX·VarY)로 [−1,1] 정규화 — E[XY]는 xy≠0인 칸만 더하면 끝.
