## 문제
(Problem 4.6.14의 연속) 시각 t = 0 에서 주식 가격은 상수 k 달러이다. 시각 t > 0 에서 주식 가격은 E[X] = k, Var[X] = t 인 가우시안 확률변수 X 이다. 시각 t 에서 행사가 k 의 콜옵션(Call Option at Strike k)의 가치는

V = (X − k)^+,

이며 연산자 (·)^+ 는 (z)^+ = max(z, 0) 으로 정의된다. 매달 시작(t = 30일)마다 행사가 k 의 콜옵션을 가격 D 에 살 수 있는데, D 는 매달 변동하는 확률변수이다. 가격 D 가 임계값 d* 이하일 때만 콜을 사기로 결정한다. 기대수익 E[R] 을 최대화하는 임계값 d* 의 값은 무엇인가?

(참고 — 4.6.14 설정: 옵션을 d 달러에 사서 시각 t 에 V 달러에 팔면 수익 R = V − d.)

## 1. 문제 정리
- 주어진 것:
  - X ~ N(k, t) (평균 k, 분산 t). 콜옵션 가치 V = (X − k)^+ = max(X − k, 0).
  - 매달 옵션 매수가 D는 확률변수(시장에서 변동), 주식의 만기 가치 V와 독립.
  - 전략: D ≤ d* 이면 산다(수익 R = V − D), D > d* 이면 안 산다(R = 0).
- 구할 것: E[R]을 최대화하는 임계값 d*.

## 2. 무엇을 묻고 왜 이 도구인가
두 단계다. (1) 먼저 옵션의 기대가치 E[V] = E[(X−k)^+]를 구한다 — X − k ~ N(0, t)이므로 "정규의 양의 부분 기댓값"이라는 파생분포 기댓값 계산 (Lecture06, p.5의 정규 + Lecture05, p.7 기댓값 규칙). (2) E[R]을 d*의 함수로 쓰고 미분해 0으로 두는 최적화. 여기서 V ⊥ D 이므로 기댓값이 분해된다 (Lecture07, p.23: 독립 ⇒ E[VD] = E[V]E[D]). "사느냐 마느냐"는 지시함수 1{D ≤ d*}로 표현.

## 3. 핵심 통찰
E[R](d*)를 d*로 미분하면 f_D(d*)·(E[V] − d*) 가 되어, 가격이 기대가치와 같아지는 지점 d* = E[V] 에서 정확히 0(최댓값)이 된다 — "기대 페이오프보다 싸면 사라"가 최적.

## 4. 풀이

**Step 1 — 옵션의 기대가치 E[V] (= 4.6.14(a)):**
X ~ N(k, t) ⇒ Z := X − k ~ N(0, t) (정규의 평행이동, Lecture06, p.7). σ^2 = t, 즉 표준편차 σ = √t.
V = Z^+ = max(Z, 0). 기댓값 규칙 (Lecture05, p.7):
E[V] = ∫_(−∞)^∞ max(z, 0)·f_Z(z) dz = ∫_0^∞ z · (1/√(2πt)) e^(−z^2/(2t)) dz.

치환 u = z^2/(2t), du = z/t dz ⇒ z dz = t du:
E[V] = (1/√(2πt)) · t ∫_0^∞ e^(−u) du = (t/√(2πt)) · 1 = √(t)/√(2π) = √( t / (2π) ).

(일반식: 평균0·분산σ^2 정규의 양의부분 기댓값 = σ/√(2π); 여기 σ = √t.)

**Step 2 — E[R]을 d*의 함수로:**
사면(D ≤ d*) R = V − D, 안 사면(D > d*) R = 0. 지시함수로:
R = (V − D)·1{D ≤ d*}.
V ⊥ D 이므로 (Lecture07, p.23: 독립 ⇒ E[V·(D항)] 분리):
E[R] = E[ V·1{D ≤ d*} ] − E[ D·1{D ≤ d*} ]
   = E[V]·P(D ≤ d*) − E[ D·1{D ≤ d*} ]
   = E[V]·F_D(d*) − ∫_(−∞)^(d*) u · f_D(u) du,
여기서 F_D는 D의 CDF, f_D는 PDF (Lecture05, p.9–10).

**Step 3 — d*로 미분해 최댓값:**
미적분 기본정리로 d*에 대한 미분 (적분 상한 미분 → 피적분함수 대입):
d/dd* E[R] = E[V]·f_D(d*) − d*·f_D(d*) = f_D(d*)·( E[V] − d* ).

f_D(d*) ≥ 0 이므로 부호는 (E[V] − d*)가 결정:
- d* < E[V] : 도함수 > 0 (E[R] 증가)
- d* > E[V] : 도함수 < 0 (E[R] 감소)
따라서 E[R]은 d* = E[V] 에서 최대.

**결론:**
d* = E[V] = √( t / (2π) ).
즉, 옵션 가격 D가 기대 페이오프 E[V] = √(t/(2π)) 이하일 때만 사는 전략이 기대수익을 최대화한다.

## 5. 검산·직관
- d* = E[V]는 "기대가치보다 싸게 살 때만 매수"라는 합리적 의사결정 규칙과 정확히 일치 — 옵션의 기대 페이오프가 곧 공정가격(손익분기 기댓값)이기 때문.
- 차원·부호: E[V] = √(t/(2π)) > 0, 시간 t가 클수록(주가 불확실성 ↑) 옵션 가치 ↑ → 임계값 ↑. 직관 일치.
- 최댓값 확인: 도함수 부호가 d* = E[V] 좌우로 +→− 전환 ⇒ 극대(최대) ✓.
- 4.6.14(b)의 d_0(이익 확률 1/2 기준 = 중앙값)와 다름에 유의 — 여기 기준은 "기대수익 최대"라 평균 E[V]가 답. 두 전략의 기준이 다르면 임계값도 다르다.

## 6. 한 줄 요약
> 기대수익을 최대화하는 매수 임계값은 옵션의 기대가치와 같다 — d* = E[(X−k)^+] = √(t/(2π)), 즉 "기대 페이오프보다 싸면 사라".
