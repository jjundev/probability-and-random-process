## 문제
Problem 5.2.1의 확률변수 X와 Y에 대해 다음을 구하라.
(a) 주변 PMF P_X(x)와 P_Y(y)
(b) 기댓값 E[X]와 E[Y]
(c) 표준편차 σ_X와 σ_Y

(Problem 5.2.1: X, Y의 결합 PMF는 P_{X,Y}(x,y) = c·x·y,  x = 1, 2, 4; y = 1, 3;  otherwise = 0.)

## 1. 문제 정리
주어진 것: P_{X,Y}(x,y) = c·x·y, x∈{1,2,4}, y∈{1,3}, otherwise = 0. 상수 c는 미정 → 먼저 정규화로 결정.
구할 것: (a) 두 주변 PMF, (b) 두 기댓값, (c) 두 표준편차.

## 2. 무엇을 묻고 왜 이 도구인가
- 정규화: Σ_x Σ_y P_{X,Y} = 1로 c 결정. (Lecture03, p.20)
- 주변 PMF: 한 변수를 합으로 소거. P_X(x) = Σ_y P_{X,Y}(x,y), P_Y(y) = Σ_x P_{X,Y}(x,y). (Lecture04 / Lecture03, p.20)
- E, σ: E[X]=Σ x P_X(x), Var[X]=E[X^2]−(E[X])^2, σ=√Var. (Lecture03, p.14, p.18)

## 3. 핵심 통찰
결합 PMF가 x·y로 곱분리되므로, 이중합 Σ_x Σ_y c·x·y = c·(Σ_x x)(Σ_y y)로 분해된다 — 정규화와 주변 PMF가 두 일변량 합의 곱으로 간단해진다. (게다가 P_{X,Y} = P_X·P_Y가 되어 X ⊥ Y.)

## 4. 풀이

**상수 c**: Σ_x x = 1+2+4 = 7, Σ_y y = 1+3 = 4.
Σ_x Σ_y c·x·y = c·7·4 = 28c = 1  ⇒  c = 1/28.

따라서 P_{X,Y}(x,y) = xy/28,  x∈{1,2,4}, y∈{1,3},  otherwise = 0.

**(a) 주변 PMF**

P_X(x) = Σ_{y∈{1,3}} xy/28 = (x/28)(1+3) = 4x/28 = x/7

P_X(1) = 1/7,  P_X(2) = 2/7,  P_X(4) = 4/7
P_X(x) = 0,  otherwise

P_Y(y) = Σ_{x∈{1,2,4}} xy/28 = (y/28)(1+2+4) = 7y/28 = y/4

P_Y(1) = 1/4,  P_Y(3) = 3/4
P_Y(y) = 0,  otherwise

**(b) 기댓값**

E[X] = Σ x P_X(x) = 1·(1/7) + 2·(2/7) + 4·(4/7) = (1 + 4 + 16)/7 = 21/7 = 3

E[Y] = Σ y P_Y(y) = 1·(1/4) + 3·(3/4) = (1 + 9)/4 = 10/4 = 5/2 = 2.5

**(c) 표준편차**

E[X^2] = 1^2·(1/7) + 2^2·(2/7) + 4^2·(4/7) = (1 + 8 + 64)/7 = 73/7
Var[X] = E[X^2] − (E[X])^2 = 73/7 − 9 = 73/7 − 63/7 = 10/7
σ_X = √(10/7) ≈ 1.195

E[Y^2] = 1^2·(1/4) + 3^2·(3/4) = (1 + 27)/4 = 28/4 = 7
Var[Y] = E[Y^2] − (E[Y])^2 = 7 − (5/2)^2 = 7 − 25/4 = 3/4
σ_Y = √(3/4) = √3/2 ≈ 0.866

## 5. 검산·직관
- 주변 PMF 합: P_X: 1/7+2/7+4/7 = 1 ✓; P_Y: 1/4+3/4 = 1 ✓.
- E[X]=3 ∈ [1,4], E[Y]=2.5 ∈ [1,3] — 둘 다 지지구간 안 ✓.
- Var 양수, (E[X])^2=9 ≤ E[X^2]=73/7≈10.43 ✓; (E[Y])^2=6.25 ≤ E[Y^2]=7 ✓.
- 곱분리 확인: P_X(x)P_Y(y) = (x/7)(y/4) = xy/28 = P_{X,Y}(x,y) → X ⊥ Y ✓ (Lecture04, slide 19). 독립이라 주변에서 E·σ를 따로 구해도 일관.

## 6. 한 줄 요약
> 곱분리형 P_{X,Y}=cxy는 c=1/28로 정규화되고 주변이 P_X=x/7, P_Y=y/4로 깔끔히 갈라져(독립) E[X]=3, E[Y]=2.5, σ_X=√(10/7), σ_Y=√3/2가 나온다.
