## 문제
X_1, …, X_n 은 각각 기댓값 5인 지수(exponential) 랜덤변수의 i.i.d. 수열이다.
(a) 9회 시행에 기반한 표본평균의 분산 Var[M_9(X)] 는 얼마인가?
(b) 한 결과가 7을 초과할 확률 P[X_1 > 7] 은 얼마인가?
(c) 중심극한정리를 사용하여, 9회 시행의 표본평균이 7을 초과할 확률 P[M_9(X) > 7] 을 추정하라.

## 1. 문제 정리
| 항목 | 내용 |
|---|---|
| 분포 | X_i ~ Exponential, E[X_i] = μ = 5 (i.i.d.) |
| 지수의 분산 | Var(X_i) = μ^2 = 25 (지수는 평균=1/λ, 분산=1/λ^2=μ^2) |
| n | 9 |
| 표본평균 | M_9 = (X_1+…+X_9)/9 |
| (a) | Var[M_9] |
| (b) | P[X_1 > 7] (정확값) |
| (c) | P[M_9 > 7] (CLT 근사) |

지수 PDF (λ = 1/μ = 1/5):

  f_X(x) = (1/5) e^(−x/5),   x ≥ 0
  f_X(x) = 0,   otherwise

## 2. 무엇을 묻고 왜 이 도구인가
표본평균 M_n 의 분산과 그 꼬리확률을 묻는 표준 CLT 문제. (a)는 표본평균 분산 공식 Var[M_n] = σ^2/n 의 직접 적용, (b)는 지수의 정확한 CDF, (c)는 M_9 의 분포를 가우시안으로 근사. (b)와 (c)를 나란히 묻는 의도: "단일 X의 꼬리"와 "9개 평균의 꼬리"가 얼마나 다른지(평균이 7을 넘는 것이 훨씬 드묾) 체감시키기 위함. (Lecture09, slide 6, slide 9)

## 3. 핵심 통찰
표본평균은 μ를 중심으로 분산이 σ^2/n 로 줄어든다(Lecture09, slide 6) — n=9면 표준편차가 1/√9 = 1/3 로 축소. 그래서 단일 X_1 이 7을 넘을 확률(b)은 꽤 크지만, 9개 평균이 7을 넘을 확률(c)은 훨씬 작아진다.

## 4. 풀이

(a) 표본평균의 분산 — (Lecture09, slide 6; "주요 공식" Var[M_n] = σ^2/n):

  Var[M_9] = σ^2 / n = 25 / 9 ≈ 2.7778

(b) 지수의 정확한 꼬리확률. 지수 CDF는 P(X ≤ x) = 1 − e^(−x/μ) (x ≥ 0)이므로

  P[X_1 > 7] = e^(−7/5) = e^(−1.4) ≈ 0.2466

(이 부분은 지수분포의 정확한 꼬리식 — Lecture09는 표본평균·CLT 중심이라 지수 CDF 자체는 ⚠ 강의 밖 일반 지식. 연속분포 지수 CDF는 Lecture05~06 소관.)

(c) CLT로 P[M_9 > 7] 근사. 표본평균의 평균·표준편차 (Lecture09, slide 6):

  E[M_9] = μ = 5
  σ(M_9) = √(σ^2/n) = √(25/9) = 5/3 ≈ 1.6667

표준화 (Lecture09, slide 9 — Z_n = (S_n − nμ)/(σ√n), M_n 형태로는 (M_n − μ)/(σ/√n)):

  Z = (M_9 − 5)/(5/3)  ≈  N(0,1)

  P[M_9 > 7] = P( Z > (7 − 5)/(5/3) )
             = P( Z > 2/(1.6667) )
             = P( Z > 1.2 )
             = 1 − Φ(1.2)

표준정규 분포표에서 Φ(1.2) ≈ 0.8849 이므로

  P[M_9 > 7] ≈ 1 − 0.8849 = 0.1151

## 5. 검산·직관
- (b) vs (c): 단일 X가 7 초과 ≈ 0.247, 9개 평균이 7 초과 ≈ 0.115 — 평균이 더 드물게 7을 넘는다(분산 축소 효과). 방향 일치 ✓.
- (a) 단위: X가 시간이면 분산은 시간^2, 25/9 도 시간^2 차원. ✓
- 주의(근사 한계): n=9는 작고 지수는 크게 비대칭(skew)이라 CLT 근사 오차가 다소 큼 — 본 문제는 "estimate"라고 명시해 근사임을 인정. 정확값은 M_9 의 합 S_9 ~ Erlang(9, 1/5)로 계산해야 하나 문제는 CLT를 지시.

## 6. 한 줄 요약
> 표본평균은 분산이 σ^2/n로 줄어 μ에 집중하므로(Var[M_9]=25/9), 단일 X의 꼬리 0.247보다 9개 평균의 꼬리 P[M_9>7]≈1−Φ(1.2)=0.115 가 훨씬 작다.
