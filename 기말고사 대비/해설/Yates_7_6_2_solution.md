# Yates 7.6.2 풀이
> 출처: Probability and Stochastic Processes (Yates & Goodman), L08_PSP_7.6.2.png, 1순위 근거 Lecture08, 풀이일 2026-06-08
> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X와 Y는 결합 가우시안(jointly Gaussian) 확률변수로 E[X] = E[Y] = 0 이고 Var[X] = Var[Y] = 1 이다. 또한 E[Y|X] = X/2 이다. f_X,Y(x,y) 를 구하라.

## 1. 문제 정리
주어진 것:
- E[X] = μ_X = 0, E[Y] = μ_Y = 0.
- Var[X] = σ_X^2 = 1, Var[Y] = σ_Y^2 = 1 (σ_X = σ_Y = 1).
- E[Y|X] = X/2 (선형 회귀선).

구할 것: 결합 PDF f_X,Y(x,y) — 즉 상관계수 ρ 하나만 더 찾으면 표준 이변량정규 공식에 대입.

## 2. 무엇을 묻고 왜 이 도구인가
결합 가우시안은 (μ_X, μ_Y, σ_X, σ_Y, ρ) 다섯 파라미터로 완전히 결정된다. 네 개는 주어졌고 ρ만 모른다. 결정적 다리: 결합 가우시안의 조건부 평균은 선형이고 그 기울기가 ρσ_Y/σ_X 다 — 정리노트의 상관계수가 "선형 의존성의 지표"라는 해석(Lecture08, p.6)과 정확히 연결된다. 주어진 E[Y|X]=X/2 의 기울기 1/2 를 ρσ_Y/σ_X 와 맞추면 ρ가 나온다.

결합 가우시안의 조건부 평균이 선형이고 기울기가 ρσ_Y/σ_X 라는 사실, 그리고 이변량정규 PDF 공식은 Lecture08 정리노트에 명시 없음 → ⚠ 강의 밖(결합정규 고유 성질), 일반 교과서 지식으로 보강.

## 3. 핵심 통찰
결합 가우시안에서 E[Y|X=x] = μ_Y + ρ(σ_Y/σ_X)(x − μ_X). 여기에 μ=0, σ=1 대입하면 = ρx. 주어진 값 x/2 와 같으니 ρ = 1/2. 나머지는 공식 대입.

## 4. 풀이

### ρ 결정
결합 가우시안의 조건부 평균 (⚠ 강의 밖):
E[Y|X=x] = μ_Y + ρ·(σ_Y/σ_X)·(x − μ_X) = 0 + ρ·(1/1)·(x − 0) = ρx.
주어진 E[Y|X=x] = x/2 와 비교:
ρ = 1/2.
따라서 Cov(X,Y) = ρσ_Xσ_Y = (1/2)·1·1 = 1/2. (Lecture08, p.6 의 ρ 정의)

### 결합 PDF 대입
표준 이변량정규 PDF (μ_X=μ_Y=0, σ_X=σ_Y=1, ρ=1/2), ⚠ 강의 밖 공식:

일반형:
f_X,Y(x,y) = 1 / (2π σ_X σ_Y √(1−ρ^2)) · exp( −1/(2(1−ρ^2)) · [ (x−μ_X)^2/σ_X^2 − 2ρ(x−μ_X)(y−μ_Y)/(σ_Xσ_Y) + (y−μ_Y)^2/σ_Y^2 ] )

값 대입: σ_X=σ_Y=1, μ=0, ρ=1/2 ⟹ 1−ρ^2 = 1 − 1/4 = 3/4, √(1−ρ^2) = √3/2, 2π√(1−ρ^2) = 2π·(√3/2) = π√3, 그리고 1/(2(1−ρ^2)) = 1/(2·3/4) = 2/3:

f_X,Y(x,y) = 1/(π√3) · exp( −(2/3)·(x^2 − xy + y^2) ),   −∞ < x < ∞, −∞ < y < ∞.
f_X,Y(x,y) = 0,   otherwise.

(정규분포는 전 평면에서 양의 밀도를 가지므로 "otherwise"에 해당하는 유한 경계는 없으나, 형식상 정의역 밖 = 0 을 명시한다.)

여기서 −2ρ = −2·(1/2) = −1 이라 교차항이 −xy 가 되고, 괄호 안은 x^2 − xy + y^2 이다.

## 5. 검산·직관
- ρ = 1/2 ∈ (−1, 1) ✓ (유효한 상관계수, Lecture08 p.6 의 −1≤ρ≤1).
- 대칭성: x^2 − xy + y^2 는 x↔y 교환 대칭 — σ_X=σ_Y, μ_X=μ_Y 인 대칭 설정과 일치 ✓. 실제로 E[X|Y]=Y/2 도 같은 형태여야 하며 식이 이를 만족.
- 회귀 일관성: E[Y|X]=X/2 의 기울기 1/2 < 1 → X의 절반만 Y로 전달, 양의 상관 ✓.
- 지수 안 이차형식 x^2−xy+y^2 는 양의 정부호(판별식 1−4 = −3 < 0)라 PDF가 적분가능 ✓.

## 6. 한 줄 요약
> 결합 가우시안은 다섯 파라미터로 끝나며, 모르는 ρ는 "조건부 평균의 기울기 = ρσ_Y/σ_X" 다리로 E[Y|X]=X/2에서 ρ=1/2를 읽어 이변량정규 공식에 대입한다.
