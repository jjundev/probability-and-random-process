> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X와 Y는 다음 결합 PDF를 갖는 결합연속(jointly continuous) 확률변수이다.

f_XY(x, y) = (1/2)x^2 + (2/3)y,   −1 ≤ x ≤ 1, 0 ≤ y ≤ 1
f_XY(x, y) = 0,   otherwise

E[Y | X = 0]과 Var(Y | X = 0)을 구하라.

## 1. 문제 정리
- 주어진 것: 직사각형 영역 −1 ≤ x ≤ 1, 0 ≤ y ≤ 1 위의 결합 PDF f_XY = (1/2)x^2 + (2/3)y, 그 밖은 0.
- 구할 것: X = 0 조건에서 Y의 조건부 기댓값 E[Y|X=0]과 조건부 분산 Var(Y|X=0).

## 2. 무엇을 묻고 왜 이 도구인가
조건부 평균·분산을 구하려면 먼저 조건부 PDF f_(Y|X)(y|0)가 필요하다. 정의 f_(Y|X)(y|x) = f_XY(x,y) / f_X(x) (단 f_X(x) > 0) (Lecture06, p.18)에 따라, 주변 PDF f_X(x)를 적분으로 얻은 뒤 x = 0을 대입한다. 그다음 E[Y|X=0] = ∫ y·f_(Y|X)(y|0) dy, Var = E[Y^2|X=0] − (E[Y|X=0])^2 로 계산한다 (Lecture06, p.18; 조건부 2차 모멘트는 Lecture08, p.21의 조건부 분산 정의).

## 3. 핵심 통찰
X = 0을 대입하면 f_XY(0, y) = (2/3)y 로 x항이 사라져, 조건부 PDF가 [0,1]에서 y에 비례하는 삼각형 밀도 2y 로 단순해진다.

## 4. 풀이

**Step 1 — 주변 PDF f_X(x):** (Lecture06, p.11)
f_X(x) = ∫_0^1 [ (1/2)x^2 + (2/3)y ] dy
= (1/2)x^2·[y]_0^1 + (2/3)·[y^2/2]_0^1
= (1/2)x^2 + (2/3)·(1/2) = (1/2)x^2 + 1/3,   −1 ≤ x ≤ 1
f_X(x) = 0,   otherwise

x = 0 대입: f_X(0) = 0 + 1/3 = 1/3  (> 0 이므로 조건부 정의 가능).

**Step 2 — 조건부 PDF f_(Y|X)(y|0):** (Lecture06, p.18)
f_(Y|X)(y|0) = f_XY(0, y) / f_X(0) = [ (1/2)·0 + (2/3)y ] / (1/3)
= (2/3)y · 3 = 2y,   0 ≤ y ≤ 1
f_(Y|X)(y|0) = 0,   otherwise

검증: ∫_0^1 2y dy = [y^2]_0^1 = 1 ✓ (정규화된 PDF).

**Step 3 — E[Y | X = 0]:** (Lecture06, p.18)
E[Y | X = 0] = ∫_0^1 y · 2y dy = ∫_0^1 2y^2 dy = [ (2/3)y^3 ]_0^1 = 2/3.

**Step 4 — Var(Y | X = 0):** (Lecture08, p.21: Var = E[Y^2|·] − (E[Y|·])^2)
E[Y^2 | X = 0] = ∫_0^1 y^2 · 2y dy = ∫_0^1 2y^3 dy = [ (1/2)y^4 ]_0^1 = 1/2.
Var(Y | X = 0) = E[Y^2|X=0] − (E[Y|X=0])^2 = 1/2 − (2/3)^2 = 1/2 − 4/9
= 9/18 − 8/18 = 1/18 ≈ 0.0556.

## 5. 검산·직관
- 조건부 PDF 2y가 [0,1]에서 적분 1 ✓.
- E[Y|X=0] = 2/3 은 무조건 평균 1/2보다 크다 — 밀도 2y가 큰 y쪽으로 쏠려 있으니 평균이 오른쪽으로 이동, 직관 일치.
- Beta(2,1) 분포(밀도 2y)의 알려진 분산 = 1·2/((2+1)^2(2+1+1)) = 2/(9·4) = 1/18 과 정확히 일치 ✓.

## 6. 한 줄 요약
> 조건부 평균/분산은 f_(Y|X)(y|x) = f_XY/f_X 로 조건부 PDF부터 만들고 (여기선 X=0이 x항을 죽여 2y가 됨), 그 위에서 적분으로 모멘트를 구한다.
