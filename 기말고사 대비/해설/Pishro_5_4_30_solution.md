## 문제
Problem 29 에서, X 와 Y 가 독립 Uniform(0, 1) 확률변수라고 하자. R 과 Θ 의 결합 PDF 를 구하라. R 과 Θ 는 독립인가?

(Problem 29 의 설정: (X,Y) 의 극좌표 (R, Θ), R ≥ 0, −π < Θ ≤ π, 역변환
 X = R cos Θ,  Y = R sin Θ.)

## 1. 문제 정리
주어진 것: X, Y ~ Uniform(0,1) 독립 →
  f_(X,Y)(x,y) = 1,  0 ≤ x ≤ 1, 0 ≤ y ≤ 1
  f_(X,Y)(x,y) = 0,  otherwise   (Lecture05, p.7 균등 × 독립곱, Lecture06, p.18)
변환: X = R cos Θ, Y = R sin Θ (극좌표). R = √(X^2+Y^2), Θ = atan2(Y, X).
구할 것: 결합 PDF f_(R,Θ)(r, θ) 와 R, Θ 의 독립성 판정.

⚠ 강의 밖 보강: Lecture07 정리노트는 단일 변수 파생분포(CDF→미분)와 합의 convolution 까지만 다루고, **2변수 → 2변수 변환의 야코비안(Jacobian) 공식은 정리노트에 없다**. 따라서 변환 공식은 일반 교과서 지식으로 보강한다. 단 문제 의식("X 의 분포를 알 때 g(X,Y) 의 분포", Lecture07, p.5)과 독립 판정(f_(R,Θ)=f_R f_Θ 분리 여부, Lecture06, p.18)은 정리노트 도구 그대로다.

## 2. 무엇을 묻고 왜 이 도구인가
2변수 결합변환 문제. 1변수면 CDF→미분(Lecture07, p.10)이지만, (X,Y)→(R,Θ)처럼 둘을 동시에 바꿀 땐 **야코비안 변환 공식**을 쓴다:
  f_(R,Θ)(r,θ) = f_(X,Y)(x(r,θ), y(r,θ)) · |J|,
여기서 |J| 는 변환 (r,θ)→(x,y) 의 야코비안 행렬식 절댓값. 극좌표의 표준 결과 |J| = r 을 쓴다. 독립 판정은 결합이 r-함수 × θ-함수로 분리되고 지지가 직사각형(곱집합)인지로 본다 (Lecture06, p.18).

## 3. 핵심 통찰
"야코비안 |J| = r 이 넓이 왜곡을 보정한다" — 균등(밀도 1)을 극좌표로 옮기면 밀도가 r 로 가중되고, 지지가 단위정사각형이라 r 과 θ 의 범위가 서로 얽혀(coupled) 결국 독립이 아니다.

## 4. 풀이

(단계 1) 야코비안
x = r cos θ, y = r sin θ 에 대한 야코비안 행렬:
∂x/∂r = cos θ,   ∂x/∂θ = −r sin θ
∂y/∂r = sin θ,   ∂y/∂θ =  r cos θ
J = (∂x/∂r)(∂y/∂θ) − (∂x/∂θ)(∂y/∂r) = cos θ·(r cos θ) − (−r sin θ)·sin θ
  = r cos²θ + r sin²θ = r.
|J| = r (r ≥ 0).

(단계 2) 변환 공식 적용
f_(R,Θ)(r,θ) = f_(X,Y)(r cos θ, r sin θ)·|J| = f_(X,Y)(r cos θ, r sin θ)·r.
f_(X,Y) = 1 은 (r cos θ, r sin θ) 가 단위정사각형 0 ≤ x ≤ 1, 0 ≤ y ≤ 1 안일 때만, 그 밖은 0.
조건 0 ≤ r cos θ ≤ 1 과 0 ≤ r sin θ ≤ 1 ⟹ 1사분면(0 ≤ θ ≤ π/2)이고 r 은 정사각형 안에 머무는 동안.
따라서
f_(R,Θ)(r,θ) = r,  (r,θ) 가 0 ≤ θ ≤ π/2 이고 (r cos θ, r sin θ) ∈ [0,1]×[0,1]
f_(R,Θ)(r,θ) = 0,  otherwise

지지를 θ 로 풀어 쓰면 (단위정사각형의 r-반지름 한계):
- 0 ≤ θ ≤ π/4: x = r cos θ 가 먼저 1 에 닿음 → 0 ≤ r ≤ 1/cos θ.
- π/4 ≤ θ ≤ π/2: y = r sin θ 가 먼저 1 에 닿음 → 0 ≤ r ≤ 1/sin θ.
즉 r 의 상한이 θ 에 의존한다(정사각형의 모서리까지 거리).

(단계 3) R 과 Θ 는 독립인가? — 아니다.
독립이려면 f_(R,Θ)(r,θ) = f_R(r)·f_Θ(θ) 로 분리되고 지지가 r-구간 × θ-구간(직사각형)이어야 한다 (Lecture06, p.18). 그런데
- 밀도 부분 r 자체는 θ 무관이지만,
- **지지가 직사각형이 아니다**: r 의 허용 상한이 θ 에 따라 1/cos θ ↔ 1/sin θ 로 바뀐다. 지지가 곱집합이 아니므로 분리 불가능.
따라서 R 과 Θ 는 **독립이 아니다**.
(대조: Problem 29 처럼 X,Y 가 표준정규였다면 f_(X,Y) ∝ e^(−r²/2) 가 전 평면을 지지로 가져 지지가 r×θ 직사각형이 되고 R⊥Θ 가 성립한다. 균등(0,1)은 지지가 유한 정사각형이라 이 좋은 성질이 깨진다.)

## 5. 검산·직관
- 정규화: ∬ f_(R,Θ) dr dθ 는 야코비안 정의상 ∬ f_(X,Y) dx dy = 단위정사각형 넓이 1 과 같아야 한다. 실제 ∫_0^(π/4)∫_0^(1/cosθ) r dr dθ + ∫_(π/4)^(π/2)∫_0^(1/sinθ) r dr dθ. 첫 항 = ∫_0^(π/4) (1/2)sec²θ dθ = (1/2)[tan θ]_0^(π/4) = 1/2; 둘째 항 대칭으로 1/2; 합 = 1 ✓.
- 밀도가 r 에 비례 → 원점에서 멀수록 무게↑. 균등정사각형에서 큰 반지름 영역(모서리 쪽)이 더 넓은 호를 차지하니 직관과 일치.
- 독립 깨짐의 본질은 "정사각형은 원형 대칭이 아니다" — 지지의 모양이 변수 결합을 만든다.

## 6. 한 줄 요약
> 2변수 극좌표 변환은 야코비안 |J|=r 로 f_(R,Θ)=r·f_(X,Y) 를 만들고, 균등(0,1)²는 지지가 정사각형이라 r 의 범위가 θ 에 얽혀 R 과 Θ 가 독립이 아니다.
