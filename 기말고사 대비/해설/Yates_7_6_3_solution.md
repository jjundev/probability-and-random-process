# Yates 7.6.3 풀이
> 출처: Probability and Stochastic Processes (Yates & Goodman), L08_PSP_7.6.3.png, 1순위 근거 Lecture08, 풀이일 2026-06-08
> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
자전거 라이더 연구에서, 남성 사이클리스트의 속도 X(century 주행에서 mph)와 체중 Y(kg)는 이변량 가우시안 PDF f_X,Y(x,y) 로 모델링되며 파라미터는 μ_X = 20, σ_X = 2, μ_Y = 75, σ_Y = 5, ρ_X,Y = −0.6 이다. 여성 사이클리스트의 속도 X′와 체중 Y′ 역시 이변량 가우시안 f_X′,Y′(x′,y′) 로 μ_X′ = 15, σ_X′ = 2, μ_Y′ = 50, σ_Y′ = 5, ρ_X′,Y′ = −0.6 이다. 남녀 모두 속도와 체중이 음의 상관(−0.6) — "빠른 사이클리스트는 마르다"는 통념과 일치. 혼합 집단에서 무작위로 한 명을 뽑으면 남성일 확률은 p = 0.80 이다.

남녀 모두 ρ = −0.6 이므로 차이를 무시해도 될 것 같다. 이를 확인하기 위해, 큰 혼합 집단에서 무작위로 뽑은 사이클리스트의 속도 X̂ 와 체중 Ŷ 를 연구한다. X̂ 와 Ŷ 는 어떻게 상관되어 있는가? 답을 설명하라.

## 1. 문제 정리
M = 성별 지시자: 남성(확률 p=0.8), 여성(확률 1−p=0.2).
- 남성: μ_X=20, σ_X=2, μ_Y=75, σ_Y=5, ρ=−0.6 → Cov(X,Y|남)=−0.6·2·5 = −6.
- 여성: μ_X′=15, σ_X′=2, μ_Y′=50, σ_Y′=5, ρ=−0.6 → Cov(X,Y|여)=−0.6·2·5 = −6.

구할 것: 혼합 집단에서 (X̂, Ŷ) 의 상관계수 ρ(X̂, Ŷ).

## 2. 무엇을 묻고 왜 이 도구인가
함정 문제다. 혼합 집단 변수 X̂ 는 "성별 M을 먼저 뽑고, 그 그룹의 분포에서 X를 뽑는" 2단 구조다. 따라서 평균·분산·공분산을 성별로 조건화해 반복기댓값(Lecture08, p.17)과 분산분해(Law of total variance, Lecture08, p.23)로 합쳐야 한다. 그룹 내 상관 −0.6 은 그룹 내 변동만 말할 뿐, 혼합 집단의 상관은 "그룹 간 평균 차이"가 추가로 기여한다 — 정리노트의 LTV 해석 "그룹 내 분산 + 그룹 간 분산"(Lecture08, p.24)이 정확히 이 상황이다.

혼합분포(두 가우시안의 혼합)는 Lecture08 정리노트에 명시 없으나, 풀이 도구(반복기댓값·LTV·공분산 정의)는 모두 Lecture08 범위 안이다. 혼합 자체의 비정규성 언급만 ⚠ 강의 밖.

## 3. 핵심 통찰
혼합 집단의 공분산 = (그룹 내 공분산의 평균) + (그룹 평균들 사이의 공분산). 그룹 내는 둘 다 −6 로 음수지만, 남성은 (속도↑, 체중↑) 여성은 (속도↓, 체중↓)이라 그룹 평균이 같은 방향으로 움직여 큰 양(+)의 기여를 만든다. 후자가 전자를 압도해 ρ(X̂,Ŷ)는 양수가 된다 — Simpson의 역설.

## 4. 풀이

### 혼합 평균 (반복기댓값, Lecture08 p.17)
E[X̂] = E[E[X̂|M]] = 0.8·20 + 0.2·15 = 16 + 3 = 19.
E[Ŷ] = E[E[Ŷ|M]] = 0.8·75 + 0.2·50 = 60 + 10 = 70.

### 혼합 공분산
공분산 정의 Cov(X̂,Ŷ) = E[X̂Ŷ] − E[X̂]E[Ŷ]. 곱의 기댓값을 성별 조건화로 (반복기댓값):
E[X̂Ŷ] = E[ E[X̂Ŷ|M] ]. 각 그룹 내부에서 E[XY|M] = Cov(X,Y|M) + E[X|M]E[Y|M]:
- 남성: E[XY|남] = −6 + 20·75 = −6 + 1500 = 1494.
- 여성: E[XY|여] = −6 + 15·50 = −6 + 750 = 744.
E[X̂Ŷ] = 0.8·1494 + 0.2·744 = 1195.2 + 148.8 = 1344.

Cov(X̂,Ŷ) = 1344 − (19)(70) = 1344 − 1330 = 14.

(동치 해석 — LTV의 공분산판:
"그룹 내 공분산 평균" = 0.8·(−6) + 0.2·(−6) = −6.
"그룹 평균 간 공분산" = 0.8·(20−19)(75−70) + 0.2·(15−19)(50−70)
   = 0.8·(1)(5) + 0.2·(−4)(−20) = 4 + 16 = 20.
합 = −6 + 20 = 14. ✓ 위와 일치.)

### 혼합 분산 (Law of total variance, Lecture08 p.23)
Var(X̂) = E[Var(X|M)] + Var(E[X|M]).
- E[Var(X|M)] = 0.8·(2^2) + 0.2·(2^2) = 0.8·4 + 0.2·4 = 4.
- Var(E[X|M]): E[X|M] ∈ {20(p=0.8), 15(p=0.2)}, 평균 19.
  E[(E[X|M])^2] = 0.8·400 + 0.2·225 = 320 + 45 = 365 → Var = 365 − 19^2 = 365 − 361 = 4.
- Var(X̂) = 4 + 4 = 8.

Var(Ŷ) = E[Var(Y|M)] + Var(E[Y|M]).
- E[Var(Y|M)] = 0.8·(5^2) + 0.2·(5^2) = 25.
- Var(E[Y|M]): E[Y|M] ∈ {75(p=0.8), 50(p=0.2)}, 평균 70.
  E[(E[Y|M])^2] = 0.8·5625 + 0.2·2500 = 4500 + 500 = 5000 → Var = 5000 − 70^2 = 5000 − 4900 = 100.
- Var(Ŷ) = 25 + 100 = 125.

### 상관계수 (Lecture08 p.6 의 ρ 정의)
ρ(X̂, Ŷ) = Cov(X̂,Ŷ) / √(Var(X̂)·Var(Ŷ)) = 14 / √(8·125) = 14 / √1000 = 14/(10√10) ≈ 0.4427.

따라서 ρ(X̂, Ŷ) ≈ +0.44.

### 설명
혼합 집단에서 속도와 체중은 양(+0.44)의 상관을 가진다 — 각 성별 내부의 음(−0.6)의 상관과 정반대 부호다. 이유: 남성은 (빠르고 무겁고), 여성은 (느리고 가볍다). 성별을 무시하고 섞으면 "빠른 사람=무거운 사람(남성)" 경향이 그룹 내의 "마른 사람이 빠르다" 경향을 압도한다. 따라서 남녀 차이를 무시하면 안 된다(Simpson의 역설).

(이 문제는 새 PMF/PDF를 정의하지 않는다. 혼합분포의 PDF는 0.8·f_남 + 0.2·f_여 형태로 전 평면에서 양의 밀도를 가지므로 "otherwise = 0" 구간이 없다.)

## 5. 검산·직관
- Var(X̂)=8 > 그룹 내 분산 4 ✓ (그룹 간 평균 차이가 분산을 키움 — LTV의 두 번째 항).
- 두 방법(곱의 기댓값 vs LTV 공분산판)으로 Cov=14 일치 ✓.
- |ρ| = 0.44 ≤ 1 ✓ (유효 상관계수).
- 부호 반전이 핵심 — "각 그룹 −0.6 이니 전체도 음수겠지"라는 직관이 틀림을 보임. 그룹 평균 간 공분산(+20)이 그룹 내(−6)를 압도 ✓.

## 6. 한 줄 요약
> 혼합 집단의 상관은 "그룹 내 공분산 + 그룹 간 평균 공분산"으로 분해되며, 남녀 평균이 같은 방향(빠름·무거움 vs 느림·가벼움)이라 ρ가 −0.6에서 +0.44로 뒤집힌다 — Simpson의 역설.
