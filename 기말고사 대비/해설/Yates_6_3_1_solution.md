> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X가 다음 CDF를 가진다.
F_X(x) = 0,           x < −1
F_X(x) = x/3 + 1/3,   −1 ≤ x < 0
F_X(x) = x/3 + 2/3,    0 ≤ x < 1
F_X(x) = 1,            1 ≤ x

Y = g(X), 여기서
g(X) = 0,     X < 0
g(X) = 100,   X ≥ 0

(a) F_Y(y)는?  (b) f_Y(y)는?  (c) E[Y]는?

## 1. 문제 정리
- 주어진 것: X의 CDF(혼합형 — 연속 구간 + 점프), 변환 g(계단함수).
- 구할 것: (a) Y의 CDF, (b) Y의 PDF/PMF, (c) E[Y].

## 2. 무엇을 묻고 왜 이 도구인가
파생분포의 대원칙: **무조건 CDF부터 계산하고 미분** (Lecture07, p.10; STT 260518, "cdf부터 계산한다"). 그런데 g가 두 값 {0, 100}만 취하므로 Y는 **이산** 확률변수다. 이때 핵심은 g의 분기 경계 X=0에서 P(X<0), P(X≥0)를 X의 CDF로부터 정확히 읽는 것 — 특히 X가 x=0에서 **점프(점질량)** 를 가지므로 부등호(< vs ≤)를 엄밀히 다뤄야 한다.

## 3. 핵심 통찰
F_X는 x=0에서 좌극한 1/3, 값 2/3으로 **크기 1/3의 점프**가 있다 → X에 P(X=0)=1/3 점질량이 박혀 있다. g(X)는 X≥0을 100으로 보내므로 이 점질량은 Y=100 쪽에 포함된다. 따라서 P(Y=0)=P(X<0)=F_X(0⁻)=1/3, P(Y=100)=P(X≥0)=2/3.

## 4. 풀이

(준비) X의 구조 파악 — x=0에서의 점프:
F_X(0⁻) = lim_{x→0⁻} (x/3 + 1/3) = 1/3
F_X(0)  = 0/3 + 2/3 = 2/3
점프 크기 = 2/3 − 1/3 = 1/3 = P(X = 0).  (x=−1, x=1에서는 좌극한=값이라 점프 없음)

두 값의 확률:
P(Y = 0)   = P(X < 0)  = F_X(0⁻) = 1/3
P(Y = 100) = P(X ≥ 0)  = 1 − P(X < 0) = 1 − 1/3 = 2/3

(a) Y의 CDF — Y는 {0, 100}만 취하는 이산형:
F_Y(y) = 0,     y < 0
F_Y(y) = 1/3,   0 ≤ y < 100
F_Y(y) = 1,     100 ≤ y

(b) Y의 PDF — Y가 이산이므로 PMF로 주는 것이 자연스럽다:
P_Y(0) = 1/3,   P_Y(100) = 2/3,   P_Y(y) = 0   otherwise
임펄스(델타)로 쓴 PDF 형태:
f_Y(y) = (1/3)·δ(y) + (2/3)·δ(y − 100),   그 외 0

(c) 기댓값:
E[Y] = 0·(1/3) + 100·(2/3) = 200/3 ≈ 66.67

## 5. 검산·직관
- 확률 합: P(Y=0) + P(Y=100) = 1/3 + 2/3 = 1 ✓.
- F_Y 단조비감소, 우연속, F_Y(−∞)=0, F_Y(+∞)=1 ✓.
- 부등호 점검: g는 X≥0 → 100이므로 x=0의 점질량(1/3)은 반드시 Y=100에 귀속. 만약 P(X≤0)으로 잘못 읽으면 2/3을 Y=0에 줘서 답이 뒤집힌다 — 혼합형 CDF에서 < 와 ≤ 구분이 결정적.
- E[Y] = 200/3 ≈ 66.7 로 100 쪽(확률 2/3)에 더 가까움 → 직관 일치.

## 6. 한 줄 요약
> g가 유한개 값을 취하면 Y는 이산 — CDF의 점프로 P(X=0)을 읽고, g의 분기 경계에서 < / ≤ 를 엄밀히 적용해 각 값의 확률을 모은다.
