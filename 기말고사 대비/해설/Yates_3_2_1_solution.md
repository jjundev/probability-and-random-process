## 문제
확률변수 N이 다음 PMF를 갖는다.

P_N(n) = c·(1/2)^n,   n = 0, 1, 2
P_N(n) = 0,   otherwise

(a) 상수 c의 값은?
(b) P[N ≤ 1] 은?

## 1. 문제 정리
- 주어진 것: P_N(n) = c(1/2)^n on n ∈ {0,1,2}, 0 otherwise.
- 구할 것: (a) 정규화 상수 c, (b) P[N ≤ 1].

## 2. 무엇을 묻고 왜 이 도구인가
PMF 정규화 공리. 모든 PMF는 Σ_n P_N(n) = 1 을 만족해야 하므로 (Lecture03, p.20의 결합/PMF 합=1 원칙; 단일 변수도 동일), 이 한 조건이 c를 결정한다. (b)는 PMF를 해당 값에서 더하는 단순 합.

## 3. 핵심 통찰
"PMF의 총합 = 1"이 미지수 c를 푸는 방정식이다. 유한한 세 항 c(1 + 1/2 + 1/4)을 1로 놓으면 끝.

## 4. 풀이

(a) 정규화:
Σ_(n=0)^2 c(1/2)^n = c[(1/2)^0 + (1/2)^1 + (1/2)^2] = c[1 + 1/2 + 1/4] = c·(7/4) = 1.
⇒ c = 4/7.

따라서 완전한 PMF:
P_N(0) = 4/7, P_N(1) = (4/7)(1/2) = 2/7, P_N(2) = (4/7)(1/4) = 1/7, otherwise = 0.
(합 4/7 + 2/7 + 1/7 = 7/7 = 1 ✓)

(b) P[N ≤ 1]:
P[N ≤ 1] = P_N(0) + P_N(1) = 4/7 + 2/7 = 6/7.
(또는 = c(1 + 1/2) = (4/7)(3/2) = 6/7.)

## 5. 검산·직관
- 각 P_N(n) ∈ [0,1] ✓ (4/7, 2/7, 1/7 모두 양수, 합 1).
- 여사건 점검: P[N ≤ 1] = 1 − P[N = 2] = 1 − 1/7 = 6/7 ✓.
- 직관: 확률이 n에 따라 4:2:1 로 기하적으로 감소 — 작은 값일수록 무겁고, N=2가 가장 드묾.

## 6. 한 줄 요약
> PMF 총합=1이 정규화 상수를 결정한다: c(1+1/2+1/4)=1 ⇒ c=4/7, 그리고 P[N≤1]=4/7+2/7=6/7.
