## 문제
원판 X² + Y² ≤ r² 위에서 확률변수 X와 Y가 균등(uniform) PDF를 가진다.
f_{X,Y}(x,y) = 1/(πr²),   x² + y² ≤ r²
f_{X,Y}(x,y) = 0,         otherwise

(a) f_{Y|X}(y|x)는?
(b) E[Y|X = x]는?

## 1. 문제 정리
- 주어진 것: 반지름 r 원판 위 균등 결합 PDF, 그 외 0.
- 구할 것: (a) 조건부 PDF f_{Y|X}(y|x), (b) 조건부 기댓값 E[Y|X=x].

## 2. 무엇을 묻고 왜 이 도구인가
조건부 PDF의 정의 f_{Y|X}(y|x) = f_{X,Y}(x,y) / f_X(x), 단 f_X(x) > 0 (Lecture06, p.18) — 1순위 강의노트(Lecture07)는 covariance 위주라 조건부 PDF 정의가 없어, 그 공식이 실제로 정리된 Lecture06으로 한 단계 보강한다. 분모의 주변 PDF는 f_X(x) = ∫ f_{X,Y}(x,y) dy (Lecture06, p.11). 균등분포의 기하 덕분에 적분이 "x에서의 세로 현(chord) 길이"로 환원된다.

## 3. 핵심 통찰
균등 결합밀도에서 X=x로 자른 단면은 y ∈ [−√(r²−x²), √(r²−x²)] 위에서 **상수**다. 상수 밀도를 적분 1로 정규화하면 그 단면은 곧 **그 구간 위의 균등분포** — 그래서 조건부 PDF는 현의 길이의 역수, 조건부 기댓값은 구간의 중점 0이 된다.

## 4. 풀이

(준비) 주변 PDF f_X(x). 고정된 x(|x| ≤ r)에서 (x,y)가 원판 안이려면 y² ≤ r² − x², 즉 |y| ≤ √(r²−x²):
f_X(x) = ∫_{−√(r²−x²)}^{√(r²−x²)} 1/(πr²) dy = (1/(πr²))·2√(r²−x²) = 2√(r²−x²)/(πr²),   −r ≤ x ≤ r
f_X(x) = 0,   otherwise

(a) 조건부 PDF (Lecture06, p.18). −r < x < r 에서 (분모 > 0):
f_{Y|X}(y|x) = f_{X,Y}(x,y) / f_X(x) = [1/(πr²)] / [2√(r²−x²)/(πr²)] = 1 / (2√(r²−x²)),   −√(r²−x²) ≤ y ≤ √(r²−x²)
f_{Y|X}(y|x) = 0,   otherwise
즉 Y | X=x ~ Uniform[ −√(r²−x²), √(r²−x²) ].

(b) 조건부 기댓값. 구간 [−a, a] (a = √(r²−x²)) 위 균등분포의 평균은 중점:
E[Y|X = x] = ∫_{−a}^{a} y · 1/(2a) dy = (1/(2a))·[y²/2]_{−a}^{a} = (1/(2a))·(a²/2 − a²/2) = 0
E[Y|X = x] = 0,   −r < x < r

## 5. 검산·직관
- 정규화: ∫_{−a}^{a} 1/(2a) dy = (2a)/(2a) = 1 ✓ (조건부 PDF는 항상 적분 1).
- 분모 양수 조건: |x| = r 에서 √(r²−x²) = 0 이라 f_X = 0 → 조건부 PDF 정의 안 됨 (확률 0 사건). −r < x < r 에서만 유효 (Lecture06, p.18 "f_Y(y)>0 필수").
- E[Y|X=x] = 0 은 원판이 x축에 대칭이라 단면이 0을 중심으로 대칭 → 직관 일치.
- f_{Y|X}는 y에 의존하지 않는 상수(x에만 의존) → 단면이 균등임을 재확인 ✓.

## 6. 한 줄 요약
> 균등 결합밀도에서 X=x 단면은 그 세로 현 위의 균등분포 — f_{Y|X} = f_{X,Y}/f_X = 1/(현 길이), E[Y|X=x]는 현의 중점 0이다.
