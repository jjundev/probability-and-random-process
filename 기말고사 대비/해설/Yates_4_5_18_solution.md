# Yates 4.5.18 풀이
> 출처: Probability and Stochastic Processes (Yates & Goodman), L05_PSP_4.5.18.png, 1순위 근거 Lecture05, 풀이일 2026-06-08
> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
이 문제에서 Theorem 4.11 의 증명을 개략한다.

(a) X_n 을 Erlang(n, λ) 확률변수라 하자. Erlang PDF 의 정의를 이용해, 임의의 x ≥ 0 에 대해
    F_(X_n)(x) = ∫_0^x [ λ^n · t^(n−1) · e^(−λt) / (n−1)! ] dt
임을 보여라.
(b) 이 적분에 부분적분(Appendix B, Math Fact B.10)을 적용해, x ≥ 0 에 대해
    F_(X_n)(x) = F_(X_(n−1))(x) − (λx)^(n−1) · e^(−λx) / (n−1)!
임을 보여라.
(c) F_(X_1)(x) = 1 − e^(−λx) (x ≥ 0) 임을 이용해 Theorem 4.11 의 주장을 검증하라.

(Theorem 4.11: Erlang(n, λ) 의 CDF 는 x < 0 이면 0, x ≥ 0 이면
 F_(X_n)(x) = 1 − Σ_{k=0}^{n−1} (λx)^k · e^(−λx) / k! .)

## 1. 문제 정리
주어진 것: Erlang(n, λ) PDF (Lecture05 정리노트에는 Erlang 항목이 없어 일반 교과서 정의로 보강 — 아래 명시)
  f_(X_n)(t) = λ^n · t^(n−1) · e^(−λt) / (n−1)!,  t ≥ 0
  f_(X_n)(t) = 0,                                otherwise (t < 0)
구할 것: (a) CDF 의 적분 표현, (b) 부분적분으로 점화식(recursion), (c) 귀납적으로 닫힌형 CDF 검증.

⚠ 강의 밖 보강: Erlang 분포(= n 개 독립 Exp(λ) 의 합)는 Lecture05 정리노트에 직접 등장하지 않는다. 다만 Lecture05 의 핵심 도구 두 가지 — CDF = ∫ PDF (p.9) 와 지수 PDF/CDF (p.13–14), 그리고 "지수 적분은 부분적분으로" (STT 260413, "이거는 인간적으로 손으로 합시다") — 가 그대로 증명의 엔진이다.

## 2. 무엇을 묻고 왜 이 도구인가
PDF → CDF 적분을 점화식으로 푸는 증명 문제. 핵심 도구는 (i) CDF 정의 F_X(x) = ∫_(−∞)^x f_X(t) dt (Lecture05, p.9), (ii) 부분적분 ∫ u dv = uv − ∫ v du. Erlang 의 t^(n−1) e^(−λt) 꼴은 다항×지수라, 다항을 미분해 차수를 낮추는 부분적분이 자연스럽다(Lecture05 가 지수 모멘트 유도에서 쓴 바로 그 기법, p.14).

## 3. 핵심 통찰
"부분적분 한 번이 Erlang 차수 n 을 n−1 로 내린다" — 그러면 F_(X_n) 이 F_(X_(n−1)) 로 표현되고, base case F_(X_1) = 1 − e^(−λx) 에서 위로 쌓아 닫힌형이 나온다.

## 4. 풀이

(a) CDF 의 적분 표현
연속 RV 의 CDF 정의 F_(X_n)(x) = ∫_(−∞)^x f_(X_n)(t) dt (Lecture05, p.9). f_(X_n)(t) = 0 (t < 0) 이므로 하한이 0 으로 바뀐다.
F_(X_n)(x) = ∫_0^x [ λ^n · t^(n−1) · e^(−λt) / (n−1)! ] dt,  x ≥ 0.
(x < 0 이면 적분 구간이 비어 F = 0.) — 정의 그대로의 결과.

(b) 부분적분으로 점화식
적분 I = ∫_0^x λ^n t^(n−1) e^(−λt) / (n−1)! dt 에 부분적분.
u = t^(n−1),        du = (n−1) t^(n−2) dt
dv = λ^n e^(−λt) / (n−1)! dt,   v = −λ^(n−1) e^(−λt) / (n−1)!
(v 확인: dv/dt = −λ^(n−1)·(−λ)e^(−λt)/(n−1)! = λ^n e^(−λt)/(n−1)! ✓.)

∫ u dv = [u v]_0^x − ∫_0^x v du:
[u v]_0^x = [ −t^(n−1) λ^(n−1) e^(−λt) / (n−1)! ]_0^x
         = −x^(n−1) λ^(n−1) e^(−λx) / (n−1)! − 0
         = −(λx)^(n−1) e^(−λx) / (n−1)!.
(t=0 항은 n ≥ 2 일 때 t^(n−1)=0 으로 소멸.)

−∫_0^x v du = −∫_0^x [ −λ^(n−1) e^(−λt) / (n−1)! ]·(n−1) t^(n−2) dt
           = ∫_0^x λ^(n−1) (n−1) t^(n−2) e^(−λt) / (n−1)! dt
           = ∫_0^x λ^(n−1) t^(n−2) e^(−λt) / (n−2)! dt
(여기서 (n−1)/(n−1)! = 1/(n−2)! 사용).
이 적분은 정확히 F_(X_(n−1))(x) 의 정의식 [ (a) 에서 n→n−1 ] 이다.

따라서
F_(X_n)(x) = F_(X_(n−1))(x) − (λx)^(n−1) e^(−λx) / (n−1)!,  x ≥ 0.
부분적분 한 번이 차수를 n 에서 n−1 로 내렸다.

(c) 귀납으로 닫힌형 검증
주장(Theorem 4.11): F_(X_n)(x) = 1 − Σ_{k=0}^{n−1} (λx)^k e^(−λx) / k!  (x ≥ 0).

Base (n=1): 주어진 F_(X_1)(x) = 1 − e^(−λx). 주장식의 n=1 은 1 − Σ_{k=0}^0 (λx)^k e^(−λx)/k! = 1 − (λx)^0 e^(−λx)/0! = 1 − e^(−λx) ✓.

귀납 단계: F_(X_(n−1))(x) = 1 − Σ_{k=0}^{n−2} (λx)^k e^(−λx)/k! 가 성립한다고 가정. (b) 의 점화식에 대입:
F_(X_n)(x) = [ 1 − Σ_{k=0}^{n−2} (λx)^k e^(−λx)/k! ] − (λx)^(n−1) e^(−λx)/(n−1)!.
마지막 항이 합의 k = n−1 항이므로 두 항을 합치면 합의 상한이 n−1 로 확장된다:
F_(X_n)(x) = 1 − Σ_{k=0}^{n−1} (λx)^k e^(−λx)/k!.
이는 주장식 그대로 — 수학적 귀납법으로 모든 n ≥ 1, x ≥ 0 에서 성립. ∎

## 5. 검산·직관
- n=1 에서 Erlang = Exponential(λ) 로 환원, CDF 1 − e^(−λx) 와 일치 (Lecture05, p.13) ✓.
- 닫힌형의 합 부분 Σ_{k=0}^{n−1} (λx)^k e^(−λx)/k! 는 평균 λx 인 Poisson 의 P(N ≤ n−1) 이다. 의미: "시간 x 동안 도착이 n−1 회 이하면 n 번째 사건(Erlang 대기시간)은 아직 안 옴", 즉 F_(X_n)(x) = 1 − P(N ≤ n−1) = P(N ≥ n) — Poisson↔Erlang 쌍대성, 직관적으로 타당.
- x → ∞ 에서 e^(−λx)·다항 → 0 이므로 F → 1 ✓; x = 0 에서 F = 1 − e^0·1 = 0 ✓.

## 6. 한 줄 요약
> Erlang CDF 는 PDF 를 적분한 뒤 부분적분으로 차수를 한 칸씩 내리는 점화식(F_n = F_(n−1) − (λx)^(n−1)e^(−λx)/(n−1)!)을 세우고, base F_1 = 1 − e^(−λx) 에서 귀납으로 닫힌형을 얻는다.
