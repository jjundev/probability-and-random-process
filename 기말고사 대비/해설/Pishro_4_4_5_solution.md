> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X가 다음 PDF를 갖는 연속 확률변수라 하자.

f_X(x) = (5/32)·x^4,   0 < x ≤ 2
f_X(x) = 0,   otherwise

그리고 Y = X² 이라 하자.
a. Y의 CDF를 구하라.
b. Y의 PDF를 구하라.
c. E[Y] 를 구하라.

## 1. 문제 정리
- 주어진 것: f_X(x) = (5/32)x^4 on (0,2], 0 otherwise. Y = X².
- 구할 것: F_Y(y), f_Y(y), E[Y].
- support: X ∈ (0, 2] ⇒ Y = X² ∈ (0, 4]. (X가 양수만 취하므로 X² ↔ X 가 일대일.)

## 2. 무엇을 묻고 왜 이 도구인가
파생분포 문제. 원칙대로 **CDF부터 구하고 미분한다.** (Lecture07, p.10) 일반적으로 Y = X² 는 F_Y(y) = F_X(√y) − F_X(−√y) 형태지만 (Lecture07, p.2.4 / p.10), 여기서 X는 양수 구간 (0,2]에만 분포하므로 −√y 쪽 항은 0이 되어 한쪽만 남는다.

## 3. 핵심 통찰
X > 0 이 보장되므로 "X² ≤ y" ⇔ "X ≤ √y" (음의 근은 X의 support 밖이라 버림). 양수 RV의 제곱은 단조변환처럼 다뤄진다.

## 4. 풀이

먼저 X의 CDF가 필요하다. 0 < x ≤ 2 에서

F_X(x) = ∫_0^x (5/32)t^4 dt = (5/32)·(x^5/5) = x^5/32.

(검산: F_X(2) = 32/32 = 1 ✓.)

(a) CDF. 0 < y ≤ 4 에서, √y ∈ (0, 2] 이므로

F_Y(y) = P(X² ≤ y) = P(X ≤ √y) = F_X(√y) = (√y)^5 / 32 = y^(5/2) / 32.

정리하면

F_Y(y) = 0,   y ≤ 0
F_Y(y) = y^(5/2)/32,   0 < y ≤ 4
F_Y(y) = 1,   y > 4

(경계: F_Y(4) = 4^(5/2)/32 = 32/32 = 1 ✓.)

(b) PDF. d/dy [y^(5/2)/32] = (1/32)·(5/2)y^(3/2) = (5/64)y^(3/2). 따라서

f_Y(y) = (5/64)·y^(3/2),   0 < y ≤ 4
f_Y(y) = 0,   otherwise

(c) E[Y]. LOTUS로 X에 대해 적분하는 편이 가장 간단하다:

E[Y] = E[X²] = ∫_0^2 x² · (5/32)x^4 dx = (5/32) ∫_0^2 x^6 dx = (5/32)·[x^7/7]_0^2 = (5/32)·(128/7) = 640/224 = 20/7.

수치로 약 2.857.

(검산용 PDF 적분: E[Y] = ∫_0^4 y·(5/64)y^(3/2) dy = (5/64)∫_0^4 y^(5/2) dy = (5/64)·[ (2/7) y^(7/2) ]_0^4 = (5/64)·(2/7)·4^(7/2) = (5/64)·(2/7)·128 = 1280/448 = 20/7 ✓ 동일.)

## 5. 검산·직관
- 정규화: ∫_0^4 (5/64)y^(3/2) dy = (5/64)·(2/5)·y^(5/2) |_0^4 = (1/32)·4^(5/2) = 32/32 = 1 ✓.
- support: Y ∈ (0,4]. E[Y] = 20/7 ≈ 2.86 < 4 이고 분포가 큰 값(4 근처)으로 쏠리는 f_Y ∝ y^(3/2) 형태와 일관 ✓.
- 직관: 원래 f_X ∝ x^4 가 x=2 근처에 질량을 몰아주므로 X²도 4 근처가 무겁다.

## 6. 한 줄 요약
> 양수 RV의 제곱 Y = X² 는 음의 근이 support 밖이라 "X² ≤ y ⇔ X ≤ √y" 한쪽만 남으며, 기댓값은 LOTUS로 E[X²]를 직접 적분하면 빠르다.
