## 문제
어떤 실험에서 A와 B는 서로 배반(mutually exclusive)인 사건이고 P[A] = 1/4, P[B] = 1/8이다.
(a) P[A∩B], P[A∪B], P[A∩Bᶜ], P[A∪Bᶜ]을 구하라.
(b) A와 B는 독립인가?

## 1. 문제 정리
주어진 것: A, B 배반(A∩B = ∅), P[A]=1/4, P[B]=1/8.
구할 것: (a) 네 가지 확률, (b) 독립 여부 판정.

## 2. 무엇을 묻고 왜 이 도구인가
"배반(disjoint)"과 "독립(independent)"의 차이를 시험하는 전형 문제다. 도구는 두 가지:
- 합집합 공식 P[A∪B] = P[A] + P[B] − P[A∩B].
- 독립 정의 A ⊥ B ⇔ P[A∩B] = P[A]·P[B]. (Lecture02, p.19)

## 3. 핵심 통찰
A∩B = ∅이면 A는 통째로 Bᶜ 안에 들어간다(A ⊆ Bᶜ). 이 한 가지 포함관계가 A∩Bᶜ = A, A∪Bᶜ = Bᶜ를 즉시 준다 — 따로 적분/계산이 필요 없다.

## 4. 풀이

**(a)**

배반이므로:
P[A∩B] = P[∅] = 0

P[A∪B] = P[A] + P[B] − P[A∩B] = 1/4 + 1/8 − 0 = 2/8 + 1/8 = 3/8

A ⊆ Bᶜ (A가 B와 겹치지 않으므로) → A∩Bᶜ = A:
P[A∩Bᶜ] = P[A] = 1/4

A ⊆ Bᶜ → A∪Bᶜ = Bᶜ:
P[A∪Bᶜ] = P[Bᶜ] = 1 − P[B] = 1 − 1/8 = 7/8

(교차검증: P[A∪Bᶜ] = P[A] + P[Bᶜ] − P[A∩Bᶜ] = 1/4 + 7/8 − 1/4 = 7/8 ✓)

**(b) 독립 여부**

P[A∩B] = 0,   P[A]·P[B] = (1/4)·(1/8) = 1/32

0 ≠ 1/32 이므로 P[A∩B] ≠ P[A]P[B] → **A와 B는 독립이 아니다.**

## 5. 검산·직관
- "배반 ≠ 독립" — 오히려 정반대다. A가 일어나면 B는 절대 못 일어나므로(P[B|A]=0 ≠ P[B]=1/8) 강하게 의존한다. (Lecture02, p.19, p.82) ✓
- 두 사건의 확률이 모두 양수면 배반은 결코 독립일 수 없다(P[A∩B]=0인데 P[A]P[B]>0). 일반 규칙과 일치 ✓.
- P[A∪B]=3/8 ≤ 1, 각 사건 확률보다 큼 ✓.

## 6. 한 줄 요약
> 배반이면 P[A∩B]=0이고 A⊆Bᶜ라 나머지 확률이 술술 나오지만, 바로 그 P[A∩B]=0 ≠ P[A]P[B] 때문에 배반은 독립이 아니다.
