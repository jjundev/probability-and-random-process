## 문제
Problem 7.5.3의 확률변수 A, B에 대해, U = E[B|A] 로 둔다. PMF P_U(u) 를 구하라. 그리고 E[U] = E[E[B|A]] 는 얼마인가?

(Problem 7.5.3의 모델)
A의 PMF:
P_A(a) = 1/3,  a = −1 ;  P_A(a) = 2/3,  a = 1 ;  P_A(a) = 0, otherwise.
조건부 B|A:
P_{B|A}(b|−1) = 1/3, b=0 ;  2/3, b=1 ;  0, otherwise.
P_{B|A}(b|1)  = 1/2, b=0 ;  1/2, b=1 ;  0, otherwise.

## 1. 문제 정리
주어진 것:
- A ∈ {−1, 1}, P(A=−1)=1/3, P(A=1)=2/3.
- B|A=−1: P(B=1)=2/3 → E[B|A=−1] = 2/3.
- B|A=1: P(B=1)=1/2 → E[B|A=1] = 1/2.

구할 것:
- U = E[B|A] 의 PMF P_U(u)
- E[U] = E[E[B|A]] (반복기댓값)

## 2. 무엇을 묻고 왜 이 도구인가
정리노트의 결정적 구분: E[B|A]는 숫자가 아니라 A의 함수인 확률변수다 — A=a일 때 값 g(a)=E[B|A=a]를 갖는다 (Lecture08, p.16 "E[X|Y]는 RV, E[X|Y=y]는 숫자"). 따라서 U=E[B|A]는 A가 두 값을 가지면 그에 대응해 두 값을 갖는 이산 RV이고, 그 PMF는 A의 PMF를 그대로 물려받는다. E[U]는 반복기댓값 법칙 E[E[B|A]] = E[B] (Lecture08, p.17)로 곧장 나온다.

## 3. 핵심 통찰
U는 A의 함수. A=−1 ⟹ U=2/3, A=1 ⟹ U=1/2. 그러니 U의 PMF는 "A가 −1·1을 갖던 확률"을 "U가 2/3·1/2를 갖는 확률"로 옮겨 적은 것뿐이다. E[U]는 그냥 E[B].

## 4. 풀이

### 각 조건부 기댓값 (U가 갖는 값)
E[B|A=−1] = 0·(1/3) + 1·(2/3) = 2/3.
E[B|A=1]  = 0·(1/2) + 1·(1/2) = 1/2.

따라서 U = E[B|A] 는:
- A = −1 (확률 1/3) 일 때 U = 2/3
- A = 1  (확률 2/3) 일 때 U = 1/2

### PMF P_U(u)
U의 두 값은 A의 두 값에 일대일 대응하므로 확률을 그대로 옮긴다:
P_U(u) = 1/3,   u = 2/3
P_U(u) = 2/3,   u = 1/2
P_U(u) = 0,     otherwise.

### E[U] = E[E[B|A]]
방법 1 — 직접 (U의 PMF로):
E[U] = (2/3)·(1/3) + (1/2)·(2/3) = 2/9 + 1/3 = 2/9 + 3/9 = 5/9.

방법 2 — 반복기댓값 법칙 (Lecture08, p.17: E[E[B|A]] = E[B]):
E[B] = Σ_a P_A(a)·E[B|A=a] = (1/3)·(2/3) + (2/3)·(1/2) = 2/9 + 1/3 = 5/9.

두 방법 일치: E[U] = E[E[B|A]] = E[B] = 5/9.

## 5. 검산·직관
- P_U 합 = 1/3 + 2/3 = 1 ✓.
- U의 값들이 [0,1] 안(2/3, 1/2): B∈{0,1}이라 조건부 기댓값은 [0,1] 범위여야 함 ✓.
- 반복기댓값으로 얻은 5/9가 직접계산과 같음 — 정리노트 핵심정리(p.17)의 작동 확인 ✓.
- 5/9 ≈ 0.556 ∈ [1/2, 2/3] — 두 조건부 평균 사이의 가중평균이라 그 사이에 있어야 함 ✓.

## 6. 한 줄 요약
> E[B|A]는 A의 함수인 RV이므로 그 PMF는 A의 PMF를 값만 바꿔 물려받고, 그 기댓값은 반복기댓값 법칙으로 곧 E[B] = 5/9.
