## 문제
기숙사에 50명의 학생이 산다. 주차장 수용 능력은 자동차 30대다. 각 학생은 다른 학생과 독립적으로 확률 1/2 로 자동차를 보유한다. CLT(연속성 보정 포함)를 사용하여, 모든 자동차를 댈 주차 공간이 부족할 확률을 구하라.

## 1. 문제 정리
- 주어진 것: X_i = i번째 학생의 차 보유 여부 ~ Bernoulli(p=1/2), i.i.d., n = 50.
  - E[X_i] = 1/2, Var(X_i) = p(1−p) = 1/4.
- Y = 총 자동차 수 = X_1 + … + X_50 ~ Binomial(50, 1/2).
- 구할 것: 주차 공간 부족 확률 = P(Y > 30) = P(Y ≥ 31). (보정 포함 요구)

## 2. 무엇을 묻고 왜 이 도구인가
i.i.d. 베르누이 50개의 합(이항)의 꼬리확률. 이항 꼬리를 직접 더할 수 있긴 하나, 문제가 명시적으로 CLT + 연속성 보정을 요구한다. 정규 근사로 이산 이항을 연속 가우시안으로 바꿀 때, "정수값 사건"을 ±0.5 폭의 구간으로 보정하면 근사가 더 정확해진다 — 이것이 연속성 보정의 동기다. (⚠ 강의 밖: 연속성 보정 자체는 Lecture09 정리노트에 명시 없음 — 일반 교과서 지식으로 보강. CLT 표준화 골격은 Lecture09, slide 9.)

## 3. 핵심 통찰
"공간 부족" ⇔ "차가 30대 초과" ⇔ Y ≥ 31. 이산 이항을 정규로 근사할 때 P(Y ≥ 31)은 연속변수 기준 P(Y > 30.5)로 보정한다.

## 4. 풀이
합의 평균·분산 (i.i.d. 단순 합) — (Lecture09, slide 4):

  E[Y] = np = 50 × 1/2 = 25
  Var(Y) = np(1−p) = 50 × 1/4 = 12.5
  σ(Y) = √12.5 ≈ 3.5355

사건 정리: 30대까지는 수용 가능, 31대 이상이면 부족.

  P(부족) = P(Y > 30) = P(Y ≥ 31)

연속성 보정(⚠ 강의 밖, 일반 지식): P(Y ≥ 31) ≈ P(Y_continuous > 30.5).

CLT 표준화 (Lecture09, slide 9):

  Z = (Y − 25)/3.5355  ≈  N(0,1)

  P(Y > 30.5) = P( Z > (30.5 − 25)/3.5355 )
              = P( Z > 5.5/3.5355 )
              = P( Z > 1.5556 )
              = 1 − Φ(1.5556)

표준정규 분포표에서 Φ(1.56) ≈ 0.9406 이므로

  P(부족) ≈ 1 − 0.9406 = 0.0594  ≈  0.06

(보정 없이 30으로 잡으면 P(Z > 5/3.5355 = 1.4142) = 1 − 0.9214 = 0.0786 — 문제가 보정을 요구하므로 0.0594가 답.)

## 5. 검산·직관
- 보정 방향: P(Y ≥ 31) 은 "31 이상"이므로 경계 30과 31의 중간인 30.5에서 자른다. 임계값이 보정 전(30)보다 커져 확률이 약간 작아짐(0.0786 → 0.0594) — 합리적.
- 직관: 평균 25대, 공간 30대 → 약 1.56σ 여유. 부족할 확률 약 6%로 가끔 발생. 학생 수 50의 절반이 평균이므로 30대 초과는 흔치 않음.
- 단위: 자동차 수(정수)의 표준화 → 무차원. ✓

## 6. 한 줄 요약
> 이항(50, 1/2) 차량 수의 부족 확률은 평균 25·분산 12.5로 표준화하되, 이산→연속 연속성 보정으로 30 대신 30.5에서 잘라 1 − Φ((30.5−25)/√12.5) ≈ 0.06 으로 구한다.
