## 문제
어느 ECE 2학년생의 키 X(가장 가까운 피트로 반올림)와 학점 Y(가장 가까운 정수로 반올림)는 다음 joint PMF를 갖는다.

P_(X,Y)(x,y):
            y=1     y=2     y=3     y=4
  x=5      0.05    0.1     0.2     0.05
  x=6      0.1     0.1     0.3     0.1

E[X+Y] 와 Var[X+Y] 를 구하라.

## 1. 문제 정리
- 주어진 것: 위 joint PMF (그 외 (x,y)에서 P = 0, otherwise = 0). X ∈ {5,6}, Y ∈ {1,2,3,4}.
- 구할 것: E[X+Y], Var[X+Y].

## 2. 무엇을 묻고 왜 이 도구인가
다중 RV의 합의 기댓값·분산. 도구:
- 기댓값 선형성 E[X+Y] = E[X] + E[Y] (독립 여부 무관, Lecture03, p.22).
- 분산은 E[X+Y]를 직접 못 쪼개므로 var[X+Y] = E[(X+Y)²] − (E[X+Y])² 로 가거나, var[X+Y] = var(X)+var(Y)+2cov(X,Y) (Lecture07, p.25)로 간다. **X,Y의 독립을 확인하지 않았으므로** 공분산을 포함한 정공법으로 푼다.

## 3. 핵심 통찰
주변 PMF만으로 E[X], E[Y]는 나오지만 분산에는 X,Y 의존성이 들어간다. 가장 안전한 길은 W = X+Y 의 PMF를 직접 만들거나, E[XY]까지 결합표로 계산해 cov를 넣는 것. 여기선 결합표로 E[X], E[Y], E[X²], E[Y²], E[XY]를 모두 구해 조립한다.

## 4. 풀이

먼저 주변 PMF.
- 행 합(X): P_X(5) = 0.05+0.1+0.2+0.05 = 0.40, P_X(6) = 0.1+0.1+0.3+0.1 = 0.60. (otherwise = 0)
- 열 합(Y): P_Y(1) = 0.05+0.1 = 0.15, P_Y(2) = 0.1+0.1 = 0.20, P_Y(3) = 0.2+0.3 = 0.50, P_Y(4) = 0.05+0.1 = 0.15. (otherwise = 0)
  (합 0.15+0.20+0.50+0.15 = 1 ✓)

E[X] = 5(0.40) + 6(0.60) = 2.0 + 3.6 = 5.6.
E[X²] = 25(0.40) + 36(0.60) = 10.0 + 21.6 = 31.6.
var(X) = 31.6 − 5.6² = 31.6 − 31.36 = 0.24.

E[Y] = 1(0.15) + 2(0.20) + 3(0.50) + 4(0.15) = 0.15 + 0.40 + 1.50 + 0.60 = 2.65.
E[Y²] = 1(0.15) + 4(0.20) + 9(0.50) + 16(0.15) = 0.15 + 0.80 + 4.50 + 2.40 = 7.85.
var(Y) = 7.85 − 2.65² = 7.85 − 7.0225 = 0.8275.

E[XY] = Σ Σ xy·P(x,y). 항별:
- x=5: 5·1·0.05 + 5·2·0.1 + 5·3·0.2 + 5·4·0.05 = 0.25 + 1.0 + 3.0 + 1.0 = 5.25.
- x=6: 6·1·0.1 + 6·2·0.1 + 6·3·0.3 + 6·4·0.1 = 0.6 + 1.2 + 5.4 + 2.4 = 9.6.
E[XY] = 5.25 + 9.6 = 14.85.

cov(X,Y) = E[XY] − E[X]E[Y] = 14.85 − (5.6)(2.65) = 14.85 − 14.84 = 0.01. (Lecture07, p.23)

조립:
E[X+Y] = E[X] + E[Y] = 5.6 + 2.65 = 8.25.
Var[X+Y] = var(X) + var(Y) + 2cov(X,Y) = 0.24 + 0.8275 + 2(0.01) = 0.24 + 0.8275 + 0.02 = 1.0875.

결과:
E[X+Y] = 8.25.
Var[X+Y] = 1.0875.

## 5. 검산·직관
- W = X+Y 의 PMF 직접 구성으로 교차검산. W 가능값과 확률:
  W=6: (5,1) → 0.05
  W=7: (5,2)+(6,1) → 0.1+0.1 = 0.20
  W=8: (5,3)+(6,2) → 0.2+0.1 = 0.30
  W=9: (5,4)+(6,3) → 0.05+0.3 = 0.35
  W=10: (6,4) → 0.10
  (합 0.05+0.20+0.30+0.35+0.10 = 1 ✓)
  E[W] = 6(0.05)+7(0.20)+8(0.30)+9(0.35)+10(0.10) = 0.30+1.40+2.40+3.15+1.00 = 8.25 ✓.
  E[W²] = 36(0.05)+49(0.20)+64(0.30)+81(0.35)+100(0.10) = 1.80+9.80+19.20+28.35+10.00 = 69.15.
  var(W) = 69.15 − 8.25² = 69.15 − 68.0625 = 1.0875 ✓ 정확히 일치.
- cov(X,Y) ≈ 0.01 로 거의 0 — X와 Y가 거의 무상관(독립에 가까움). 그래서 var(X)+var(Y) = 1.0675 에 2cov = 0.02 만 더해진다.

## 6. 한 줄 요약
> 합의 평균은 선형성으로 E[X]+E[Y]지만, 합의 분산은 var(X)+var(Y)+2cov(X,Y) 이므로 결합표에서 E[XY]까지 구해 공분산을 넣어야 한다(여기선 cov≈0.01).
