## 문제
어떤 기계가 한 작업을 처리하는 데 필요한 시간은 평균 E[X_i] = 10분, 분산 Var(X_i) = 2분² 인 확률변수다. 서로 다른 작업의 처리 시간은 독립이다. 이 기계가 7시간 동안 40개 이하의 작업을 처리할 확률을 구하라.

## 1. 문제 정리
- 주어진 것: X_i i.i.d., μ = 10분, σ² = 2분² (σ = √2), 작업 시간 독립.
- 구할 것: 7시간(= 420분) 동안 처리하는 작업 수가 40개 이하일 확률.
- n = 40, S₄₀ = X₁ + ⋯ + X₄₀ = 40개 작업의 총 처리 시간.

## 2. 무엇을 묻고 왜 이 도구인가
i.i.d. 합 S_n의 분포는 직접 구하기 어렵지만(컨볼루션 40회), 중심극한정리(CLT)로 표준화하면 정규분포로 근사된다 (Lecture09, slide 9). n = 40은 충분히 커서 CLT 근사가 타당하다. 핵심은 "7시간에 40개 이하 처리"라는 **개수 사건**을 "40개를 처리하는 데 걸리는 시간 사건"으로 번역하는 것.

## 3. 핵심 통찰
**7시간 동안 작업을 40개 이하로 처리한다 ⟺ 40개를 처리하는 데 7시간 이상 걸린다**, 즉 {7시간에 처리량 ≤ 40} ⟺ {S₄₀ ≥ 420분}. (40번째 작업이 420분 시점 또는 그 이후에 끝나야 420분 안에 40개를 못 넘긴다.) 이렇게 바꾸면 S₄₀에 CLT를 곧장 적용할 수 있다.

## 4. 풀이

합의 평균·분산 (i.i.d., Lecture09, slide 4·14):
E[S₄₀] = n·μ = 40·10 = 400 (분)
Var(S₄₀) = n·σ² = 40·2 = 80 → 표준편차 √80 = 4√5 ≈ 8.944 (분)

CLT 표준화 (Lecture09, slide 9):
Z = (S₄₀ − E[S₄₀]) / √(Var(S₄₀)) = (S₄₀ − 400) / √80,   Z →(분포) N(0,1)

구하는 확률:
P(7시간에 ≤ 40개) = P(S₄₀ ≥ 420)
 = P( Z ≥ (420 − 400)/√80 )
 = P( Z ≥ 20/√80 )
 = P( Z ≥ √5 )            (∵ 20/√80 = 20/(4√5) = 5/√5 = √5 ≈ 2.236)
 = 1 − Φ(√5)
 ≈ 1 − Φ(2.236)
 ≈ 1 − 0.9873
 ≈ 0.0127

## 5. 검산·직관
- 420분은 평균 400분보다 20분(= √5 ≈ 2.24 표준편차) 위 → 꼬리확률이 작아야 함. 0.0127(약 1.3%)로 작음 → 직관 일치.
- 단위 점검: 시간은 모두 분으로 통일(7시간 = 420분, μ = 10분) ✓.
- 부등호 방향: "≤ 40개"가 "S₄₀ ≥ 420"으로 뒤집힘에 주의 — 처리량이 적다는 건 시간이 더 걸린다는 뜻. 방향을 반대로 잡으면 0.987로 틀림.
- σ = √(nσ²) = √80 이지 n·σ가 아님(독립 합은 분산이 더해짐) ✓ (Lecture09, slide 14).

## 6. 한 줄 요약
> "시간 T 안에 처리량 ≤ n" ⟺ "S_n ≥ T"로 번역한 뒤, i.i.d. 합 S_n을 CLT로 표준화하면 P(S₄₀ ≥ 420) = 1 − Φ(√5) ≈ 0.0127.
