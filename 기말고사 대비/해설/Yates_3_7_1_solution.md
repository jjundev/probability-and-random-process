> 출처: Probability and Stochastic Processes (Yates & Goodman), 복습L03_PSP_3.7.1.png, 1순위 근거 Lecture03, 풀이일 2026-06-08

## 문제
n = 1 일째부터 매일 복권 한 장을 산다. 각 복권은 1 달러이고, 서로 독립으로 확률 0.1 로 당첨되어 5 달러로 교환된다(아니면 가치 없음). X_n 을 n 일 후의 순이익(net profit)이라 하자. E[X_n] 은?

## 1. 문제 정리
주어진 것:
- 매일 1 장 구매, 1 장 비용 = 1 달러 → n 일 총비용 = n 달러(확정).
- 각 복권 당첨금 G_i = 5 (확률 0.1), 0 (확률 0.9), 독립.
구할 것: 순이익 X_n = (총당첨금) − (총비용) 의 기댓값 E[X_n].

각 G_i 는 "5·Bernoulli(0.1)" 꼴. 총당첨금 W_n = Σ_{i=1}^n G_i. X_n = W_n − n.

## 2. 무엇을 묻고 왜 이 도구인가
기댓값 선형성으로 푸는 문제. 당첨금 합의 분포 전체는 필요 없고 평균만 구하면 되므로, E[Σ G_i] = Σ E[G_i] (독립 여부 무관, Lecture03, p.22) 와 E[aX+b] = aE[X]+b (선형성, Lecture03, p.16) 만 쓴다. 복잡한 RV(이익)를 단순 조각(하루 당첨금)으로 분해하는 전형.

## 3. 핵심 통찰
"하루 기대당첨 0.5 달러 < 하루 비용 1 달러" — 하루당 기대순이익이 −0.5 달러로 일정하니, n 일이면 E[X_n] = −0.5n. 선형성이 합을 한 장의 평균으로 환원한다.

## 4. 풀이
한 장의 당첨금 G_i 의 기댓값 (Lecture03, p.14 정의):
E[G_i] = 5·P(당첨) + 0·P(꽝) = 5·0.1 + 0·0.9 = 0.5 (달러).

총당첨금 W_n = Σ_{i=1}^n G_i 의 기댓값 (선형성, Lecture03, p.22, 독립 불필요):
E[W_n] = Σ_{i=1}^n E[G_i] = n·0.5 = 0.5n.

순이익 X_n = W_n − n (비용 n 은 상수). 선형성 E[aX+b]=aE[X]+b (Lecture03, p.16):
E[X_n] = E[W_n] − n = 0.5n − n = −0.5n.

즉 E[X_n] = −n/2 (달러).

## 5. 검산·직관
- 단위·부호: 하루 기대수입 0.5 달러, 지출 1 달러 → 하루 −0.5 달러 손실, n 일이면 −0.5n ✓ (음수 = 평균적으로 손해, 복권의 본질).
- 특수값 n=1: E[X_1] = −0.5. 직접 확인 — X_1 = 4(당첨: 5−1) 또는 −1(꽝). E = 4·0.1 + (−1)·0.9 = 0.4 − 0.9 = −0.5 ✓.
- 분산은 묻지 않았으나 평균만으로 충분 — 선형성은 독립 가정 없이도 성립하므로 안전 (Lecture03, p.100).

## 6. 한 줄 요약
> 순이익은 "총당첨금 − 총비용"이고, 기댓값 선형성으로 하루 평균(0.5−1=−0.5달러)을 n 배 하여 E[X_n] = −n/2.
