## 문제
데이터베이스에서 레코드 하나에 접근하는 대기시간(밀리초) W 는 연속균등 (0,10) 랜덤변수다. 읽기시간 R(디스크에서 주기억장치로 정보를 옮기는 시간)은 3밀리초다. 랜덤변수 X(밀리초)는 디스크에서 정보 블록 하나를 얻는 데 드는 총 접근시간(대기시간 + 읽기시간)이다. 어떤 작업을 수행하기 전, 컴퓨터는 디스크에서 12개의 서로 다른 블록에 접근해야 한다(서로 다른 블록의 접근시간은 독립). 모든 정보의 총 접근시간은 랜덤변수 A(밀리초)다.
(a) E[X] 는?
(b) Var[X] 는?
(c) E[A] 는?
(d) σ_A 는?
(e) CLT를 사용해 P[A > 116ms] 를 추정하라.
(f) CLT를 사용해 P[A < 86ms] 를 추정하라.

## 1. 문제 정리
| 항목 | 내용 |
|---|---|
| W | Uniform(0, 10), E[W] = 5, Var(W) = (10−0)^2/12 = 100/12 ≈ 8.333 |
| R | 상수 3 |
| X | X = W + 3 (블록 1개 접근시간) |
| A | A = X_1 + … + X_12, 12개 i.i.d. 블록의 합 |

균등 (0,10)의 PDF:

  f_W(w) = 1/10,   0 ≤ w ≤ 10
  f_W(w) = 0,   otherwise

## 2. 무엇을 묻고 왜 이 도구인가
(a)-(d)는 합 S_n = A 의 평균·분산을 구하는 준비단계(상수 더하기는 평균만 이동·분산 불변), (e)(f)는 12개 i.i.d. 합의 꼬리/하단 확률을 CLT로 근사. 균등 12개의 합은 컨볼루션이 매우 번거로우므로(Lecture09, slide 5), 표준화해 가우시안으로 근사하는 것이 자연스럽다. (Lecture09, slide 9)

## 3. 핵심 통찰
A = ΣX_i 는 평균 nμ_X, 분산 nσ_X^2 을 가지며 (Lecture09, slide 4), 표준화 (A − nμ_X)/(σ_X√n) 이 N(0,1)로 수렴(Lecture09, slide 9). 상수 R=3은 분산을 바꾸지 않고 평균만 +3 한다.

## 4. 풀이

(a) E[X]. 상수를 더하면 평균은 그만큼 이동:

  E[X] = E[W] + 3 = 5 + 3 = 8 ms

(b) Var[X]. 상수를 더해도 분산은 불변. 균등 (0,10)의 분산:

  Var[X] = Var(W) = (10−0)^2/12 = 100/12 ≈ 8.333 ms^2

(c) E[A]. 합의 평균 = nμ (i.i.d. 단순 합) — (Lecture09, slide 4):

  E[A] = 12 × E[X] = 12 × 8 = 96 ms

(d) σ_A. 합의 분산 = nσ^2 (i.i.d.) — (Lecture09, slide 4):

  Var[A] = 12 × Var[X] = 12 × (100/12) = 100 ms^2
  σ_A = √100 = 10 ms

(e) CLT로 P[A > 116] (Lecture09, slide 9):

  Z = (A − 96)/10  ≈  N(0,1)

  P[A > 116] = P( Z > (116 − 96)/10 ) = P( Z > 2 ) = 1 − Φ(2)

  Φ(2) ≈ 0.9772  ⇒  P[A > 116] ≈ 1 − 0.9772 = 0.0228

(f) CLT로 P[A < 86]:

  P[A < 86] = P( Z < (86 − 96)/10 ) = P( Z < −1 ) = Φ(−1) = 1 − Φ(1)

  Φ(1) ≈ 0.8413  ⇒  P[A < 86] ≈ 1 − 0.8413 = 0.1587

## 5. 검산·직관
- (d) 깔끔한 σ_A = 10: Var(W)=100/12 에 n=12를 곱해 정확히 100이 되도록 설계된 문제 — 분모 12와 블록 수 12가 상쇄. 좋은 자기검산 신호. ✓
- (e)(f) 대칭성: 116은 평균 96에서 +2σ, 86은 −1σ. 우측 2σ 꼬리(0.0228)가 좌측 1σ 꼬리(0.1587)보다 훨씬 작다 — 정규분포 모양과 일치.
- 단위: 시간(ms) 합 → ms, 분산 → ms^2, 표준화 → 무차원. ✓

## 6. 한 줄 요약
> 균등접근시간 12개 합은 상수 R로 평균만 이동(E[A]=96)·분산은 nσ^2=100(σ_A=10)이며, 꼬리확률은 표준화 후 P[A>116]=1−Φ(2)=0.0228, P[A<86]=Φ(−1)=0.1587 로 근사한다.
