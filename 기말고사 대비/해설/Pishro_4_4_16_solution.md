## 문제
어떤 회사가 기기를 만든다. 기기 수명에 관심이 있다. 약 2% 의 기기는 처음부터 불량이라 수명이 0 년이다. 불량이 아니면 수명은 모수 λ = 2 (년) 의 지수분포를 따른다. X 를 무작위로 고른 기기의 수명이라 하자.

(a) X 의 일반화 PDF(generalized PDF) 를 구하라.
(b) P(X ≥ 1) 을 구하라.
(c) P(X > 2 | X ≥ 1) 을 구하라.
(d) EX 와 Var(X) 를 구하라.

## 1. 문제 정리
주어진 것:
- P(불량) = 0.02 → X = 0 (점질량).
- P(정상) = 0.98 → X | 정상 ~ Exponential(λ=2), 즉 f_(X|정상)(x) = 2e^(−2x) for x ≥ 0, otherwise = 0 (Lecture05, p.13).
구할 것: (a) 혼합형 X 의 일반화 PDF, (b)(c) 확률, (d) 평균·분산.

핵심 구조: X = 0 에 이산 질량 0.02 + (0,∞) 에 연속 밀도 0.98·(지수 PDF). 이는 전체 확률 정리(분할 = {불량, 정상})로 묶인 혼합 RV.

## 2. 무엇을 묻고 왜 이 도구인가
혼합 RV + 지수분포 문제. "확률 p 로 한 점에 박히고, 확률 1−p 로 연속분포" 구조는 분할 {불량, 정상} 에 대한 전체 확률 정리로 다룬다(연속판: f_X(x) = Σ_i P(A_i) f_(X|A_i)(x), Lecture06 형식과 동일). 점질량은 디랙 델타 δ 로 일반화 PDF 에 흡수한다.

지수분포 핵심 공식(Lecture05, p.13–14): CCDF P(X>x) = e^(−λx) (x≥0), EX = 1/λ, Var = 1/λ^2. 그리고 (c) 는 지수분포의 무기억성(memoryless)을 시험한다 — P(X > n+m | X > m) = P(X > n) (Lecture06, p.15).

## 3. 핵심 통찰
"불량(질량 0.02 at 0) + 정상(지수밀도 ×0.98)" 두 성분을 전체 확률로 가중합하면 끝. 단 조건부 (c) 에서 조건 {X ≥ 1} 은 X=0 점질량을 배제하므로 사실상 순수 지수분포가 되고, 무기억성으로 즉답이 나온다.

## 4. 풀이

(a) 일반화 PDF
전체 확률 정리로 두 성분 가중합. X=0 의 점질량은 디랙 델타로 표기한다.
f_X(x) = 0.02·δ(x) + 0.98·2e^(−2x),  x ≥ 0
f_X(x) = 0,                          x < 0  (otherwise = 0)
즉 x = 0 에 무게 0.02 의 충격(델타), x > 0 에 연속 밀도 0.98·2e^(−2x) = 1.96 e^(−2x).
(점검: ∫_(−∞)^∞ f_X dx = 0.02 + 0.98·∫_0^∞ 2e^(−2x)dx = 0.02 + 0.98·1 = 1 ✓.)

(b) P(X ≥ 1)
1 > 0 이므로 점질량(x=0)은 포함되지 않고, 정상 성분만 기여한다.
P(X ≥ 1) = P(정상)·P(Exp ≥ 1) + P(불량)·0
          = 0.98·e^(−2·1) + 0.02·0
          = 0.98·e^(−2) ≈ 0.98·0.1353 ≈ 0.1327.
(지수 CCDF P(X≥1) = e^(−λ·1) = e^(−2), Lecture05, p.14.)

(c) P(X > 2 | X ≥ 1)
조건 {X ≥ 1} 은 X ≥ 1 > 0 이므로 불량(X=0) 가능성을 완전히 배제 → 조건부 세계에서 X 는 순수 Exponential(2).
정의에 따라
P(X > 2 | X ≥ 1) = P(X > 2, X ≥ 1) / P(X ≥ 1) = P(X > 2) / P(X ≥ 1).
P(X > 2) = 0.98·e^(−2·2) = 0.98·e^(−4) (역시 정상 성분만).
P(X > 2 | X ≥ 1) = [0.98·e^(−4)] / [0.98·e^(−2)] = e^(−4+2) = e^(−2) ≈ 0.1353.
무기억성 확인: 조건부가 지수이므로 P(X > 1+1 | X ≥ 1) = P(X > 1) = e^(−2) ✓ (Lecture06, p.15).

(d) EX 와 Var(X)
전체 기댓값 정리(Lecture06, p.16): EX = P(불량)·E[X|불량] + P(정상)·E[X|정상].
E[X|불량] = 0, E[X|정상] = 1/λ = 1/2.
EX = 0.02·0 + 0.98·(1/2) = 0.49.

분산은 Var(X) = E[X^2] − (EX)^2 (Lecture05, p.7).
E[X^2] = 0.02·E[X^2|불량] + 0.98·E[X^2|정상] = 0.02·0 + 0.98·(2/λ^2)
       = 0.98·(2/4) = 0.98·0.5 = 0.49.
(지수 E[X^2] = 2/λ^2 = 2/4 = 0.5, Lecture05, p.14.)
Var(X) = 0.49 − (0.49)^2 = 0.49·(1 − 0.49) = 0.49·0.51 = 0.2499.

## 5. 검산·직관
- 모든 확률 ∈ [0,1] ✓, EX = 0.49 < 정상 기기 평균수명 0.5 (불량이 평균을 0 쪽으로 약간 끌어내림) — 직관과 일치.
- Var = 0.2499 ≈ 0.25. 순수 지수면 Var = 1/λ^2 = 0.25 인데, 2% 불량으로 분산이 미세하게 달라진 값(E[X^2] 도 줄고 (EX)^2 도 줆)이라 합리적.
- (b)(c) 에서 X=0 질량을 실수로 더하면 안 된다 — 둘 다 양의 임계값이라 점질량은 자동 배제.

## 6. 한 줄 요약
> "점질량 + 지수" 혼합 RV 는 전체 확률/기댓값 정리로 두 성분을 가중합하고, 양의 임계값 조건은 점질량을 배제해 지수의 무기억성으로 즉답한다.
