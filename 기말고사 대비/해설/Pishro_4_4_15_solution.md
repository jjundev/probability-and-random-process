## 문제
X는 다음 일반화 PDF(generalized PDF)를 갖는 혼합형(mixed) 확률변수이다.

f_X(x) = (1/3)·δ(x+2) + (1/6)·δ(x−1) + (1/2)·(1/√(2π))·e^(−x^2/2)

a. P(X = 1)과 P(X = −2)를 구하라.
b. P(X ≥ 1)을 구하라.
c. P(X = 1 | X ≥ 1)을 구하라.
d. E[X]와 Var(X)를 구하라.

## 1. 문제 정리
- 주어진 것: 혼합형 RV X의 일반화 PDF.
  - 이산 덩어리(점질량) 둘: x = −2 에 계수 1/3, x = 1 에 계수 1/6 (디랙 델타 δ).
  - 연속 부분: 계수 1/2 짜리 표준정규 N(0,1) 밀도 (1/√(2π))·e^(−x^2/2).
- 구할 것: (a) 두 점의 확률질량, (b) P(X ≥ 1), (c) 조건부확률, (d) 평균·분산.

## 2. 무엇을 묻고 왜 이 도구인가
혼합형 RV는 "이산 점질량 + 연속 밀도"가 한 변수에 공존한다. 디랙 델타 δ(x−a) 앞의 계수가 곧 그 점의 확률질량 P(X = a)이고, 나머지 매끄러운 항은 연속 RV처럼 적분으로 확률을 준다. 정규분포 밀도 (1/√(2π))e^(−x^2/2)와 그 표준화·대칭성은 (Lecture06, p.5)의 표준정규 정의에 근거하며, 연속 RV의 확률은 적분이라는 원칙은 (Lecture05, p.5)에 따른다. (혼합형 RV 자체는 ⚠ 강의에서 별도 정식화하지 않음 — 일반 교과서의 점질량+밀도 분해로 풀이.)

## 3. 핵심 통찰
δ(x−a)의 계수 = P(X = a)이고, 연속 부분은 P(X = 그 한 점) = 0 (Lecture05, p.6). 그래서 "한 점의 확률"은 오직 델타 계수에서만 나온다.

## 4. 풀이

**가중치 확인 (총합 = 1):**
점질량 합 = 1/3 + 1/6 = 1/2, 연속 부분 적분 = (1/2)·∫_(−∞)^∞ (1/√(2π))e^(−x^2/2) dx = (1/2)·1 = 1/2.
1/2 + 1/2 = 1 ✓ — 올바른 분포다.

연속 부분을 N(0,1) RV W로 보면, 표준정규 CDF Φ에 대해 그 항의 확률은 (1/2)·Φ(·) 형태로 나온다. 표준값: Φ(1) ≈ 0.8413, 1 − Φ(1) ≈ 0.1587 (Lecture06, p.8 예시의 표준정규 표).

**(a) 점질량:**
P(X = 1) = 1/6  (δ(x−1) 계수)
P(X = −2) = 1/3  (δ(x+2) 계수)
연속 항은 한 점에서 0이므로 추가 기여 없음.

**(b) P(X ≥ 1):**
X ≥ 1 사건에 기여하는 것: 점 x = 1 (질량 1/6) + 연속 부분 중 W ≥ 1 영역.
P(X ≥ 1) = 1/6 + (1/2)·P(W ≥ 1) = 1/6 + (1/2)·(1 − Φ(1))
= 1/6 + (1/2)·0.1587 = 0.16667 + 0.07935 ≈ 0.2460.
(점 x = −2 는 1보다 작으니 제외.)

**(c) P(X = 1 | X ≥ 1):**
조건부확률 정의 P(A|B) = P(A∩B)/P(B) (Lecture02, p.7). {X = 1} ⊆ {X ≥ 1}이므로
P(X = 1 | X ≥ 1) = P(X = 1) / P(X ≥ 1) = (1/6) / 0.2460 ≈ 0.6775.

**(d) E[X]와 Var(X):**
일반화 PDF의 기댓값은 점질량은 합으로, 연속은 적분으로 처리한다 (Lecture05, p.7의 기댓값 규칙 + 점질량).
E[X] = (−2)·(1/3) + (1)·(1/6) + (1/2)·E[W],  W ~ N(0,1), E[W] = 0
= −2/3 + 1/6 + 0 = −4/6 + 1/6 = −3/6 = −1/2.

E[X^2] = (−2)^2·(1/3) + (1)^2·(1/6) + (1/2)·E[W^2],  E[W^2] = Var(W) + (E[W])^2 = 1
= 4/3 + 1/6 + (1/2)·1 = 8/6 + 1/6 + 3/6 = 12/6 = 2.

Var(X) = E[X^2] − (E[X])^2 = 2 − (−1/2)^2 = 2 − 1/4 = 7/4 = 1.75.

## 5. 검산·직관
- 가중치 총합 1/2(이산) + 1/2(연속) = 1 ✓.
- (c) 답이 P(X=1)/P(X≥1)이고 0 < 0.6775 < 1 ✓ — 조건부확률 범위 정상.
- E[X] = −1/2 는 음수: 큰 음의 점질량(−2에 1/3)이 평균을 왼쪽으로 끌어당김. 직관 일치.
- E[X^2] = 2 가 깔끔하게 떨어지는 건 점질량과 정규의 2차 모멘트가 맞물린 결과.

## 6. 한 줄 요약
> 혼합형 RV는 δ(x−a) 계수 = P(X=a)로 읽고, 점질량은 합·연속부분(정규)은 적분으로 평균/분산을 더하면 된다.
