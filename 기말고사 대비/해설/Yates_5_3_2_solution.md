## 문제
Problem 5.2.2의 확률변수 X와 Y에 대하여 다음을 구하라.
(a) 주변 PMF P_X(x)와 P_Y(y),
(b) 기댓값 E[X]와 E[Y],
(c) 표준편차 σ_X와 σ_Y.

(Problem 5.2.2 설정: X와 Y의 결합 PMF는
P_(X,Y)(x, y) = c·|x + y|,   x = −2, 0, 2;  y = −1, 0, 1
P_(X,Y)(x, y) = 0,   otherwise.
앞서 5.2.2(a)에서 상수 c를 구한다.)

## 1. 문제 정리
- 주어진 것: 결합 PMF P_(X,Y)(x,y) = c|x+y|, x ∈ {−2, 0, 2}, y ∈ {−1, 0, 1}, 그 밖 0.
- 구할 것: (a) 주변 PMF, (b) E[X], E[Y], (c) σ_X, σ_Y. (먼저 c 결정 필요.)

## 2. 무엇을 묻고 왜 이 도구인가
결합 PMF에서 한 변수의 정보만 뽑으려면 다른 변수를 합으로 소거하는 marginalization P_X(x) = Σ_y P_(X,Y)(x,y) (Lecture04, slide 8 영역; Lecture03, p.20). 상수 c는 전체합 = 1 조건으로 결정 (PMF 정규화, Lecture03, p.6). 표준편차는 σ = √Var, Var = E[X^2] − (E[X])^2 (Lecture03, p.18).

## 3. 핵심 통찰
|x+y|와 격자 {−2,0,2}×{−1,0,1}가 원점대칭이라 E[X] = E[Y] = 0이 거의 자명 — 분산만 2차 모멘트로 계산하면 된다.

## 4. 풀이

**Step 0 — 결합 PMF 표 (값 = c|x+y|):**

  (x \ y)   y=−1    y=0    y=1   | 행합
  x=−2:     3c      2c     1c    |  6c
  x= 0:     1c      0      1c    |  2c
  x= 2:     1c      2c     3c    |  6c
  ─────────────────────────────
  열합:     5c      4c     5c    |  14c

전체합 = 14c = 1 ⇒ **c = 1/14**  (5.2.2(a) 답).

**(a) 주변 PMF:** (Lecture04, slide 8 / Lecture03, p.20)
P_X(x) = Σ_y P_(X,Y)(x,y) → 행합:
P_X(−2) = 6/14 = 3/7
P_X(0)  = 2/14 = 1/7
P_X(2)  = 6/14 = 3/7
P_X(x)  = 0,   otherwise   (합 3/7 + 1/7 + 3/7 = 1 ✓)

P_Y(y) = Σ_x P_(X,Y)(x,y) → 열합:
P_Y(−1) = 5/14
P_Y(0)  = 4/14 = 2/7
P_Y(1)  = 5/14
P_Y(y)  = 0,   otherwise   (합 5/14 + 4/14 + 5/14 = 1 ✓)

**(b) 기댓값:** (Lecture03, p.14)
E[X] = (−2)(3/7) + (0)(1/7) + (2)(3/7) = −6/7 + 6/7 = 0.
E[Y] = (−1)(5/14) + (0)(2/7) + (1)(5/14) = −5/14 + 5/14 = 0.

**(c) 표준편차:** (Lecture03, p.18: Var = E[·^2] − (E[·])^2, σ = √Var)
E[X^2] = (−2)^2(3/7) + 0 + (2)^2(3/7) = 4·(3/7) + 4·(3/7) = 24/7.
Var[X] = 24/7 − 0^2 = 24/7 ≈ 3.4286.
σ_X = √(24/7) ≈ 1.852.

E[Y^2] = (−1)^2(5/14) + 0 + (1)^2(5/14) = 5/14 + 5/14 = 10/14 = 5/7.
Var[Y] = 5/7 − 0^2 = 5/7 ≈ 0.7143.
σ_Y = √(5/7) ≈ 0.845.

## 5. 검산·직관
- 주변 PMF 각각 합 = 1 ✓; 전체합 14c = 1로 c = 1/14 ✓.
- 대칭성: P_X(−2) = P_X(2), P_Y(−1) = P_Y(1) — |x+y|와 격자의 원점대칭 때문 → E[X] = E[Y] = 0이 자연스럽다.
- σ_X > σ_Y (1.852 > 0.845): X는 ±2까지 퍼지고 Y는 ±1까지만 퍼지므로 X의 산포가 더 큰 게 당연 ✓.

## 6. 한 줄 요약
> 결합 PMF에서 주변은 행/열 합으로 marginalize하고(정규화로 c=1/14), 대칭이라 평균은 0, 표준편차는 2차 모멘트로 σ_X=√(24/7), σ_Y=√(5/7).
