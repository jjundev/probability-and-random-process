# Yates 5.6.6 풀이
> 출처: Probability and Stochastic Processes (Yates & Goodman), L06_PSP_5.6.6.png, 1순위 근거 Lecture06, 풀이일 2026-06-08
> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X_1 과 X_2 는 독립인 확률변수로, X_i 는 다음 PDF를 갖는다.

f_(X_i)(x) = λ_i·e^(−λ_i x),   x ≥ 0
f_(X_i)(x) = 0,   otherwise

P[X_2 < X_1] 은 무엇인가?

## 1. 문제 정리
- 주어진 것: X_1 ~ Exponential(λ_1), X_2 ~ Exponential(λ_2), 서로 독립. (각 X_i는 자기 고유의 rate λ_i.)
- 구할 것: P[X_2 < X_1].

## 2. 무엇을 묻고 왜 이 도구인가
두 독립 연속 RV의 대소 비교 확률은 결합 PDF를 해당 영역 {x_2 < x_1}에서 이중적분하는 문제다. 독립이므로 f_(X_1,X_2)(x_1,x_2) = f_(X_1)(x_1)·f_(X_2)(x_2) (Lecture06, p.18). 한 변수를 먼저 적분하면 CCDF P(X > x) = e^(−λx) (Lecture05, p.13)가 바로 등장해 계산이 간결해진다.

## 3. 핵심 통찰
X_2를 먼저 0부터 x_1까지 적분하면 그 안쪽 적분이 곧 P(X_2 < x_1) = 1 − e^(−λ_2 x_1) (X_2의 CDF)이 되어, 남은 한 겹만 풀면 된다.

## 4. 풀이

독립이므로 결합 PDF (Lecture06, p.18):
f_(X_1,X_2)(x_1, x_2) = λ_1 e^(−λ_1 x_1) · λ_2 e^(−λ_2 x_2),   x_1 ≥ 0, x_2 ≥ 0
f_(X_1,X_2) = 0,   otherwise

구하는 확률은 영역 {0 ≤ x_2 < x_1} 위의 적분:
P[X_2 < X_1] = ∫_0^∞ ∫_0^(x_1) λ_1 e^(−λ_1 x_1) · λ_2 e^(−λ_2 x_2) dx_2 dx_1.

**안쪽 적분 (x_2: 0 → x_1):** = X_2의 CDF (Lecture05, p.13)
∫_0^(x_1) λ_2 e^(−λ_2 x_2) dx_2 = 1 − e^(−λ_2 x_1).

**바깥 적분 (x_1: 0 → ∞):**
P[X_2 < X_1] = ∫_0^∞ λ_1 e^(−λ_1 x_1) · ( 1 − e^(−λ_2 x_1) ) dx_1
= ∫_0^∞ λ_1 e^(−λ_1 x_1) dx_1 − ∫_0^∞ λ_1 e^(−(λ_1+λ_2) x_1) dx_1.

첫 적분 = 1 (Exponential PDF 정규화).
둘째 적분 = λ_1 · [ 1/(λ_1+λ_2) ] = λ_1/(λ_1+λ_2).

따라서
P[X_2 < X_1] = 1 − λ_1/(λ_1+λ_2) = λ_2/(λ_1+λ_2).

## 5. 검산·직관
- λ_1 = λ_2 (동일 분포): P = 1/2 ✓ — 대칭이라 어느 쪽이 작을 확률이 반반.
- λ_2 → ∞ (X_2가 거의 즉시 0 근처): P → 1 ✓ — 빠르게 작아지는 X_2가 X_1보다 작을 게 거의 확실.
- λ_2 → 0 (X_2가 매우 큼): P → 0 ✓.
- P[X_2 < X_1] + P[X_1 < X_2] = λ_2/(λ_1+λ_2) + λ_1/(λ_1+λ_2) = 1 ✓ (연속이라 동점 확률 0, Lecture05 p.6).

## 6. 한 줄 요약
> 독립 지수 RV의 "누가 더 작냐"는 rate에 비례 — P[X_2 < X_1] = λ_2/(λ_1+λ_2), 즉 더 빨리 일어나는(λ 큰) 쪽이 더 작을 확률이 크다.
