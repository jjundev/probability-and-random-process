## 문제
어느 할인 증권사에서, 가치가 $10,000 미만인 매매는 거래액의 1% 를 수수료로 부과한다. $10,000 을 초과하는 거래는 $100 에 더해 $10,000 초과분의 0.5% 를 부과한다. 단, 1 센트의 소수부에 대해서는 증권사가 항상 1 페니 전체로 올림한다. 주가 D(달러)가 다음 PMF 를 갖는 주식 100 주를 사려 한다.

P_D(d) = 1/3,  d = 99.75, 100, 100.25
P_D(d) = 0,    otherwise

주식을 사는 데 드는 비용 C (수수료 포함)의 PMF 는?

## 1. 문제 정리
- 주어진 것: 100 주 매수, 주당 가격 D ∈ {99.75, 100, 100.25} 각 확률 1/3.
- 거래액 V = 100·D. 수수료 규칙:
  - V < 10,000: 수수료 = 0.01·V.
  - V > 10,000: 수수료 = 100 + 0.005·(V − 10,000), 페니 단위 올림.
- 구할 것: 총비용 C = V + 수수료 의 PMF.

## 2. 무엇을 묻고 왜 이 도구인가
C 는 이산 RV D 의 함수 C = g(D). 이산 derived distribution 의 규칙: P_C(c) = Σ_{d: g(d)=c} P_D(d) (Lecture03, p.21). 각 d 값에서 V 와 수수료를 계산해 C 로 보내고, 그 확률 1/3 을 옮기면 된다.

## 3. 핵심 통찰
세 개의 d 값이 각각 "10,000 미만 / 정확히 10,000 / 초과"의 세 수수료 구간에 정확히 하나씩 떨어진다. g 가 일대일(서로 다른 d → 서로 다른 C)이라 확률 1/3 이 그대로 전달되고, 초과 케이스에서만 페니 올림이 작동한다.

## 4. 풀이

각 d 에 대해 V = 100d, 수수료, C 계산:

### d = 99.75 → V = 9,975 (< 10,000)
수수료 = 0.01 × 9,975 = 99.75 (정확히 페니 단위, 올림 불필요).
C = 9,975 + 99.75 = 10,074.75.

### d = 100 → V = 10,000 (경계)
경계에서는 두 규칙이 동일한 값을 줌: 0.01 × 10,000 = 100, 또는 100 + 0.005×0 = 100.
수수료 = 100. C = 10,000 + 100 = 10,100.

### d = 100.25 → V = 10,025 (> 10,000)
초과분 = 10,025 − 10,000 = 25. 수수료 = 100 + 0.005 × 25 = 100 + 0.125 = 100.125.
페니 올림: $100.125 → $100.13.
C = 10,025 + 100.13 = 10,125.13.

세 C 값이 모두 다르므로 확률 1/3 씩 그대로 전달:

P_C(c) = 1/3,  c = 10,074.75, 10,100, 10,125.13
P_C(c) = 0,    otherwise

## 5. 검산·직관
- 정규화: 1/3 + 1/3 + 1/3 = 1 ✓, 세 값이 서로 달라 합쳐지는 항 없음 ✓.
- 단조성: D 가 클수록 V 도 C 도 큼 (10,074.75 < 10,100 < 10,125.13) — 주가↑ ⇒ 총비용↑, 직관과 일치 ✓.
- 올림 효과: 초과 케이스의 수수료 100.125 → 100.13 으로 0.005 만큼 가입자에게 불리하게 올림됨(증권사 유리). 문제의 "항상 1 페니 전체로 올림" 조건이 정확히 반영됨 ✓.

## 6. 한 줄 요약
> 이산 RV 의 함수 C = g(D) 는 각 d 값을 수수료 규칙에 통과시켜 C 로 보내고 확률 1/3 을 그대로 옮기며, $10,000 초과 케이스에서만 0.5% 수수료를 페니 단위로 올림한다.
