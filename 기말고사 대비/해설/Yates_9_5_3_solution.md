## 문제
휴대전화 통화 시간은 기댓값 150 초인 지수확률변수다. 가입자의 요금제는 월 300 분을 포함해 $30.00 이고, 총 통화시간이 300 분을 초과하면 초과 1 분당 $0.40 을 추가로 낸다. 어느 달에 가입자는 120 통의 통화를 했다.
(a) 중심극한정리(CLT)로 가입자의 요금이 $36 보다 클 확률을 추정하라. (모든 통화시간은 상호 독립이고, 통신사가 통화시간을 정확히 측정해 분의 소수부를 올림하지 않고 그대로 과금한다고 가정.)
(b) 통신사가 소수부 1 분마다 1 분 전체로 과금한다고 하자. 요금이 $36 보다 클 확률을 다시 추정하라.

## 1. 문제 정리
- 통화시간 X_i ~ Exponential, E[X_i] = 150 초 = 2.5 분 ⇒ λ = 1/2.5 = 0.4 /분, Var[X_i] = 1/λ² = 6.25 분².
- 통화 수 n = 120, 독립.
- 요금 = 30 + 0.40·(초과분). 요금 > 36 ⟺ 초과분 > 6/0.40 = 15 분 ⟺ 총 과금시간 > 315 분.

## 2. 무엇을 묻고 왜 이 도구인가
120 개 i.i.d. 합 S_n 의 분포를 직접 구하려면 convolution 120 번 — 사실상 불가능. 그래서 CLT (Lecture09, slide 9): 표준화 Z_n = (S_n − nμ)/(σ√n) → N(0,1). 합의 평균·분산은 E[S_n] = nμ, Var[S_n] = nσ² (Lecture09, slide 4–6). (b) 는 각 통화를 올림하므로 ⌈X_i⌉ 의 평균·분산이 새로 필요한데, ⌈Exp⌉ = Geometric 이라는 사실을 쓴다.

## 3. 핵심 통찰
(a) 합 S_n = ΣX_i 는 평균 300·분산 750 의 정규로 근사 → 임계 315 분까지의 z 점수만 계산. (b) 지수의 올림 ⌈X_i⌉ 는 **기하분포**가 된다(⌈X⌉=k ⟺ X∈(k−1,k], 확률 q^(k−1)(1−q), q=e^(−λ)). 올림이 통화당 평균을 2.5 분 → 약 3.03 분으로 끌어올려 과금시간 기대가 315 분을 훌쩍 넘기므로 확률이 급증한다.

## 4. 풀이

### (a) 올림 없음
X_i ~ Exp, μ = 2.5 분, σ² = 6.25 분². S = X_1 + ... + X_120 (총 통화시간).
E[S] = 120·2.5 = 300 분,  Var[S] = 120·6.25 = 750 분²,  σ_S = √750 ≈ 27.386 분.

요금 > 36 ⟺ S > 315. CLT 표준화:
P[S > 315] = P( (S − 300)/27.386 > (315 − 300)/27.386 ) = P(Z > 15/27.386) = P(Z > 0.5477)
  = 1 − Φ(0.5477) ≈ 1 − 0.7080 = 0.292.

### (b) 통화별 올림 과금 (⌈X_i⌉ 분)
올림 통화시간 Y_i = ⌈X_i⌉. 정수 k ≥ 1 에 대해
P(Y_i = k) = P(k−1 < X_i ≤ k) = e^(−λ(k−1)) − e^(−λk) = (1 − e^(−λ))·(e^(−λ))^(k−1).
이는 Geometric(p) 형태, p = 1 − e^(−λ) = 1 − e^(−0.4) = 0.3297:

P(Y_i = k) = p(1−p)^(k−1),  k = 1, 2, 3, …   (p = 0.3297)
P(Y_i = k) = 0,             otherwise

Geometric 모멘트 (Lecture04, p.16–17 의 E=1/p, 분산 (1−p)/p²):
E[Y_i] = 1/p = 1/0.3297 = 3.033 분.
Var[Y_i] = (1−p)/p² = 0.6703 / 0.3297² = 0.6703/0.10869 = 6.167 분².

총 과금시간 T = Y_1 + ... + Y_120:
E[T] = 120·3.033 = 363.98 분,  Var[T] = 120·6.167 = 740.1 분²,  σ_T = √740.1 ≈ 27.20 분.

요금 > 36 ⟺ T > 315. CLT:
P[T > 315] = P( Z > (315 − 363.98)/27.20 ) = P(Z > −1.801) = Φ(1.801) ≈ 0.964.

## 5. 검산·직관
- (a) σ_S = 27.4 분, 임계 315 는 평균 300 에서 +0.55σ 떨어진 곳 → 확률 ≈ 0.29, 절반보다 약간 작음(평균이 정확히 포함분 300 이라 절반 근처가 타당) ✓.
- (b) 올림이 통화당 +0.53 분을 더해 120 통이면 +64 분 → 기대 과금 364 분이 한도 300 을 크게 초과. 그래서 요금이 $36 을 넘을 확률이 0.29 → 0.96 으로 급등 — "분 단위 올림 과금"이 가입자에게 얼마나 불리한지 정량적으로 드러남, 직관과 일치 ✓.
- Geometric 정합성: p = 1 − e^(−λ) 는 Lecture05, p.18 의 Geometric↔Exponential 슬롯 성공확률 공식과 동일 → 올림이 지수를 기하로 보내는 것이 일관됨 ✓.

## 6. 한 줄 요약
> i.i.d. 합은 CLT 로 정규근사해 임계값의 z 점수로 확률을 구하며, 통화별 올림은 지수를 기하분포로 바꿔 통화당 평균을 ~0.53 분 키워 초과확률을 0.29 에서 0.96 으로 끌어올린다.
