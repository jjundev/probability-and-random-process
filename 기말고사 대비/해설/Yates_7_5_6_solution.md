## 문제
X와 Y는 결합 PDF가 다음과 같은 확률변수다.

f_XY(x,y) = 1/2,   −1 ≤ x ≤ y ≤ 1
f_XY(x,y) = 0,   otherwise

(a) f_Y(y)는?
(b) f_{X|Y}(x|y)는?
(c) E[X | Y = y]는?

## 1. 문제 정리
주어진 것: 삼각 영역 {−1 ≤ x ≤ y ≤ 1}에서 균등한 결합 PDF f_XY = 1/2.
구할 것: (a) 주변 PDF f_Y, (b) 조건부 PDF f_{X|Y}, (c) 조건부 기댓값.

## 2. 무엇을 묻고 왜 이 도구인가
조건부 PDF 정의 f_{X|Y}(x|y) = f_XY(x,y)/f_Y(y) (단 f_Y(y)>0). (Lecture06, p.18) 따라서 먼저 f_Y를 적분으로 구하고, 그다음 나눈다. 조건부 기댓값은 E[X|Y=y] = ∫ x·f_{X|Y}(x|y) dx. (Lecture06, p.18 / Lecture08, p.16)

## 3. 핵심 통찰
부등식 사슬 −1 ≤ x ≤ y ≤ 1을 "y 고정 → x는 −1부터 y까지"로 읽는다. 결합 PDF가 상수 1/2이므로, 조건부 분포 f_{X|Y}(·|y)는 구간 [−1, y]에서의 **균등분포**가 된다 — 그러면 (c)는 적분 없이 중점으로 바로 읽힌다.

## 4. 풀이

**(a) f_Y(y)** — y 고정, x를 −1부터 y까지 적분:

f_Y(y) = ∫_{−1}^{y} (1/2) dx = (1/2)(y − (−1)) = (y+1)/2

f_Y(y) = (y+1)/2,   −1 ≤ y ≤ 1
f_Y(y) = 0,   otherwise

**(b) f_{X|Y}(x|y)** — 정의대로 나눔 (−1 < y ≤ 1이라 f_Y(y) > 0):

f_{X|Y}(x|y) = f_XY(x,y) / f_Y(y) = (1/2) / ((y+1)/2) = 1/(y+1)

f_{X|Y}(x|y) = 1/(y+1),   −1 ≤ x ≤ y
f_{X|Y}(x|y) = 0,   otherwise

→ 이는 구간 [−1, y]에서의 균등분포 (폭 y−(−1) = y+1, 높이 1/(y+1)).

**(c) E[X | Y = y]**:

E[X | Y=y] = ∫_{−1}^{y} x·(1/(y+1)) dx = (1/(y+1))·[x^2/2]_{−1}^{y} = (1/(y+1))·(y^2 − 1)/2

y^2 − 1 = (y−1)(y+1)이므로:

E[X | Y=y] = (y−1)(y+1) / (2(y+1)) = (y−1)/2

## 5. 검산·직관
- f_Y 정규화: ∫_{−1}^1 (y+1)/2 dy = (1/2)[y^2/2 + y]_{−1}^1 = (1/2)[(3/2) − (−1/2)] = 1 ✓.
- (b)는 균등분포이므로 ∫_{−1}^y 1/(y+1) dx = (y+1)/(y+1) = 1 ✓.
- (c) 균등분포 [−1, y]의 평균은 중점 (−1 + y)/2 = (y−1)/2 — 적분 결과와 일치 ✓. 또한 E[X|Y=y] ≤ y가 항상 성립(조건 x≤y) ✓.

## 6. 한 줄 요약
> 상수 결합 PDF를 부등식 −1≤x≤y로 자르면 조건부는 [−1,y] 균등분포가 되고, 조건부 평균은 그 구간 중점 (y−1)/2다.
