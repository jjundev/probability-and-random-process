## 문제
X가 다음 균등(uniform) PMF를 갖는다.

P_X(x) = 0.01,   x = 1, 2, ..., 100
P_X(x) = 0,   otherwise

(a) X의 최빈값(mode) x_mod 를 구하라. 최빈값이 유일하지 않으면 모든 최빈값의 집합 X_mod 를 구하라.
(b) X의 중앙값(median) x_med 를 구하라. 중앙값이 유일하지 않으면 모든 중앙값 x의 집합 X_med 를 구하라.

## 1. 문제 정리
- 주어진 것: X ~ Uniform{1,...,100} (이산 균등), P_X(x) = 0.01 for x ∈ {1,...,100}, 0 otherwise.
- 구할 것: (a) 최빈값(집합), (b) 중앙값(집합).
- 이는 정수 구간의 이산 균등분포다. (Lecture03, p.10: Uniform(a,b), P_X(i) = 1/(b−a+1), 여기 a=1, b=100, 1/100 = 0.01, otherwise = 0.)

## 2. 무엇을 묻고 왜 이 도구인가
위치 측도(mode, median) 정의 적용.
- 최빈값 = PMF를 최대로 만드는 값. 균등분포는 모든 값의 확률이 같으므로 최빈값이 유일하지 않다.
- 중앙값 x_med = P[X ≤ x_med] ≥ 1/2 이면서 P[X ≥ x_med] ≥ 1/2 인 값. 균등이라 누적이 0.5를 "정확히" 치는 지점이 생겨 중앙값 집합이 구간이 된다.
(균등 PMF 근거: Lecture03, p.10. mode/median 정의는 Yates 본문 표준 정의 — 정리노트에 명시 항목은 없어 일반 교과서 정의로 보강.)

## 3. 핵심 통찰
균등분포에서는 "모든 결과가 동등"하다는 성질이 두 측도를 모두 비유일하게 만든다. 최빈값은 전체 support, 중앙값은 누적확률이 정확히 1/2이 되는 50과 51 사이 전 구간이 된다(짝수 개 100개라 가운데가 두 점 50·51 사이로 벌어짐).

## 4. 풀이

(a) 최빈값.
P_X(x) = 0.01 은 x = 1, ..., 100 전부에서 동일한 최댓값. 어느 한 값도 다른 값보다 크지 않으므로 최빈값은 유일하지 않다.

X_mod = {1, 2, 3, ..., 100}  (즉 1 ≤ x ≤ 100 인 모든 정수)

(b) 중앙값.
중앙값 정의: P[X ≤ x_med] ≥ 1/2 이고 P[X ≥ x_med] ≥ 1/2.
누적확률: 정수 k (1 ≤ k ≤ 100)에 대해 P[X ≤ k] = 0.01k.
- P[X ≤ 50] = 0.50 ≥ 0.5 ✓, 그리고 P[X ≥ 50] = 0.01·(100 − 50 + 1) = 0.51 ≥ 0.5 ✓ → 50은 중앙값.
- P[X ≤ 51] = 0.51 ≥ 0.5 ✓, P[X ≥ 51] = 0.01·(100 − 51 + 1) = 0.50 ≥ 0.5 ✓ → 51도 중앙값.
- 50과 51 사이의 임의의 실수 x (50 < x < 51)에 대해서도, 그 사이엔 확률질량이 없으므로 P[X ≤ x] = 0.50 ≥ 0.5 이고 P[X ≥ x] = 0.50 ≥ 0.5 → 중앙값.
- x < 50 이면 P[X ≤ x] ≤ 0.49 < 0.5 (위배), x > 51 이면 P[X ≥ x] ≤ 0.49 < 0.5 (위배) → 제외.

따라서 중앙값은 유일하지 않고

X_med = {x : 50 ≤ x ≤ 51}  (닫힌 구간 [50, 51])

(대표값 하나를 꼽으라면 통상 중점 x_med = 50.5 를 쓴다.)

## 5. 검산·직관
- 최빈값: 균등이라 "가장 흔한 값"이 따로 없음 — 전체가 동률 ✓.
- 중앙값: 100개(짝수)를 절반으로 가르면 아래 50개{1..50}, 위 50개{51..100}. 경계가 50과 51 사이라는 결과와 일치 ✓. 누적이 정확히 0.50을 50에서 치므로 50~51 구간 전체가 조건을 만족하는 것이 균등(이산)의 전형적 비유일 중앙값 현상.
- 합 점검: P[X≤50] + P[X≥51] = 0.50 + 0.50 = 1 ✓ (50과 51 사이엔 질량 없음).

## 6. 한 줄 요약
> 이산 균등은 모든 값이 동률이라 최빈값이 support 전체 {1,...,100}이고, 100개(짝수)라 누적이 정확히 1/2을 치는 중앙값은 한 점이 아니라 구간 [50, 51](대표값 50.5)이 된다.
