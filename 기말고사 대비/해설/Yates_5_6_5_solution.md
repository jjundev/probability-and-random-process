## 문제
X 는 연속균등(0, 2) 확률변수이다. Y 는 연속균등(0, 5) PDF 를 가지며 X 와 독립이다. 결합 PDF f_(X,Y)(x,y) 는?

## 1. 문제 정리
주어진 것: X ~ Uniform(0,2), Y ~ Uniform(0,5), X ⊥ Y (독립).
구할 것: 결합 PDF f_(X,Y)(x,y).

균등분포 PDF (Lecture05, p.7): f(x) = 1/(b−a) on [a,b], otherwise 0.
독립 → 결합 = 주변의 곱 (Lecture06, p.18).

## 2. 무엇을 묻고 왜 이 도구인가
독립인 두 연속 RV 의 결합 PDF 구성. 핵심 정리 하나면 끝: X ⊥ Y ⟺ f_(X,Y)(x,y) = f_X(x)·f_Y(y) for all x,y (Lecture06, p.18). 각 주변은 균등 PDF.

## 3. 핵심 통찰
"독립이면 결합 = 곱" — 두 균등 밀도를 곱하면 직사각형 [0,2]×[0,5] 위의 상수 밀도가 된다.

## 4. 풀이
주변 PDF (Lecture05, p.7, 균등):
f_X(x) = 1/(2−0) = 1/2,  0 ≤ x ≤ 2;   f_X(x) = 0, otherwise.
f_Y(y) = 1/(5−0) = 1/5,  0 ≤ y ≤ 5;   f_Y(y) = 0, otherwise.

독립이므로 결합 = 곱 (Lecture06, p.18):
f_(X,Y)(x,y) = f_X(x)·f_Y(y) = (1/2)·(1/5) = 1/10,  0 ≤ x ≤ 2, 0 ≤ y ≤ 5
f_(X,Y)(x,y) = 0,                                     otherwise

## 5. 검산·직관
- 정규화: ∬ f = (1/10)·Area([0,2]×[0,5]) = (1/10)·(2·5) = (1/10)·10 = 1 ✓.
- 직사각형 지지 위 상수 1/10 = 두 균등 높이(1/2, 1/5)의 곱 — 독립 균등쌍은 항상 직사각형 위 균등결합.

## 6. 한 줄 요약
> 독립인 두 균등 RV 의 결합 PDF 는 주변 밀도의 곱, 즉 직사각형 [0,2]×[0,5] 위에서 1/10 (그 밖 0) 이다.
