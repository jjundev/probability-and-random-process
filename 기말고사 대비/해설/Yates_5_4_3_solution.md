## 문제
확률변수 X, Y 의 결합 PDF 가

f_{X,Y}(x, y) = 6e^(−(2x+3y)),  x ≥ 0, y ≥ 0
f_{X,Y}(x, y) = 0,              otherwise

(a) P[X > Y] 와 P[X + Y ≤ 1] 을 구하라.
(b) P[min(X, Y) ≥ 1] 을 구하라.
(c) P[max(X, Y) ≤ 1] 을 구하라.

## 1. 문제 정리
- 주어진 것: 위 결합 PDF.
- 구할 것: (a) P[X>Y], P[X+Y≤1]; (b) P[min(X,Y)≥1]; (c) P[max(X,Y)≤1].

## 2. 무엇을 묻고 왜 이 도구인가
결합 PDF 가 x 만의 함수 × y 만의 함수로 **곱 분해**되는지부터 본다. 분해되고 정의역이 사각형(곱집합)이면 독립 (Lecture06, p.18: X ⊥ Y ⇔ f_{X,Y} = f_X f_Y). 6e^(−2x−3y) = (2e^(−2x))(3e^(−3y)) 이고 정의역 {x≥0}×{y≥0} 도 곱집합 → X ~ Exp(2), Y ~ Exp(3) 독립. 그러면 min/max 사건은 독립 곱으로, X>Y·X+Y≤1 은 영역적분으로 푼다. min/max 는 L07 의 "교집합으로 바꾸는" 논리(Lecture07, p.11) 를 그대로 쓴다.

## 3. 핵심 통찰
f_{X,Y} 가 곱으로 쪼개지는 순간 X ⊥ Y 인 두 지수분포로 분해된다. 그러면 (b)(c) 의 min·max 는 적분 없이 **독립 곱**으로 끝나고, Exponential 의 CCDF P(X>x) = e^(−λx) (Lecture05, p.13) 만 쓰면 된다.

## 4. 풀이

분해: f_X(x) = 2e^(−2x) (x ≥ 0, otherwise = 0), f_Y(y) = 3e^(−3y) (y ≥ 0, otherwise = 0). 즉 X ~ Exp(λ=2), Y ~ Exp(λ=3), 독립.
CCDF: P(X > a) = e^(−2a), P(Y > b) = e^(−3b) (a, b ≥ 0).

### (a-1) P[X > Y]
x 를 먼저 (y → ∞) 적분: ∫_y^∞ 2e^(−2x) dx = e^(−2y). 이어서
P[X > Y] = ∫_0^∞ 3e^(−3y) · e^(−2y) dy = ∫_0^∞ 3e^(−5y) dy = 3/5.

### (a-2) P[X + Y ≤ 1]
삼각영역 0 ≤ y ≤ 1, 0 ≤ x ≤ 1 − y:
내부 ∫_0^(1−y) 2e^(−2x) dx = 1 − e^(−2(1−y)).
P[X+Y≤1] = ∫_0^1 3e^(−3y)[1 − e^(−2(1−y))] dy
  = (1 − e^(−3)) − 3e^(−2)∫_0^1 e^(−y) dy
  = (1 − e^(−3)) − 3e^(−2)(1 − e^(−1))
  = 1 − 3e^(−2) + 2e^(−3).
수치: 1 − 0.40601 + 0.09957 ≈ 0.6936.

### (b) P[min(X, Y) ≥ 1]
min(X,Y) ≥ 1 ⇔ X ≥ 1 그리고 Y ≥ 1 (둘 다 1 이상). 독립:
P[min(X,Y) ≥ 1] = P(X ≥ 1)·P(Y ≥ 1) = e^(−2)·e^(−3) = e^(−5) ≈ 0.00674.

### (c) P[max(X, Y) ≤ 1]
max(X,Y) ≤ 1 ⇔ X ≤ 1 그리고 Y ≤ 1. 독립:
P[max(X,Y) ≤ 1] = P(X ≤ 1)·P(Y ≤ 1) = (1 − e^(−2))(1 − e^(−3)) ≈ 0.8645 · 0.9502 ≈ 0.8216.

## 5. 검산·직관
- 곱 분해의 정규화: ∫_0^∞ 2e^(−2x)dx = 1, ∫_0^∞ 3e^(−3y)dy = 1 → 결합적분 1 ✓.
- min vs max 부등호 방향: min ≥ 1 은 둘 다 큰(작은 확률 e^(−5)), max ≤ 1 은 둘 다 작은(큰 확률 0.82). e^(−5) ≪ 0.82 로 방향이 맞음.
- P[X>Y] = 3/5 직관: Y 의 율 λ=3 이 더 커서 Y 가 더 빨리(작게) 끝나므로 X > Y 가 절반 이상 — 일반식 λ_Y/(λ_X+λ_Y) = 3/5 와 일치.

## 6. 한 줄 요약
> 결합 PDF 가 곱으로 쪼개지면 독립 지수분포로 분해되고, min/max 사건은 적분 없이 CCDF·CDF 의 독립 곱으로 끝난다.
