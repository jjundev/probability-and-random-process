# Yates 6.2.1 풀이

> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
1 Ω 저항 양단의 전압 X 는 모수 0 과 1 의 균등 확률변수이다. 순간 전력은 Y = X² 이다. Y 의 CDF F_Y(y) 와 PDF f_Y(y) 를 구하라.

## 1. 문제 정리
주어진 것: X ~ Uniform(0,1), 즉
  f_X(x) = 1,  0 ≤ x ≤ 1
  f_X(x) = 0,  otherwise   (Lecture05, p.7)
  CDF: F_X(x) = x for 0 ≤ x ≤ 1 (0 if x<0, 1 if x>1).
변환: Y = X² = g(X).
구할 것: F_Y(y), f_Y(y).

## 2. 무엇을 묻고 왜 이 도구인가
파생분포(derived distribution) 문제. 원칙: **무조건 CDF 부터 구하고 미분한다** (Lecture07, p.20, "첫 번째 뭐부터 계산한다? cdf 부터"). Y = X² 처럼 비단조(일반엔 ±√y 두 가지)일 땐 PDF 직접 공식보다 사건 {X² ≤ y} 를 X 의 구간으로 바꿔 CDF 를 세우는 게 안전하다. 여기선 X ∈ [0,1] 로 비음이라 √ 한 갈래만 살아남는다.

## 3. 핵심 통찰
"Y ≤ y ⇔ X² ≤ y ⇔ X ≤ √y" (X ≥ 0 이라 −√y 가지는 무관) — 사건을 X 의 단순 부등식으로 번역하면 F_Y(y) = F_X(√y) 가 즉시 나온다.

## 4. 풀이

(단계 1) Y 의 치역
X ∈ [0,1] → Y = X² ∈ [0,1]. 따라서 0 ≤ y ≤ 1 에서만 비자명, y < 0 이면 F_Y = 0, y > 1 이면 F_Y = 1.

(단계 2) CDF (Lecture07, p.22 Step 1)
0 ≤ y ≤ 1 에서:
F_Y(y) = P(Y ≤ y) = P(X² ≤ y) = P(−√y ≤ X ≤ √y).
X ≥ 0 (지지 [0,1]) 이므로 음수 가지는 무시 → = P(X ≤ √y) = F_X(√y) = √y (0 ≤ √y ≤ 1).
정리:
F_Y(y) = 0,    y < 0
F_Y(y) = √y,   0 ≤ y ≤ 1
F_Y(y) = 1,    y > 1

(단계 3) PDF (Lecture07, p.23 Step 2: 미분)
f_Y(y) = d/dy F_Y(y). 0 < y < 1 에서 d/dy √y = 1/(2√y).
f_Y(y) = 1/(2√y),  0 < y ≤ 1
f_Y(y) = 0,         otherwise
(otherwise = 0: y ≤ 0 및 y > 1 에서 F_Y 가 상수라 미분 0. y=0 은 f 가 발산하나 적분 가능한 적분특이점.)

(검증 — Lecture07, p.37 의 Y=X² 일반공식과 대조: f_Y(y) = [f_X(√y) + f_X(−√y)] / (2√y). f_X(−√y)=0(음수 지지 밖), f_X(√y)=1 → f_Y = 1/(2√y) ✓ 동일.)

## 5. 검산·직관
- 정규화: ∫_0^1 1/(2√y) dy = [√y]_0^1 = 1 ✓.
- f_Y(y) = 1/(2√y) 는 y→0 에서 ∞ 로 치솟음 — X 가 0 근처에 균등하게 깔려도 Y=X² 가 0 근방으로 압축되어 밀도가 쌓이기 때문(작은 전압이 더 작은 전력으로 뭉침). 직관과 일치.
- 경계: F_Y(0)=0, F_Y(1)=1 ✓; F_Y 는 [0,1] 에서 √y 로 증가(오목) ✓.

## 6. 한 줄 요약
> 파생분포는 "Y≤y ⇔ X≤√y" 로 CDF 부터 세워 F_Y(y)=√y 를 얻고 미분해 f_Y(y)=1/(2√y) (0<y≤1, 그 밖 0) 를 구한다.
