## 문제
러미(rummy) 게임에서 7 장의 카드를 받는다.
(a) 받은 패가 모두 빨간 카드일 확률 P[R_7] 은?
(b) 받은 패가 모두 그림 카드(face card)일 확률 P[F] 는?
(c) 받은 패가 모두 빨간 그림 카드일 확률 P[R_7 F] 은? (그림 카드는 J, Q, K.)

## 1. 문제 정리
- 표준 52 장 덱에서 7 장을 무작위로 뽑음(순서 무관).
- 빨간 카드: 26 장(하트 + 다이아몬드). 그림 카드 J/Q/K: 4 무늬 × 3 = 12 장. 빨간 그림 카드: 2 무늬(하트·다이아) × 3 = 6 장.
- 구할 것: (a) 전부 빨강, (b) 전부 그림, (c) 전부 빨간 그림.

## 2. 무엇을 묻고 왜 이 도구인가
표본공간이 "52 장 중 7 장 조합"이고 모든 조합이 등확률(equally likely) — 고전적 확률 P = (유리한 경우 수)/(전체 경우 수) (Lecture02, p.7 의 확률모델 토대). 순서가 무관하므로 조합 C(n,k) = n!/(k!(n−k)!) 로 센다. ⚠ 조합 카운팅 자체는 L02(조건부·베이즈) 정리노트에 직접 수록되어 있지 않아 일반 교과서의 등확률 모델·조합 공식을 사용하되, "사건 확률 = 경우 수의 비"라는 기본 틀을 따른다.

## 3. 핵심 통찰
세 문제 모두 "특정 부분덱에서만 7 장을 모두 뽑을 확률" = C(부분덱 크기, 7) / C(52, 7). (c) 의 결정적 관찰: 빨간 그림 카드는 **6 장뿐**인데 7 장을 뽑아야 하므로 C(6,7) = 0 → 확률 0.

## 4. 풀이

전체 경우 수: C(52, 7) = 133,784,560.

### (a) 모두 빨강
빨간 카드 26 장에서 7 장: 유리한 경우 = C(26, 7) = 657,800.
P[R_7] = C(26,7) / C(52,7) = 657,800 / 133,784,560 ≈ 0.00492.

### (b) 모두 그림
그림 카드 12 장에서 7 장 (12 ≥ 7 이라 가능): C(12, 7) = 792.
P[F] = C(12,7) / C(52,7) = 792 / 133,784,560 ≈ 5.92 × 10^(−6).

### (c) 모두 빨간 그림
빨간 그림 카드는 6 장(하트·다이아의 J,Q,K)뿐인데 7 장이 필요 → 불가능.
유리한 경우 = C(6, 7) = 0.
P[R_7 F] = 0.

## 5. 검산·직관
- 크기 순서: 빨강(26장) ⊃ 사건이 더 흔하고, 그림(12장)은 좁아 훨씬 드물며, 빨간 그림(6장)은 7 장 자체가 불가능 → P[R_7] ≫ P[F] ≫ P[R_7 F] = 0, 직관과 일치 ✓.
- 포함관계 점검: R_7 F ⊆ F ⊆ ... 이고 P[R_7 F] = 0 ≤ P[F] ✓.
- 조합 값: C(12,7) = C(12,5) = 792 (대칭성 C(n,k)=C(n,n−k)) ✓.

## 6. 한 줄 요약
> 등확률 7 장 조합에서 "특정 부분덱만으로 채울 확률"은 C(부분덱,7)/C(52,7) 이며, 부분덱이 7 장보다 작으면(빨간 그림 6 장) 그 확률은 0 이다.
