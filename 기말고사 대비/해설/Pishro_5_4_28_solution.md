> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X와 Y가 독립인 N(0,1) 확률변수이고 U = X + Y 라 하자.
a. X = x 가 주어졌을 때 U의 조건부 PDF f_(U|X)(u|x) 를 구하라.
b. U의 PDF f_U(u) 를 구하라.
c. U = u 가 주어졌을 때 X의 조건부 PDF f_(X|U)(x|u) 를 구하라.
d. E[X | U = u] 와 Var(X | U = u) 를 구하라.

## 1. 문제 정리
- 주어진 것: X, Y ~ N(0,1) 독립. U = X + Y.
- 표준정규 PDF: f(t) = (1/√(2π)) e^(−t²/2), 모든 실수 t (otherwise 해당 없음 — 전 실수에서 양수).
- 구할 것: (a) f_(U|X)(u|x), (b) f_U(u), (c) f_(X|U)(x|u), (d) E[X|U=u], Var(X|U=u).

## 2. 무엇을 묻고 왜 이 도구인가
독립 정규의 합 U = X+Y 와 그에 대한 조건부 분포. 핵심 도구 둘:
1) **독립 정규합은 정규** — N(μ_x,σ_x²)+N(μ_y,σ_y²) = N(μ_x+μ_y, σ_x²+σ_y²) (Lecture07, p.18). 여기선 convolution을 직접 안 적어도 결론을 쓸 수 있으나, (a)에서 조건부 구조가 그 적분의 피적분함수임을 보인다.
2) **조건부 = 결합/주변** — f_(U|X)(u|x) = f_(X,U)(x,u)/f_X(x). U = X+Y 라는 결정적 관계 덕에 X를 고정하면 U는 Y의 평행이동이 된다.

## 3. 핵심 통찰
X = x 로 고정하면 U = x + Y 이므로, U는 상수 x만큼 평행이동된 Y다. Y ~ N(0,1) 이므로 **조건부로 U|{X=x} ~ N(x, 1)** — 평균이 x로 옮겨가고 분산은 그대로 1. 이 한 줄이 (a)를 끝내고, 나머지는 결합 = 조건부×주변을 조립·재분해하는 베이즈 회전이다.

## 4. 풀이

(a) f_(U|X)(u|x).
X = x 고정 시 U = x + Y. Y ~ N(0,1) 독립이므로 U는 N(x, 1) 분포 (평균 x, 분산 1). 즉

f_(U|X)(u|x) = (1/√(2π)) · e^(−(u−x)²/2),   모든 실수 u (otherwise = 0 해당 없음, 전 실수 양수)

(엄밀히: U = u ⇔ Y = u − x 이고 |dY/dU| = 1 이므로 f_(U|X)(u|x) = f_Y(u−x).)

(b) f_U(u).
독립 정규합 정리로 즉시: U = X + Y ~ N(0+0, 1+1) = N(0, 2). (Lecture07, p.18)

f_U(u) = (1/√(2π·2)) · e^(−u²/(2·2)) = (1/(2√π)) · e^(−u²/4),   모든 실수 u (otherwise = 0 해당 없음)

(유도 확인 — convolution f_U(u) = ∫ f_X(x) f_Y(u−x) dx (Lecture07, p.16). 피적분함수는 (1/2π) exp(−[x² + (u−x)²]/2). 지수 안 x² + (u−x)² = 2x² − 2ux + u² = 2(x − u/2)² + u²/2. x에 대해 가우시안 적분하면 ∫ exp(−(x−u/2)²) dx = √π, 남는 인자 (1/2π)·√π·e^(−u²/4) = (1/(2√π)) e^(−u²/4) ✓. N(0,2)와 일치.)

(c) f_(X|U)(x|u).
베이즈로 뒤집는다:

f_(X|U)(x|u) = f_(U|X)(u|x) · f_X(x) / f_U(u).

분자 = [(1/√(2π)) e^(−(u−x)²/2)] · [(1/√(2π)) e^(−x²/2)] = (1/2π) e^(−[(u−x)² + x²]/2).
분모 = (1/(2√π)) e^(−u²/4).

지수 부분만 정리: −[(u−x)² + x²]/2 + u²/4. 분자 지수의 괄호 = (u−x)² + x² = 2x² − 2ux + u². 따라서

−(2x² − 2ux + u²)/2 + u²/4 = −x² + ux − u²/2 + u²/4 = −x² + ux − u²/4 = −(x − u/2)².

상수 인자: (1/2π) ÷ (1/(2√π)) = (1/2π)·(2√π) = 1/√π. 따라서

f_(X|U)(x|u) = (1/√π) · e^(−(x − u/2)²),   모든 실수 x (otherwise = 0 해당 없음)

이를 정규 표준형 (1/√(2πσ²)) e^(−(x−m)²/(2σ²)) 과 맞추면 2σ² = 1 ⇒ σ² = 1/2, m = u/2. 즉

X | {U = u} ~ N(u/2, 1/2).

(d) E[X|U=u], Var(X|U=u).
(c)에서 조건부 분포가 N(u/2, 1/2) 이므로 평균·분산을 바로 읽는다:

E[X | U = u] = u/2.
Var(X | U = u) = 1/2.

## 5. 검산·직관
- (d)의 대칭성: X와 Y가 교환가능(i.i.d.)이므로 E[X|U=u] = E[Y|U=u] 여야 하고, 둘을 더하면 E[X+Y|U=u] = E[U|U=u] = u. 따라서 각각 u/2 ✓.
- 분산 직관: U를 알면 X에 대한 정보가 생겨 불확실성이 준다. 사전 Var(X)=1 → 사후 Var(X|U)=1/2 (절반으로 감소) ✓. U라는 한 개의 선형결합을 알면 분산이 정확히 반으로 주는 것은 i.i.d. 두 정규의 표준 결과.
- (b) 점검: f_U 는 N(0,2). Var(U) = Var(X)+Var(Y) = 1+1 = 2 ✓ (독립이라 공분산 0, Lecture07 p.2.8).
- (a)→(c) 일관성: f_(U|X)(u|x)·f_X(x) = f_(X|U)(x|u)·f_U(u) (둘 다 결합 f_(X,U)(x,u)) — 위 계산이 정확히 이 항등식을 사용했으므로 정합 ✓.

## 6. 한 줄 요약
> X 고정 시 U=x+Y 라 U|X ~ N(x,1), 독립합으로 U ~ N(0,2)이며, 베이즈로 뒤집으면 X|{U=u} ~ N(u/2, 1/2) — 합을 알면 각 성분의 사후평균은 u의 절반, 사후분산은 절반으로 준다.
