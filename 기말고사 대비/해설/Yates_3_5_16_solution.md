## 문제
TV 게임쇼에 똑같이 생긴 가방 3개가 있다. 첫째 가방엔 3달러, 둘째엔 30달러, 셋째엔 300달러가 있다. 무작위로 가방 하나를 고르며 게임을 시작한다. 고르지 않은 두 가방 중 사회자가 **돈이 더 많은** 가방을 연다. 그다음 사회자가 가방을 그대로 둘지 남은 다른 가방으로 바꿀지 묻는다. 결정 후 가방을 열어 안의 D달러를 갖는다. 가방을 바꿔야 하는가? 표기: C_i = 처음에 i달러 가방을 고르는 사건, O_i = 사회자가 i달러 가방을 여는 사건.

(a) 절대 안 바꾼다(원래 선택 고수). PMF P_D(d)와 E[D]를 구하라.
(b) 항상 바꾼다. PMF P_D(d)와 E[D]를 구하라.
(c) 바꾸는 규칙을 사회자가 연 가방의 금액에 따라 정해야 할까? E[D]를 최대화하는 최적 전략은? (힌트: 사회자가 i달러를 열면 확률 α_i로 바꾸는 무작위 결정을 고려.)

## 1. 문제 정리
가방 값 {3, 30, 300}. 처음 선택은 균등(각 C_i 확률 1/3). 사회자는 고르지 않은 둘 중 **최댓값** 가방을 연다. 핵심: 두 미선택 가방의 최댓값은 항상 30 또는 300 (절대 3이 아님 — 미선택 둘 중 하나는 반드시 30 또는 300).
구할 것: (a) 고수 전략, (b) 항상 바꾸기, (c) 최적 전략의 E[D].

## 2. 무엇을 묻고 왜 이 도구인가
표준 Monty Hall(사회자가 꽝을 염)과 달리 여기선 사회자가 **가장 큰 상금**을 공개한다 — 정보 구조가 정반대다. 도구: 트리(전체 경우 나열) → 각 잎의 D값과 확률 → PMF → E[D] = Σ d·P_D(d) (Lecture03, p.14). (c)는 사회자가 연 금액으로 조건화한 뒤(Lecture02 베이즈/조건부, p.7) 조건부 기댓값을 비교.

## 3. 핵심 통찰
사회자가 항상 최댓값을 열므로, 남는 미선택 가방은 **두 미선택 중 작은 쪽**이다. 따라서 "바꾸면" 항상 작은 쪽으로 간다 — 바꾸기는 결코 300을 잡지 못한다. 이 한 가지가 (b)를 형편없게, (a)를 최적과 동률로 만든다.

## 4. 풀이

**경우 나열** (각 C_i 확률 1/3, 사회자 행동은 결정적):

| 처음 선택 C_i | 미선택 둘 | 사회자가 엶 O | 남은 미선택(바꾸면 받음) |
|---|---|---|---|
| C_3 (3)   | {30, 300} | 300 | 30 |
| C_30 (30) | {3, 300}  | 300 | 3  |
| C_300(300)| {3, 30}   | 30  | 3  |

**(a) 절대 안 바꿈** — D = 원래 선택, 균등:

P_D(3) = 1/3,  P_D(30) = 1/3,  P_D(300) = 1/3
P_D(d) = 0,  otherwise

E[D] = (1/3)(3) + (1/3)(30) + (1/3)(300) = (3+30+300)/3 = 111

**(b) 항상 바꿈** — 위 표의 "남은 미선택" 열:
- C_3 → D=30 (확률 1/3)
- C_30 → D=3 (확률 1/3)
- C_300 → D=3 (확률 1/3)

P_D(3) = 1/3 + 1/3 = 2/3,  P_D(30) = 1/3,  P_D(300) = 0
P_D(d) = 0,  otherwise

E[D] = 3·(2/3) + 30·(1/3) + 300·0 = 2 + 10 = 12

→ 항상 바꾸기는 처참하다(12 ≪ 111). 사회자가 최댓값을 열기 때문에 바꾸면 절대 300을 못 잡는다.

**(c) 최적 전략** — 사회자가 연 금액 O로 조건화.

사회자는 30 또는 300만 연다.

▷ **O_30이 일어남 ⇔ C_300** (미선택이 {3,30}이라 최댓값 30을 엶). 이때 내 가방=300, 남은 미선택=3.
 - 고수 → D=300, 바꿈 → D=3. ⇒ 무조건 고수(α_30 = 0). 값 = 300.
 - P(O_30) = P(C_300) = 1/3.

▷ **O_300이 일어남 ⇔ C_3 또는 C_30** (미선택에 300 포함). 두 경우 prior 동일, 모두 결정적으로 300을 열게 함 → P(C_3|O_300) = P(C_30|O_300) = 1/2.
 - 고수: C_3→D=3, C_30→D=30 ⇒ E[D|O_300, 고수] = (1/2)(3) + (1/2)(30) = 16.5
 - 바꿈: C_3→D=30, C_30→D=3 ⇒ E[D|O_300, 바꿈] = (1/2)(30) + (1/2)(3) = 16.5
 - ⇒ 고수든 바꿈이든 **동일** (α_300 무의미). 값 = 16.5.
 - P(O_300) = P(C_3 ∪ C_30) = 2/3.

최대 기댓값:
E[D]_max = P(O_30)·300 + P(O_300)·16.5 = (1/3)(300) + (2/3)(16.5) = 100 + 11 = 111

**결론**: 사회자가 30을 열면(=내가 300을 가진 경우) 반드시 고수, 사회자가 300을 열면 고수·교환 무차별. 최적 E[D] = 111로, **(a) 절대 고수와 정확히 같다.** 즉 이 게임에선 바꿔서 이득 볼 방법이 없으며, 항상 바꾸는 건 손해다.

## 5. 검산·직관
- (a) PMF 합 = 1, (b) PMF 합 = 2/3+1/3 = 1 ✓.
- 표준 Monty Hall은 사회자가 꽝을 열어 정보가 "남은 문에 상금 몰림"으로 흘러 바꾸는 게 이득. 여기선 사회자가 최댓값을 열어 정보가 반대로 흐르므로 바꾸기가 손해 — 직관 일치 ✓.
- 최적값 111 = 무조건부 평균 (3+30+300)/3. 사회자의 공개가 평균적으로 이득을 못 주는 구조라 "그냥 평균"을 못 넘는 게 타당 ✓.

## 6. 한 줄 요약
> 사회자가 최댓값을 공개하므로 바꾸면 절대 300을 못 잡는다 — 항상 바꾸기는 E[D]=12로 최악이고, 최적(30 열리면 고수)조차 E[D]=111로 그냥 안 바꾸기와 같다.
