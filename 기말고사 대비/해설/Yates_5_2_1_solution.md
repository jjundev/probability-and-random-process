## 문제
확률변수 X와 Y가 결합 PMF
P_X,Y(x,y) = c·x·y,   x = 1, 2, 4 ; y = 1, 3
P_X,Y(x,y) = 0,        otherwise
를 가진다.
(a) 상수 c 의 값은?  (b) P[Y < X] 는?  (c) P[Y > X] 는?  (d) P[Y = X] 는?  (e) P[Y = 3] 는?

## 1. 문제 정리
주어진 것: P_X,Y(x,y) = cxy, (x,y) ∈ {1,2,4}×{1,3}, 그 외 0. 총 6개의 점.
구할 것: (a) 정규화 상수 c, (b)~(e) 네 가지 사건 확률.

## 2. 무엇을 묻고 왜 이 도구인가
모든 결합 PMF는 합이 1이어야 한다(정규화): Σ_x Σ_y P_X,Y(x,y) = 1 (Lecture03, p.20). 여기서 c를 결정한다. cxy가 x·y로 분리(separable)되므로 이중합이 (Σx)(Σy)로 인수분해되는 게 계산의 핵심. 나머지 사건 확률은 해당 (x,y) 점들의 확률을 모아 더하면 된다 — 사건은 6개 점 중 부분집합.

## 3. 핵심 통찰
PMF가 곱 형태 cxy라 Σ_x Σ_y xy = (Σx)(Σy) = 7·4 = 28 로 한 번에 끝난다. 각 사건은 6개 격자점을 부등호로 분류해 확률을 합산.

## 4. 풀이

### (a) 정규화 상수 c
Σ_x Σ_y cxy = c·(Σ_{x∈{1,2,4}} x)·(Σ_{y∈{1,3}} y) = c·(1+2+4)·(1+3) = c·7·4 = 28c.
정규화 28c = 1 → c = 1/28.

따라서 완성된 PMF:
P_X,Y(x,y) = xy/28,   (x,y) ∈ {1,2,4}×{1,3}
P_X,Y(x,y) = 0,        otherwise.

6개 점의 확률 (xy/28):
(1,1)=1/28, (1,3)=3/28, (2,1)=2/28, (2,3)=6/28, (4,1)=4/28, (4,3)=12/28.

### (b) P[Y < X]
y < x 인 점: (2,1): 1<2 ✓, (4,1): 1<4 ✓, (4,3): 3<4 ✓.
P[Y<X] = (2 + 4 + 12)/28 = 18/28 = 9/14.

### (c) P[Y > X]
y > x 인 점: (1,3): 3>1 ✓, (2,3): 3>2 ✓.
P[Y>X] = (3 + 6)/28 = 9/28.

### (d) P[Y = X]
y = x 인 점: x∈{1,2,4}, y∈{1,3} 이 같아지는 경우는 (1,1) 뿐.
P[Y=X] = 1/28.

### (e) P[Y = 3]
y=3 인 점: (1,3), (2,3), (4,3).
P[Y=3] = (3 + 6 + 12)/28 = 21/28 = 3/4.

## 5. 검산·직관
- 전체 합: (1+3+2+6+4+12)/28 = 28/28 = 1 ✓.
- 삼분할 점검: P[Y<X] + P[Y>X] + P[Y=X] = 18/28 + 9/28 + 1/28 = 28/28 = 1 ✓ (세 사건이 표본공간을 분할).
- P[Y=3] = 3/4: y=3 점들의 xy가 y=1 점들보다 3배 크므로(각 x에서 3y vs y), P[Y=3]/P[Y=1] = 3 → P[Y=3] = 3/4, P[Y=1] = 1/4 ✓.

## 6. 한 줄 요약
> 분리형 결합 PMF cxy는 Σ가 (Σx)(Σy)=28로 인수분해돼 c=1/28이 즉시 나오고, 각 사건은 6개 격자점을 부등호로 분류해 확률을 합산한다.
