## 문제
X와 Y는 결합정규(jointly normal) 확률변수로 파라미터가 μ_X = −1, σ_X^2 = 4, μ_Y = 1, σ_Y^2 = 1, ρ = −1/2 이다.

a. P(X + 2Y ≤ 3) 을 구하라.
b. Cov(X − Y, X + 2Y) 를 구하라.

## 1. 문제 정리
주어진 것:
- μ_X = −1, σ_X^2 = 4 (σ_X = 2)
- μ_Y = 1, σ_Y^2 = 1 (σ_Y = 1)
- 상관계수 ρ = ρ(X,Y) = −1/2
- 따라서 Cov(X,Y) = ρ·σ_X·σ_Y = (−1/2)·2·1 = −1

구할 것:
- (a) P(X + 2Y ≤ 3)
- (b) Cov(X − Y, X + 2Y)

## 2. 무엇을 묻고 왜 이 도구인가
결합정규의 두 가지 핵심 성질을 시험한다.
(a) "결합정규 RV들의 선형결합은 다시 정규분포(univariate normal)이다" — 그래서 W = X + 2Y 의 평균·분산만 구하면 표준화 Φ로 확률을 읽을 수 있다.
(b) Covariance는 쌍선형(bilinear) 연산자다. 정리노트의 상관계수 정의 ρ(X,Y) = Cov(X,Y)/√(var[X]var[Y]) (Lecture08, p.6) 에서 Cov(X,Y)를 역산한 뒤, Cov의 쌍선형성으로 전개한다.

상관계수↔공분산 변환과 공분산의 쌍선형 전개는 Lecture08 범위다. 단, "선형결합이 정규분포가 된다"는 결합정규의 정의적 성질은 Lecture08 정리노트에 명시 없음 → ⚠ 강의 밖(결합정규 고유 성질), 일반 교과서 지식으로 보강.

## 3. 핵심 통찰
W = aX + bY (결합정규)는 정규분포이고, 평균 = a·μ_X + b·μ_Y, 분산 = a^2 σ_X^2 + b^2 σ_Y^2 + 2ab·Cov(X,Y). 공분산은 ρ로부터 Cov(X,Y) = ρσ_Xσ_Y 로 한 번에 나온다. (b)는 Cov를 4개 항으로 쪼개는 쌍선형 전개만으로 끝난다.

## 4. 풀이

먼저 공분산 (두 부분 공통 재료):
Cov(X,Y) = ρ·σ_X·σ_Y = (−1/2)·(2)·(1) = −1.   (Lecture08, p.6 의 ρ 정의를 역산)

### (a) P(X + 2Y ≤ 3)
W = X + 2Y 로 둔다. 결합정규의 선형결합이므로 W ~ Normal(μ_W, σ_W^2). (⚠ 강의 밖: 결합정규 고유 성질)

평균:
μ_W = μ_X + 2μ_Y = (−1) + 2(1) = 1.

분산 (a=1, b=2):
σ_W^2 = (1)^2·σ_X^2 + (2)^2·σ_Y^2 + 2·(1)·(2)·Cov(X,Y)
      = 4 + 4·1 + 4·(−1)
      = 4 + 4 − 4 = 4.
σ_W = 2.

표준화:
P(W ≤ 3) = Φ((3 − μ_W)/σ_W) = Φ((3 − 1)/2) = Φ(1).

Φ(1) ≈ 0.8413.

따라서 P(X + 2Y ≤ 3) = Φ(1) ≈ 0.8413.

### (b) Cov(X − Y, X + 2Y)
공분산의 쌍선형성(bilinearity)으로 전개. 필요한 분산 사실: Cov(Z,Z) = var[Z].

Cov(X − Y, X + 2Y)
 = Cov(X, X) + 2·Cov(X, Y) − Cov(Y, X) − 2·Cov(Y, Y)
 = var[X] + 2·Cov(X,Y) − Cov(X,Y) − 2·var[Y]
 = var[X] + Cov(X,Y) − 2·var[Y]
 = 4 + (−1) − 2·(1)
 = 4 − 1 − 2 = 1.

따라서 Cov(X − Y, X + 2Y) = 1.

(이 문제에는 새로 정의되는 PMF/PDF가 없다. W의 분포는 Normal(1, 4) 의 표준 정규 PDF이며 그 외 정의역에서 추가 명시할 piecewise 경계는 없다 — 정규분포는 전 실수에서 양의 밀도를 가지므로 "otherwise = 0" 구간이 존재하지 않는다.)

## 5. 검산·직관
- (a) σ_W^2 = 4 > 0 ✓ (분산은 양수여야 한다). ρ = −1/2 의 음의 상관이 분산을 4+4 에서 4로 깎았는데, X와 2Y가 음으로 상관되어 합의 변동이 줄어든 것 — 직관과 일치.
- (a) 임계값 3이 평균 μ_W=1 보다 위(+1σ)에 있으므로 P(W≤3) > 0.5 여야 한다. Φ(1)=0.8413 > 0.5 ✓.
- (b) X−Y 와 X+2Y 가 양의 공분산(=1)을 가짐: 둘 다 X에 양의 계수로 의존하므로(var[X]=4 가 지배적) 양수가 자연스럽다 ✓.

## 6. 한 줄 요약
> 결합정규의 선형결합은 다시 정규분포이므로 평균·분산만 구해 Φ로 확률을 읽고, 공분산은 ρσ_Xσ_Y로 역산한 뒤 쌍선형 전개로 마무리한다.
