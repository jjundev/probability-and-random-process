## 문제
Problem 5.2.2의 확률변수 X와 Y에 대해 다음을 구하라. (Problem 5.3.2의 결과를 참고하라.)
(a) W = 2^(XY) 의 기댓값
(b) 상관(correlation) r_{X,Y} = E[XY]
(c) 공분산 Cov[X,Y]
(d) 상관계수 ρ_{X,Y}
(e) Var[X + Y]

참고 — Problem 5.2.2의 결합 PMF:
P_{X,Y}(x,y) = c·|x + y|,   x = −2, 0, 2;  y = −1, 0, 1
P_{X,Y}(x,y) = 0,   otherwise

## 1. 문제 정리
| 항목 | 내용 |
|---|---|
| 주어진 것 | P_{X,Y}(x,y) = c|x+y| (격자 x∈{−2,0,2}, y∈{−1,0,1}), 그 외 0 |
| 먼저 할 일 | c 결정 → 결합 PMF 확정 → 주변 PMF·모멘트 |
| 구할 것 | (a) E[2^(XY)] (b) E[XY] (c) Cov (d) ρ (e) Var(X+Y) |

## 2. 무엇을 묻고 왜 이 도구인가
모두 이산 결합분포의 기댓값 계산이다. 핵심 도구는 LOTUS(법칙: E[g(X,Y)] = Σ_{x,y} g(x,y)·P_{X,Y}(x,y))와 공분산 공식 Cov = E[XY] − E[X]E[Y] (Lecture07, p.23), 상관계수 ρ = Cov/√(VarX·VarY) (Lecture08, p.6), 그리고 Var(X+Y) = Var(X) + Var(Y) + 2Cov(X,Y) (Lecture07, p.25). 격자의 대칭성을 미리 보면 1차 모멘트가 0이라 계산이 크게 줄어든다.

## 3. 핵심 통찰
|x+y|·격자는 x → −x, y → −y 에 대해 대칭이다. 따라서 X와 Y의 분포가 0에 대칭 → E[X] = E[Y] = 0. 그러면 Cov = E[XY], r = E[XY]가 같아지고(b=c), 계산은 E[XY]·E[X²]·E[Y²] 세 개로 축소된다.

## 4. 풀이

먼저 정규화 상수 c. 각 칸의 |x+y|:
  x=−2:  |−3|=3, |−2|=2, |−1|=1
  x= 0:  |−1|=1, |0|=0, |1|=1
  x= 2:  |1|=1,  |2|=2,  |3|=3
합 Σ|x+y| = (3+2+1) + (1+0+1) + (1+2+3) = 6 + 2 + 6 = 14
c·14 = 1  ⇒  c = 1/14.

확정된 결합 PMF (분모 14, otherwise = 0):
            Y=−1     Y=0      Y=1
  X=−2      3/14     2/14     1/14
  X= 0      1/14      0       1/14
  X= 2      1/14     2/14     3/14

주변 PMF:
P_X(−2)=6/14, P_X(0)=2/14, P_X(2)=6/14,   otherwise 0
P_Y(−1)=5/14, P_Y(0)=4/14, P_Y(1)=5/14,   otherwise 0

대칭이므로:
E[X] = (−2)(6/14) + 0 + (2)(6/14) = 0
E[Y] = (−1)(5/14) + 0 + (1)(5/14) = 0

(b) 상관 r_{X,Y} = E[XY] — 각 칸 xy·P 합 (xy=0 칸은 생략):
(−2,−1): xy=2,  6/14 기여 →  +6/14
(−2, 1): xy=−2, →  −2/14
(2, −1): xy=−2, →  −2/14
(2,  1): xy=2,  →  +6/14
E[XY] = (6 − 2 − 2 + 6)/14 = 8/14 = 4/7 ≈ 0.5714
∴ r_{X,Y} = 4/7.

(c) 공분산:
Cov[X,Y] = E[XY] − E[X]E[Y] = 4/7 − 0·0 = 4/7 ≈ 0.5714

(d) 상관계수 — 분산부터:
E[X²] = 4·(6/14) + 0 + 4·(6/14) = 48/14 = 24/7 → Var(X) = 24/7 (E[X]=0)
E[Y²] = 1·(5/14) + 0 + 1·(5/14) = 10/14 = 5/7 → Var(Y) = 5/7
ρ_{X,Y} = Cov / √(Var(X)·Var(Y)) = (4/7) / √((24/7)(5/7)) = (4/7) / (√120 / 7) = 4/√120 = 2/√30 ≈ 0.3651

(e) 합의 분산:
Var(X+Y) = Var(X) + Var(Y) + 2·Cov(X,Y) = 24/7 + 5/7 + 2·(4/7) = (24+5+8)/7 = 37/7 ≈ 5.286

(a) E[W] = E[2^(XY)] — LOTUS로 각 칸 2^(xy)·P 합:
XY=2 칸 (P=3/14 두 곳): 2² = 4 → 4·(3/14)·2 = 24/14
XY=0 칸 (P = 2/14, 1/14, 1/14, 2/14 = 합 6/14): 2⁰ = 1 → 6/14
XY=−2 칸 (P=1/14 두 곳): 2^(−2) = 1/4 → (1/4)·(2/14) = (1/2)/14
E[W] = 24/14 + 6/14 + (1/2)/14 = (24 + 6 + 0.5)/14 = 30.5/14 = 61/28 ≈ 2.179

## 5. 검산·직관
- 결합 PMF 합: (6+2+6)/14 = 14/14 = 1 ✓.
- E[X]=E[Y]=0 (대칭) → r = Cov (b=c) 자동 일치 ✓.
- |ρ| = 2/√30 ≈ 0.365 ≤ 1 유계 ✓ (Lecture08, p.6). 양수 → X와 Y가 같은 부호로 약하게 동행 (대각의 큰 확률 3/14가 (−2,−1),(2,1)에 몰림).
- Var(X+Y) = 37/7 > Var(X)+Var(Y) = 29/7, 차이 8/7 = 2Cov ✓.
- E[W]: 큰 항 2²=4가 확률 6/14로 지배 → 평균 약 2.18, XY가 음수일 때 2^(XY)<1로 작아지는 효과까지 반영 ✓.

## 6. 한 줄 요약
> 격자가 (x,y)→(−x,−y) 대칭이면 E[X]=E[Y]=0 → Cov=E[XY]; 나머지는 LOTUS로 E[XY], E[X²], E[Y²], E[2^(XY)]를 칸별 합으로 긁으면 된다.
