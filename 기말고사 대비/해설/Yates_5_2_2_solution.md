## 문제
확률변수 X, Y 의 결합 PMF 가

P_{X,Y}(x, y) = c|x + y|,  x = −2, 0, 2;  y = −1, 0, 1
P_{X,Y}(x, y) = 0,         otherwise

(a) 상수 c 의 값은?
(b) P[Y < X] 는?
(c) P[Y > X] 는?
(d) P[Y = X] 는?
(e) P[X < 1] 는?

## 1. 문제 정리
- 정의역: x ∈ {−2, 0, 2}, y ∈ {−1, 0, 1} (9 개 격자점).
- 각 점의 비정규화 가중치 = |x + y|.
- 구할 것: (a) 정규화 상수 c, (b)–(e) 네 가지 사건 확률.

## 2. 무엇을 묻고 왜 이 도구인가
결합 PMF 는 모든 격자점 확률의 합이 1 이어야 한다 (Lecture03, p.20: Σ_x Σ_y p_{X,Y} = 1). 이로부터 c 를 정한다. 이후 각 사건은 조건을 만족하는 격자점의 확률을 더하면 끝 — 이산이라 적분 없이 표(table) 합산. 가중치 표를 한 번 만들어두면 (b)–(e) 가 모두 그 표의 부분합이다.

## 3. 핵심 통찰
9 개 점의 |x+y| 합이 14 라서 c = 1/14. 그러면 각 점 확률은 |x+y|/14. 사건 확률은 표에서 해당 칸의 분자(|x+y|)만 더해 14 로 나누면 된다. 대칭성(|x+y| 는 (x,y) → (−x,−y) 에 불변)이 Y<X 와 Y>X 를 같게 만든다.

## 4. 풀이

### 가중치 표 |x + y| (행 x, 열 y)
| x \\ y | −1 | 0 | 1 |
|---|---|---|---|
| −2 | 3 | 2 | 1 |
| 0  | 1 | 0 | 1 |
| 2  | 1 | 2 | 3 |

### (a) 상수 c
전체 합 = (3+2+1) + (1+0+1) + (1+2+3) = 6 + 2 + 6 = 14.
c · 14 = 1 ⇒ c = 1/14.
따라서 P_{X,Y}(x,y) = |x+y|/14 (위 9 점), otherwise = 0.

### (b) P[Y < X]
y < x 인 점의 분자 합: (0,−1)=1, (2,−1)=1, (2,0)=2, (2,1)=3 → 합 7.
P[Y < X] = 7/14 = 1/2.

### (c) P[Y > X]
y > x 인 점의 분자 합: (−2,−1)=3, (−2,0)=2, (−2,1)=1, (0,1)=1 → 합 7.
P[Y > X] = 7/14 = 1/2.

### (d) P[Y = X]
y = x 가능한 점은 (0,0) 뿐 (다른 x=±2 는 y 범위 밖). 그 분자 = |0| = 0.
P[Y = X] = 0/14 = 0.

### (e) P[X < 1]
x < 1 ⇒ x ∈ {−2, 0}. 해당 행 전체 합: 행 −2 = 6, 행 0 = 2 → 합 8.
P[X < 1] = 8/14 = 4/7.

## 5. 검산·직관
- 분할 점검: P[Y<X] + P[Y>X] + P[Y=X] = 1/2 + 1/2 + 0 = 1 ✓ (세 사건이 표본공간을 분할).
- 대칭성: 가중치 |x+y| 는 (x,y)→(−x,−y) 대칭이라 표가 180° 회전대칭 → Y<X 와 Y>X 가 똑같이 1/2 인 것이 타당 ✓.
- P[Y=X]=0 직관: x 는 짝수(±2,0), y 는 |y|≤1 이라 둘이 같아질 수 있는 곳은 0 뿐인데 거기서 |x+y|=0 이라 질량이 없음 ✓.
- P[X<1]=4/7: x=2 행(분자합 6)을 제외하므로 8/14, 절반보다 큼 — 정의역의 2/3 (x=−2,0) 를 차지하니 타당 ✓.

## 6. 한 줄 요약
> 결합 PMF 의 c 는 전체 가중치 합(=14)의 역수로 정해지고, 모든 사건 확률은 |x+y| 표의 해당 칸 분자만 더해 14 로 나누면 된다.
