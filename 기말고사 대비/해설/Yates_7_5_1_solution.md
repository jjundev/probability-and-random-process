## 문제
X와 Y는 결합 PDF
f_X,Y(x,y) = 2,   0 ≤ y ≤ x ≤ 1
f_X,Y(x,y) = 0,   otherwise
를 가진다.
주변 PDF f_Y(y), 조건부 PDF f_X|Y(x|y), 조건부 기댓값 E[X|Y=y] 를 구하라.

## 1. 문제 정리
주어진 것:
- f_X,Y(x,y) = 2 (영역 0 ≤ y ≤ x ≤ 1, 즉 단위정사각형의 아래 삼각형), 그 외 0.

구할 것:
- 주변 PDF f_Y(y)
- 조건부 PDF f_X|Y(x|y)
- 조건부 기댓값 E[X|Y=y]

## 2. 무엇을 묻고 왜 이 도구인가
조건부 분포는 결합을 주변으로 나눠 얻는다: f_X|Y(x|y) = f_X,Y(x,y)/f_Y(y) (연속판; Lecture04, slide 8의 이산판 p_X|Y = p_X,Y/p_Y 의 연속 대응). 그러려면 주변 f_Y(y)를 x에 대해 적분해 먼저 구해야 한다. 핵심은 적분 한계 — 영역 0 ≤ y ≤ x ≤ 1 에서 y를 고정하면 x는 y부터 1까지 움직인다. 조건부 기댓값은 정의 E[X|Y=y] = ∫ x·f_X|Y(x|y) dx (Lecture08, p.16 의 조건부 기댓값 개념; 연속판).

## 3. 핵심 통찰
영역 0 ≤ y ≤ x ≤ 1 에서 y를 고정하면 x ∈ [y, 1]. 이 한계만 정확히 잡으면 f_Y(y) = 2(1−y), 그리고 조건부는 [y,1] 위 균등분포 1/(1−y) 가 된다 — 따라서 E[X|Y=y]는 구간 [y,1]의 중점.

## 4. 풀이

### 주변 PDF f_Y(y)
y 고정 시 x ∈ [y, 1] (영역 y ≤ x ≤ 1):
f_Y(y) = ∫_y^1 f_X,Y(x,y) dx = ∫_y^1 2 dx = 2(1 − y),   0 ≤ y ≤ 1.
f_Y(y) = 0,   otherwise.

### 조건부 PDF f_X|Y(x|y)
0 ≤ y < 1 (f_Y(y) > 0) 인 y에 대해:
f_X|Y(x|y) = f_X,Y(x,y) / f_Y(y) = 2 / [2(1 − y)] = 1/(1 − y),   y ≤ x ≤ 1.
f_X|Y(x|y) = 0,   otherwise.

즉 Y=y가 주어지면 X는 구간 [y, 1] 위의 균등분포 Uniform(y, 1) 이다.

### 조건부 기댓값 E[X|Y=y]
균등분포 Uniform(y,1)의 평균은 구간 중점:
E[X|Y=y] = ∫_y^1 x · 1/(1−y) dx = 1/(1−y) · [x^2/2]_y^1 = 1/(1−y) · (1 − y^2)/2.
1 − y^2 = (1−y)(1+y) 이므로:
E[X|Y=y] = (1+y)/2,   0 ≤ y < 1.

## 5. 검산·직관
- f_Y가 정상 PDF인지: ∫_0^1 2(1−y) dy = 2[y − y^2/2]_0^1 = 2(1 − 1/2) = 1 ✓.
- 조건부가 정상 PDF인지: ∫_y^1 1/(1−y) dx = (1−y)/(1−y) = 1 ✓.
- E[X|Y=y] = (1+y)/2 는 [y,1]의 중점 — 균등분포 직관과 일치 ✓. y→1이면 → 1, y=0이면 → 1/2, 모두 [y,1] 안 ✓. 또한 X ≥ Y 제약 때문에 E[X|Y=y] ≥ y 여야 하는데 (1+y)/2 ≥ y ⟺ 1 ≥ y ✓.

## 6. 한 줄 요약
> 결합을 주변으로 나눠 조건부를 만들 때 적분 한계(y 고정 ⟹ x∈[y,1])가 전부 — 결과는 [y,1] 위 균등분포라 E[X|Y=y]는 그 중점 (1+y)/2.
