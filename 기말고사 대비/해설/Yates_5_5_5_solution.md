## 문제
X와 Y는 결합 PDF가 다음과 같은 확률변수다.

f_XY(x,y) = 5x^2/2,   −1 ≤ x ≤ 1, 0 ≤ y ≤ x^2
f_XY(x,y) = 0,   otherwise

(a) 주변 PDF f_X(x)는?
(b) 주변 PDF f_Y(y)는?

## 1. 문제 정리
주어진 것: 결합 PDF f_XY = 5x^2/2. 정의역은 곡선 영역 {−1≤x≤1, 0≤y≤x^2} — y는 위로 포물선 x^2에 의해 제한.
구할 것: (a) f_X(x), (b) f_Y(y).

## 2. 무엇을 묻고 왜 이 도구인가
주변 PDF는 결합 PDF에서 다른 변수를 적분해 소거한다. f_X(x) = ∫ f_XY dy, f_Y(y) = ∫ f_XY dx. (Lecture06, p.11)
핵심은 적분 한계 — 정의역이 직사각형이 아니라 포물선 경계라 적분 한계가 변수에 의존한다. 어느 변수로 적분하느냐에 따라 한계 설정 난이도가 다르다.

## 3. 핵심 통찰
조건 0 ≤ y ≤ x^2를 두 방향으로 읽는다:
- x 고정 → y는 0부터 x^2까지 (f_X 구할 때, 한계가 깔끔).
- y 고정 → x^2 ≥ y ⇔ x ≤ −√y 또는 x ≥ √y (f_Y 구할 때, 영역이 두 조각으로 갈라짐).

## 4. 풀이

**(a) f_X(x)** — x를 고정하고 y를 0부터 x^2까지 적분:

f_X(x) = ∫_0^{x^2} (5x^2/2) dy = (5x^2/2)·x^2 = (5/2)x^4

f_X(x) = (5/2)x^4,   −1 ≤ x ≤ 1
f_X(x) = 0,   otherwise

**(b) f_Y(y)** — y를 고정. 0 ≤ y ≤ x^2이므로 x^2 ≥ y, 즉 x ∈ [−1, −√y] ∪ [√y, 1] (단 0 ≤ y ≤ 1).
피적분함수 5x^2/2는 x의 우함수이므로 두 구간 적분은 오른쪽 구간의 2배:

f_Y(y) = ∫_{−1}^{−√y} (5x^2/2) dx + ∫_{√y}^{1} (5x^2/2) dx = 2·∫_{√y}^{1} (5x^2/2) dx
 = 2·(5/2)·(x^3/3)|_{√y}^{1} = (5/3)·(1 − y^{3/2})

f_Y(y) = (5/3)(1 − y^{3/2}),   0 ≤ y ≤ 1
f_Y(y) = 0,   otherwise

(√y의 세제곱 = y^{3/2}.)

## 5. 검산·직관
- f_X 정규화: ∫_{−1}^1 (5/2)x^4 dx = (5/2)·(2/5) = 1 ✓.
- f_Y 정규화: ∫_0^1 (5/3)(1 − y^{3/2}) dy = (5/3)[1 − (2/5)] = (5/3)(3/5) = 1 ✓.
- f_Y(0) = 5/3 (최대), f_Y(1) = 0: y가 작을수록 그 y를 허용하는 x 범위가 넓어 밀도가 크고, y=1에선 x=±1 한 점뿐이라 0 — 직관과 일치 ✓.
- 두 PDF 모두 정의역에서 비음 ✓.

## 6. 한 줄 요약
> 포물선 경계 0≤y≤x^2를 "x 고정→y 위한계 x^2" / "y 고정→x는 √y 밖 두 조각"으로 양방향 해석하는 것이 주변 PDF 적분 한계 설정의 전부다.
