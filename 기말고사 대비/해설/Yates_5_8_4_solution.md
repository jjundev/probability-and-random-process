## 문제
Quiz 5.3의 확률변수 H와 B에 대해 r_{H,B}와 Cov[H,B]를 구하라.

참고 — Quiz 5.3 (Yates & Goodman)의 결합 PMF P_{H,B}(h,b) (문제은행에 Quiz는 미수록 → 교재 표준 Quiz 5.3 표를 사용):
            B=0      B=2      B=4
  H=−1      0        0.4      0.2
  H= 0      0.1      0        0.1
  H= 1      0.1      0.1      0
P_{H,B} = 0,   otherwise

(주: 위 표는 제공된 문제은행에 Quiz 5.3 본문이 없어 Yates & Goodman 표준 Quiz 5.3에서 복원한 것. 합 = 1로 정합하며, 아래 방법은 표가 달라도 동일하게 적용된다.)

## 1. 문제 정리
- 주어진 것: 위 9칸 결합 PMF (그 외 = 0). H ∈ {−1,0,1}, B ∈ {0,2,4}.
- 구할 것: 상관 r_{H,B} = E[HB], 공분산 Cov[H,B].

## 2. 무엇을 묻고 왜 이 도구인가
상관은 정의상 r_{H,B} = E[HB]이고, 공분산은 Cov[H,B] = E[HB] − E[H]E[B] (Lecture07, p.23). 따라서 E[HB], E[H], E[B] 세 개만 결합·주변 PMF에서 합으로 얻으면 끝난다. 상관(r)과 공분산(Cov)의 차이는 "평균을 빼느냐"뿐임을 보여주는 전형적 문제다.

## 3. 핵심 통찰
E[HB]에서 H=0 행과 B=0 열은 hb=0이라 통째로 사라진다. 0이 아닌 칸은 (−1,2),(−1,4),(1,2) 셋뿐(나머지 비0 확률 칸은 hb=0이거나 확률 0). 이 셋만 더하면 E[HB]가 끝난다.

## 4. 풀이

주변 PMF (행·열 합):
P_H(−1) = 0 + 0.4 + 0.2 = 0.6
P_H(0)  = 0.1 + 0 + 0.1 = 0.2
P_H(1)  = 0.1 + 0.1 + 0 = 0.2     otherwise P_H = 0
P_B(0)  = 0 + 0.1 + 0.1 = 0.2
P_B(2)  = 0.4 + 0 + 0.1 = 0.5
P_B(4)  = 0.2 + 0.1 + 0 = 0.3     otherwise P_B = 0

1차 모멘트:
E[H] = (−1)(0.6) + (0)(0.2) + (1)(0.2) = −0.6 + 0.2 = −0.4
E[B] = (0)(0.2) + (2)(0.5) + (4)(0.3) = 1.0 + 1.2 = 2.2

상관 r_{H,B} = E[HB] — hb·P 합 (hb=0 또는 P=0 칸 제외):
(−1, 2): hb = −2, P = 0.4 → −0.8
(−1, 4): hb = −4, P = 0.2 → −0.8
( 1, 2): hb =  2, P = 0.1 → +0.2
E[HB] = −0.8 − 0.8 + 0.2 = −1.4
∴ r_{H,B} = −1.4

공분산:
Cov[H,B] = E[HB] − E[H]E[B] = −1.4 − (−0.4)(2.2) = −1.4 + 0.88 = −0.52

## 5. 검산·직관
- 결합 PMF 합 = 0.4+0.2+0.1+0.1+0.1+0.1 = 1 ✓.
- 부호: Cov < 0 (음의 상관). H가 작을수록(−1) B가 큰 값(2,4) 쪽에 확률이 몰려 있음 → 반대 방향 동행, 음수 부호 직관 일치.
- r 와 Cov의 차이 = E[H]E[B] = (−0.4)(2.2) = −0.88. r = −1.4, Cov = −1.4 − (−0.88) = −0.52 ✓.
- |Cov|가 |r|보다 작아짐 — 평균을 제거하니 의존성의 "순수" 부분만 남은 것.

## 6. 한 줄 요약
> r_{H,B} = E[HB] (평균 안 뺌), Cov = E[HB] − E[H]E[B] (평균 뺌) — 둘의 차이는 정확히 E[H]E[B]이며, hb≠0인 칸만 더하면 E[HB]가 나온다.
