> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X 의 CDF 가 다음과 같다.

F_X(x) = 0,        x < 0
F_X(x) = x,        0 ≤ x < 1/4
F_X(x) = x + 1/2,  1/4 ≤ x < 1/2
F_X(x) = 1,        x ≥ 1/2

(a) X 의 일반화 PDF(generalized PDF) f_X(x) 를 구하라.
(b) f_X(x) 를 이용해 EX 를 구하라.
(c) f_X(x) 를 이용해 Var(X) 를 구하라.

## 1. 문제 정리
- 주어진 것: 위 CDF F_X(x).
- 구할 것: (a) generalized PDF, (b) EX, (c) Var(X).
- 관찰: 이 CDF 는 **연속 구간 + 점프**가 섞인 혼합형(mixed) RV. x = 1/4 에서 좌극한 1/4 → 값 3/4 로 점프하므로 그 점에 확률질량(point mass)이 있다.

## 2. 무엇을 묻고 왜 이 도구인가
연속 RV 라면 PDF 는 단순히 f_X(x) = dF_X/dx (Lecture05, p.10). 하지만 CDF 가 점프를 가지면 그 점에서 미분 불가 → 이산질량을 **델타 함수(δ)** 로 표현하는 "일반화 PDF"가 필요하다. 일반화 PDF 는 연속부의 도함수에 각 점프점의 (점프 크기)·δ(x − 점프위치) 를 더해 만든다. CDF 성질 "이산: 점프 크기 = p_X(k) = F_X(k) − F_X(k⁻)" (Lecture05, p.10) 와 연속부 미분을 결합하는 셈.

## 3. 핵심 통찰
CDF 의 **점프 높이가 곧 그 점의 확률질량**이다. x = 1/4 에서 점프 = 3/4 − 1/4 = 1/2 → P(X = 1/4) = 1/2, 나머지 [0, 1/2) 구간은 기울기 1 의 연속밀도. 즉 X 는 "확률 1/2 로 정확히 1/4, 확률 1/2 로 [0,1/2) 위 균등"인 혼합 RV.

## 4. 풀이

### (a) 일반화 PDF
연속부 미분: 0 ≤ x < 1/4 에서 dF/dx = 1, 1/4 ≤ x < 1/2 에서 d(x+1/2)/dx = 1. 두 구간 모두 밀도 1 → 합치면 [0, 1/2) 에서 밀도 1.

점프 점검:
- x = 0: F(0) − F(0⁻) = 0 − 0 = 0 (점프 없음)
- x = 1/4: F(1/4) − F(1/4⁻) = (1/4 + 1/2) − 1/4 = 1/2 (점프 1/2 → 질량 1/2)
- x = 1/2: F(1/2) − F(1/2⁻) = 1 − (1/2 + 1/2) = 0 (점프 없음)

따라서 일반화 PDF 는

f_X(x) = 1 + (1/2)·δ(x − 1/4),   0 ≤ x < 1/2
f_X(x) = 0,                       otherwise

(연속밀도 1 은 0 ≤ x < 1/2 에서, δ 항은 x = 1/4 에서만 기여.)

### (b) EX
EX = ∫ x f_X(x) dx = ∫_0^(1/2) x·1 dx + (1/4)·(1/2)
   = [x²/2]_0^(1/2) + 1/8
   = (1/4)/2 + 1/8 = 1/8 + 1/8 = 1/4.

(δ 항은 ∫ x·(1/2)δ(x−1/4) dx = (1/2)·(1/4) = 1/8 로 들어간다.)

EX = 1/4.

### (c) Var(X)
먼저 E[X²] = ∫ x² f_X(x) dx = ∫_0^(1/2) x²·1 dx + (1/4)²·(1/2)
   = [x³/3]_0^(1/2) + (1/16)(1/2)
   = (1/8)/3 + 1/32 = 1/24 + 1/32.

공통분모 96: 1/24 = 4/96, 1/32 = 3/96 → E[X²] = 7/96.

Var(X) = E[X²] − (EX)² = 7/96 − (1/4)² = 7/96 − 1/16 = 7/96 − 6/96 = 1/96.

Var(X) = 1/96 ≈ 0.0104.

## 5. 검산·직관
- 전체 확률: ∫_0^(1/2) 1 dx + 질량 1/2 = 1/2 + 1/2 = 1 ✓ (CDF 가 1 에 도달하는 것과 일치).
- EX = 1/4 직관: 연속부([0,1/2) 균등, 평균 1/4)와 점질량(1/4)이 모두 1/4 근처라 평균이 정확히 1/4 로 떨어짐.
- Var 부호·크기: X 가 [0,1/2] 좁은 구간에 갇혀 있고 절반의 확률이 1/4 한 점에 몰려 있어 분산이 매우 작음(1/96). 만약 점질량 없이 [0,1/2) 순수 균등이라면 Var = (1/2)²/12 = 1/48 인데, 질량이 평균점 1/4 에 몰리며 퍼짐을 줄여 1/96 으로 절반이 된 것이 직관과 일치.

## 6. 한 줄 요약
> CDF 에 점프가 있으면 연속부는 미분, 점프는 (점프높이)·δ 로 더해 "일반화 PDF"를 만들고, 기댓값·분산은 δ 항을 점질량으로 취급해 평소처럼 적분한다.
