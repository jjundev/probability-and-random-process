> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
확률변수 X 와 Y 의 결합 PDF 가 다음과 같다.

f_(X,Y)(x,y) = 1/2,  −1 ≤ x ≤ y ≤ 1
f_(X,Y)(x,y) = 0,    otherwise

비영(nonzero) 확률 영역을 그리고 다음을 답하라.
(a) P[X > 0] 은?
(b) f_X(x) 는?
(c) E[X] 는?

## 1. 문제 정리
주어진 것: 지지 S = {(x,y) : −1 ≤ x ≤ y ≤ 1} 위에서 결합 PDF 가 상수 1/2.
구할 것: (a) X > 0 확률, (b) 주변 PDF f_X, (c) E[X].

지지 영역 그림: 정사각형 [−1,1]×[−1,1] 에서 직선 y = x 위쪽(y ≥ x) 절반. 꼭짓점 (−1,−1),(−1,1),(1,1) 의 직각삼각형. 넓이 = (1/2)·2·2 = 2, 그래서 ∬ (1/2) = 1 ✓.

## 2. 무엇을 묻고 왜 이 도구인가
삼각형 지지 위의 균등 결합분포 → 주변 PDF 추출 + 기댓값. (a) 는 넓이 비, (b) 는 주변화 f_X(x) = ∫ f_(X,Y) dy (Lecture06, p.11) 인데 y 의 적분 한계가 x 에 의존(x ≤ y ≤ 1)하는 게 포인트, (c) 는 함수의 기댓값 E[X] = ∫ x f_X(x) dx (Lecture05, p.7).

## 3. 핵심 통찰
"x 고정 시 y 는 x 부터 1 까지만" — 삼각형 지지라 단면 길이가 (1 − x) 로 변하고, 이게 그대로 f_X(x) = (1/2)(1−x) 라는 비균등 삼각형 주변분포를 만든다.

## 4. 풀이

(a) P[X > 0]
영역 B = S ∩ {x > 0} = {0 < x ≤ y ≤ 1}. 이는 꼭짓점 (0,0),(0,1),(1,1) 의 직각삼각형.
Area(B) = (1/2)·1·1 = 1/2.
P[X > 0] = (1/2)·Area(B) = (1/2)·(1/2) = 1/4.

(b) 주변 PDF f_X(x)
x 고정(−1 ≤ x ≤ 1), y 는 지지 단면 x ≤ y ≤ 1 에서 적분 (Lecture06, p.11):
f_X(x) = ∫_x^1 (1/2) dy = (1/2)(1 − x),  −1 ≤ x ≤ 1.
정리:
f_X(x) = (1 − x)/2,  −1 ≤ x ≤ 1
f_X(x) = 0,          otherwise
(검산: ∫_(−1)^1 (1−x)/2 dx = (1/2)[x − x^2/2]_(−1)^1 = (1/2)[(1 − 1/2) − (−1 − 1/2)] = (1/2)[1/2 + 3/2] = (1/2)(2) = 1 ✓.)

(c) E[X]
E[X] = ∫_(−1)^1 x·f_X(x) dx = ∫_(−1)^1 x·(1−x)/2 dx (Lecture05, p.7).
= (1/2) ∫_(−1)^1 (x − x^2) dx = (1/2) [ x^2/2 − x^3/3 ]_(−1)^1
= (1/2) { (1/2 − 1/3) − (1/2 + 1/3) } = (1/2) { (1/6) − (5/6) }
= (1/2)·(−4/6) = (1/2)·(−2/3) = −1/3.

## 5. 검산·직관
- f_X(x) = (1−x)/2 는 x=−1 에서 1(최대), x=1 에서 0(최소)인 우하향 삼각형 — 지지가 y≥x 라 작은 x 일수록 허용 y 폭(1−x)이 넓어 밀도↑, 직관과 일치.
- E[X] = −1/3 < 0: 질량이 음수 쪽으로 치우친 삼각형이므로 평균이 음수 ✓ (삼각형 주변분포의 무게중심 x = −1 + (2/3)·2 ... 실제로 [−1,1] 우하향 삼각형 평균 = −1 + (1/3)·범위? — 직접 적분값 −1/3 이 정답).
- (a) P[X>0]=1/4: 지지 절반 정사각형에서 x>0 인 작은 삼각형이 전체의 1/4 ✓.

## 6. 한 줄 요약
> 삼각형 지지 균등분포에서 주변 PDF 는 "x 단면 길이(1−x)×밀도"로 비균등하게 나오고(f_X=(1−x)/2), 평균은 그 삼각형 무게중심인 −1/3 이다.
