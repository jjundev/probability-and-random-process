## 문제
두 패킷을 모니터링하는 실험에서 비디오 패킷 수 N의 PMF는 다음과 같다.

n        0     1     2
P_N(n)   0.2   0.7   0.1

E[N], E[N^2], Var[N], σ_N을 구하라.

## 1. 문제 정리
주어진 것: 이산 PMF P_N(0)=0.2, P_N(1)=0.7, P_N(2)=0.1, otherwise = 0.
구할 것: E[N], E[N^2], Var[N], 표준편차 σ_N.

## 2. 무엇을 묻고 왜 이 도구인가
정의 그대로의 계산 문제. E[N] = Σ n·P_N(n), E[g(N)] = Σ g(n)·P_N(n) (Lecture03, p.14, p.16). 분산은 계산식 Var[N] = E[N^2] − (E[N])^2 (Lecture03, p.18; "제곱의 평균 빼기 평균의 제곱"). σ_N = √Var[N].

## 3. 핵심 통찰
Var를 정의 E[(N−μ)^2]로 직접 구하지 말고, E[N]과 E[N^2]을 따로 구한 뒤 "제곱의 평균 − 평균의 제곱"으로 합치는 것이 가장 빠르다. (Lecture03, p.18)

## 4. 풀이

P_N(n) = 0.2, 0.7, 0.1 (n=0,1,2),  otherwise = 0.

E[N] = Σ n·P_N(n) = 0·0.2 + 1·0.7 + 2·0.1 = 0.7 + 0.2 = 0.9

E[N^2] = Σ n^2·P_N(n) = 0^2·0.2 + 1^2·0.7 + 2^2·0.1 = 0 + 0.7 + 0.4 = 1.1

Var[N] = E[N^2] − (E[N])^2 = 1.1 − (0.9)^2 = 1.1 − 0.81 = 0.29

σ_N = √Var[N] = √0.29 ≈ 0.5385

## 5. 검산·직관
- PMF 합: 0.2 + 0.7 + 0.1 = 1 ✓.
- E[N]=0.9는 N의 지지점 [0,2] 안에 있고, 질량이 1에 쏠려(0.7) 평균이 1 근처인 게 타당 ✓ (a ≤ N ≤ b ⇒ a ≤ E[N] ≤ b, Lecture03, p.15).
- Var[N]=0.29 > 0 ✓. N이 거의 1에 모여 퍼짐이 작으므로 분산도 작음 ✓. σ_N ≈ 0.54 < 범위폭 2 ✓.
- 부등식 점검: (E[N])^2 = 0.81 ≤ E[N^2] = 1.1 (Jensen, 항상 성립) ✓.

## 6. 한 줄 요약
> E[N]=0.9, E[N^2]=1.1을 PMF로 직접 더한 뒤 Var = 1.1 − 0.81 = 0.29, σ_N = √0.29 ≈ 0.54 — "제곱의 평균 빼기 평균의 제곱"이 핵심.
