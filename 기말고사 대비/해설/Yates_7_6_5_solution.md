## 문제
반복 기댓값(iterated expectation)을 이용하여 정리 5.19 를 적분 없이(without integrals) 증명하라.

(Yates & Goodman 정리 5.19: X 와 Y 가 독립 확률변수이면 E[XY] = E[X]·E[Y]. 즉 독립 ⇒ 상관 r_{X,Y} = E[X]E[Y], 결국 Cov[X,Y] = 0.)

## 1. 문제 정리
- 주어진 것: X ⊥ Y (독립).
- 증명할 것: E[XY] = E[X]·E[Y].
- 제약: **적분을 쓰지 말 것** — 반복 기댓값 정리만 사용.

## 2. 무엇을 묻고 왜 이 도구인가
적분 증명은 E[XY] = ∫∫ xy f_X(x) f_Y(y) dx dy 를 두 적분의 곱으로 가르는 방식이다. 문제는 이를 **반복 기댓값**(Lecture08, p.17: E[E[X|Y]] = E[X]) 으로 대체하라는 것. 핵심 부품은 두 가지다: (i) 반복 기댓값 정리, (ii) 독립이면 조건이 정보를 주지 않는다는 사실 E[Y | X] = E[Y] (X 를 알아도 Y 의 평균은 안 바뀜).

## 3. 핵심 통찰
X 로 한 번 조건화하면 XY 안의 X 는 "이미 아는 상수"처럼 조건부 기댓값 밖으로 빠진다: E[XY | X] = X·E[Y | X]. 그리고 독립이라 E[Y | X] = E[Y] (상수). 바깥 기댓값을 취하면 그 상수 E[Y] 가 또 빠져나와 E[X]E[Y] 만 남는다. 적분은 어디에도 등장하지 않는다.

## 4. 풀이 (증명)

### 사용할 두 사실
- (F1) 반복 기댓값 정리: 임의의 적분가능 Z 에 대해 E[Z] = E[ E[Z | X] ]. (Lecture08, p.17)
- (F2) 조건부 기댓값에서 "조건변수의 함수"는 상수처럼 밖으로: E[X·Y | X] = X·E[Y | X]. (X = x 로 조건화하면 X 는 값 x 로 고정 — Lecture08, p.16 의 "E[X|Y] 는 Y 의 함수" 정의에서 자연히 따름.)
- (F3) 독립이면 조건이 무의미: X ⊥ Y ⇒ E[Y | X] = E[Y] (Lecture04 의 독립 정의 p_{X,Y}=p_X p_Y → Y 의 조건부 분포가 X 에 무관, 그래서 조건부 평균도 무관).

### 증명
Z = XY 에 (F1):
E[XY] = E[ E[XY | X] ].

안쪽에 (F2):
E[XY | X] = X · E[Y | X].

독립이므로 (F3): E[Y | X] = E[Y] (X 에 무관한 상수). 대입:
E[XY | X] = X · E[Y].

다시 바깥 기댓값 — E[Y] 는 상수라 기댓값의 선형성으로 빠져나옴 (Lecture03, p.16: E[aX]=aE[X]):
E[XY] = E[ X · E[Y] ] = E[Y] · E[X] = E[X]·E[Y].   ∎

적분을 한 번도 쓰지 않고 반복 기댓값·조건부 기댓값의 성질만으로 도착했다.

### 따름 (정리 5.19 의 결론들)
- 상관: r_{X,Y} = E[XY] = E[X]E[Y].
- 공분산: Cov[X,Y] = E[XY] − E[X]E[Y] = 0 → 독립이면 uncorrelated (Lecture07, p.23). (역은 거짓.)

## 5. 검산·직관
- 이산 합으로 즉시 재확인(역시 적분 아님): E[XY] = Σ_x Σ_y xy·p_X(x)p_Y(y) = (Σ_x x p_X(x))(Σ_y y p_Y(y)) = E[X]E[Y]. 반복 기댓값 증명의 골격과 동일하게 "X 부분과 Y 부분이 분리"되는 구조 ✓.
- 직관: 독립이면 X 를 관측해도 Y 의 평균 예측이 바뀌지 않으므로(E[Y|X]=E[Y]) 곱의 평균이 평균의 곱으로 깔끔히 분해된다.
- 한계 주의: 이 증명은 **독립**을 (F3) 에서 본질적으로 사용한다. uncorrelated(Cov=0)만으로는 E[Y|X]=E[Y] 가 보장되지 않아 거꾸로 갈 수 없다 (Lecture07, p.24 의 반례).

## 6. 한 줄 요약
> E[XY] = E[E[XY|X]] = E[X·E[Y|X]] = E[X·E[Y]] = E[X]E[Y] — X 로 조건화해 X 를 밖으로 빼고 독립으로 E[Y|X]=E[Y] 를 쓰면 적분 없이 끝난다.
