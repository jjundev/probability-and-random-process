# Yates 7.6.4 풀이
> 출처: Probability and Stochastic Processes (Yates & Goodman), L08_PSP_7.6.4.png, 1순위 근거 Lecture08, 풀이일 2026-06-08
> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X₁과 X₂가 상관계수 ρ₁₂를 갖는 이변량 가우시안(bivariate Gaussian) PDF를 따르고, 각 X_i는 Gaussian(μ_i, σ_i)이다. Y = X₁X₂의 분산이 다음과 같음을 보여라.
Var[Y] = σ₁²σ₂²(1 + ρ₁₂²) + σ₁²μ₂² + μ₁²σ₂² − μ₁²μ₂².
힌트: Problem 9.2.4를 참고하고, 반복 기댓값(iterated expectation)으로 E[X₁²X₂²] = E[ E[X₁²X₂² | X₂] ] 를 구하라.

> ⚠ 검증 결과 (중요): 아래 유도(반복 기댓값 + 기호적분 + 몬테카를로 2×10⁷ 표본, 모두 일치)에 따르면 올바른 마지막 항은 −μ₁²μ₂² 가 아니라 **+ 2ρ₁₂σ₁σ₂μ₁μ₂** 이다. 즉 교재에 인쇄된 식은 오타로 보이며, 정확한 결과는
> Var[Y] = σ₁²σ₂²(1 + ρ₁₂²) + σ₁²μ₂² + μ₁²σ₂² + 2ρ₁₂σ₁σ₂μ₁μ₂.
> (인쇄식과의 차이 = μ₁μ₂(μ₁μ₂ + 2ρ₁₂σ₁σ₂).) 본 풀이는 올바른 식을 유도한다.

## 1. 문제 정리
- 주어진 것: (X₁, X₂) 이변량 정규, 주변 X_i ~ N(μ_i, σ_i²), 상관계수 ρ₁₂.
- 구할 것: Var[Y], Y = X₁X₂. (증명형)
- 도구 힌트: 이변량 정규의 조건부분포(=Problem 9.2.4) + 반복 기댓값.

## 2. 무엇을 묻고 왜 이 도구인가
Var[Y] = E[Y²] − (E[Y])² 인데, Y² = X₁²X₂² 의 기댓값을 직접 이중적분하면 번거롭다. 반복 기댓값 E[X₁²X₂²] = E[ E[X₁²X₂² | X₂] ] (Lecture08, p.17)을 쓰면, 안쪽에서 X₂는 상수로 빠지고 E[X₁² | X₂] 만 남는다. 이변량 정규의 결정적 성질(Problem 9.2.4): **X₂를 조건으로 한 X₁도 정규**이며 그 평균·분산이 닫힌 형태로 주어진다 — 이것이 계산을 가능케 하는 한 수다.

## 3. 핵심 통찰
이변량 정규에서
E[X₁ | X₂ = x₂] = μ₁ + ρ₁₂(σ₁/σ₂)(x₂ − μ₂),   Var[X₁ | X₂ = x₂] = σ₁²(1 − ρ₁₂²).
즉 조건부 평균은 x₂의 **1차식**, 조건부 분산은 **상수**다. 따라서 E[X₁²|X₂]는 x₂의 2차식이 되고, E[X₂²·(x₂의 2차식)]은 X₂의 2·3·4차 모멘트만 알면 끝난다.

## 4. 풀이

(0) 기댓값 E[Y]. 공분산 정의에서 E[X₁X₂] = Cov(X₁,X₂) + E[X₁]E[X₂] (Lecture07, p.23):
E[Y] = E[X₁X₂] = ρ₁₂σ₁σ₂ + μ₁μ₂.

(1) 조건부 모멘트 (Problem 9.2.4). 약식 표기 ρ = ρ₁₂. X₁ | X₂=x₂ ~ N(m(x₂), v) 로,
m(x₂) = μ₁ + ρ(σ₁/σ₂)(x₂ − μ₂),   v = σ₁²(1 − ρ²).
따라서
E[X₁² | X₂ = x₂] = Var[X₁|X₂] + (E[X₁|X₂])² = v + m(x₂)².

(2) 반복 기댓값으로 E[Y²] = E[X₁²X₂²] (Lecture08, p.17):
E[X₁²X₂²] = E[ X₂² · E[X₁² | X₂] ] = E[ X₂² (v + m(X₂)²) ] = v·E[X₂²] + E[ X₂² m(X₂)² ].

m(x₂)를 1차식으로 정리: m(x₂) = a + b·x₂,  b = ρσ₁/σ₂,  a = μ₁ − ρ(σ₁/σ₂)μ₂.
그러면 m(X₂)² = a² + 2ab·X₂ + b²·X₂² 이고
E[X₂² m(X₂)²] = a²·E[X₂²] + 2ab·E[X₂³] + b²·E[X₂⁴].

(3) X₂ ~ N(μ₂, σ₂²)의 모멘트:
E[X₂²] = μ₂² + σ₂²
E[X₂³] = μ₂³ + 3μ₂σ₂²
E[X₂⁴] = μ₂⁴ + 6μ₂²σ₂² + 3σ₂⁴

(4) 대입·정리. (기호적분으로 검증한 결과) 위 합은 다음으로 정리된다:
E[Y²] = E[X₁²X₂²] = σ₁²σ₂²(1 + 2ρ²) + σ₁²μ₂² + μ₁²σ₂² + 2ρσ₁σ₂μ₁μ₂ + μ₁²μ₂².

(5) 분산:
Var[Y] = E[Y²] − (E[Y])²
(E[Y])² = (ρσ₁σ₂ + μ₁μ₂)² = ρ²σ₁²σ₂² + 2ρσ₁σ₂μ₁μ₂ + μ₁²μ₂².
Var[Y] = [σ₁²σ₂²(1 + 2ρ²) + σ₁²μ₂² + μ₁²σ₂² + 2ρσ₁σ₂μ₁μ₂ + μ₁²μ₂²]
        − [ρ²σ₁²σ₂² + 2ρσ₁σ₂μ₁μ₂ + μ₁²μ₂²]
       = σ₁²σ₂²(1 + ρ²) + σ₁²μ₂² + μ₁²σ₂² + 2ρ₁₂σ₁σ₂μ₁μ₂.

이것이 올바른 결과다. (교재의 −μ₁²μ₂² 자리에 +2ρ₁₂σ₁σ₂μ₁μ₂ 가 와야 함.)

## 5. 검산·직관
- 특수경우 μ₁ = μ₂ = 0 (영평균): Var[Y] = σ₁²σ₂²(1 + ρ²). 이는 영평균 이변량 정규 곱의 분산으로 잘 알려진 값 → ρ=0이면 σ₁²σ₂² (독립 곱의 분산 = E[X₁²]E[X₂²] − 0), ρ=±1이면 2σ₁²σ₂² (Y=±cX₂² 류) ✓. 인쇄식도 이 경우엔 마지막 항이 0이라 일치.
- 수치 검증 (μ₁,μ₂,σ₁,σ₂,ρ) = (1.5, −0.7, 2.0, 1.3, 0.4):
  유도식 = 6.76 + 1.0816 + 3.8025 + 1.96 + (−2.184) = 11.42.
  몬테카를로(2×10⁷) Var[Y] ≈ 11.41 ✓. 인쇄식(−μ₁²μ₂²) = 12.50 ✗ → 인쇄식 오타 확정.
- E[Y] = ρσ₁σ₂ + μ₁μ₂ 도 MC와 일치 (−0.010 vs −0.011) ✓.

## 6. 한 줄 요약
> 이변량 정규의 "조건부도 정규"(E[X₁|X₂] 1차·Var 상수)와 반복 기댓값으로 E[X₁²X₂²]를 X₂ 모멘트로 환원하면 Var[X₁X₂] = σ₁²σ₂²(1+ρ²) + σ₁²μ₂² + μ₁²σ₂² + 2ρσ₁σ₂μ₁μ₂ — 교재의 −μ₁²μ₂²는 오타.
