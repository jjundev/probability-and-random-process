## 문제
이 문제에서 Erlang(n, λ) PDF가 1로 적분됨을 검증한다. n차 Erlang PDF의 적분을 다음으로 표기한다.

I_n = ∫_0^∞ [ λ^n · x^(n−1) · e^(−λx) / (n−1)! ] dx.

먼저, n = 1 인 Erlang PDF가 1로 적분됨을 I_1 = 1 을 직접 확인하여 보여라. 둘째, 부분적분(Appendix B, Math Fact B.10)을 사용하여 I_n = I_(n−1) 임을 보여라.

## 1. 문제 정리
- 주어진 것: Erlang(n, λ) PDF
  f(x) = λ^n·x^(n−1)·e^(−λx)/(n−1),   x ≥ 0  (분모는 (n−1)! 팩토리얼)
  f(x) = 0,   otherwise
  그리고 그 적분 I_n = ∫_0^∞ f(x) dx.
- 구할 것: (1) I_1 = 1 직접 확인, (2) I_n = I_(n−1) (부분적분), 결과적으로 모든 n에 대해 I_n = 1.

## 2. 무엇을 묻고 왜 이 도구인가
PDF의 필수 성질은 ∫ f = 1 (Lecture05, p.5–6). Erlang은 n = 1일 때 Exponential(λ)로 환원되고, Exponential PDF λe^(−λx)의 적분이 1임은 (Lecture05, p.13)에서 확립됐다. 일반 n은 정공법으로 적분이 어려우므로, "차수를 하나 낮추는" 점화식 I_n = I_(n−1)을 부분적분으로 만들면 I_1 = 1을 바닥으로 귀납이 완성된다. 부분적분 ∫ u dv = uv − ∫ v du 는 지수·다항 혼합 적분의 표준 도구다 (Lecture05, p.14, "다항함수를 미분쪽, λe^(−λx)를 적분쪽으로").

## 3. 핵심 통찰
부분적분으로 x^(n−1) 의 차수를 한 칸 내리면 적분 모양이 그대로 I_(n−1) 로 재생산된다 — 경계항 uv는 0과 ∞ 양끝에서 모두 사라진다.

## 4. 풀이

**(1) I_1 = 1 직접 확인:**
n = 1이면 x^(n−1) = x^0 = 1, (n−1)! = 0! = 1 이므로 피적분함수는 λe^(−λx) (= Exponential(λ) PDF).
I_1 = ∫_0^∞ λe^(−λx) dx = [ −e^(−λx) ]_0^∞ = (0) − (−1) = 1. ✓
이는 Exponential CDF F(x) = 1 − e^(−λx), F(∞) = 1 과 일치 (Lecture05, p.13).

**(2) I_n = I_(n−1) (부분적분):**
I_n = ∫_0^∞ [λ^n x^(n−1) e^(−λx)] / (n−1)! dx 에서 부분적분을 적용한다. 다항을 미분쪽, 지수를 적분쪽으로:
  u = x^(n−1)        ⇒ du = (n−1) x^(n−2) dx
  dv = λ e^(−λx) dx  ⇒ v = −e^(−λx)

피적분 상수 λ^(n−1)/(n−1)! 를 앞으로 빼고 (λ^n = λ·λ^(n−1)):
I_n = [λ^(n−1)/(n−1)!] · ∫_0^∞ x^(n−1) · λe^(−λx) dx
   = [λ^(n−1)/(n−1)!] · ( [ x^(n−1)·(−e^(−λx)) ]_0^∞ − ∫_0^∞ (−e^(−λx))·(n−1)x^(n−2) dx )

경계항 [ −x^(n−1) e^(−λx) ]_0^∞:
  x → ∞: 지수 e^(−λx)가 다항 x^(n−1)을 압도 → 0.
  x = 0: x^(n−1) = 0 (n ≥ 2) → 0.
따라서 경계항 = 0.

남은 적분:
I_n = [λ^(n−1)/(n−1)!] · (n−1) · ∫_0^∞ x^(n−2) e^(−λx) dx
   = [λ^(n−1)·(n−1)/(n−1)!] · ∫_0^∞ x^(n−2) e^(−λx) dx.

여기서 (n−1)/(n−1)! = 1/(n−2)! 이므로:
I_n = [λ^(n−1)/(n−2)!] · ∫_0^∞ x^(n−2) e^(−λx) dx.

이제 λ^(n−1) = λ·λ^(n−2)로 한 개를 적분 안의 e와 묶으면 (λe^(−λx) 복원):
I_n = [λ^(n−2)/(n−2)!] · ∫_0^∞ x^(n−2) · λe^(−λx) dx
   = ∫_0^∞ [λ^(n−1) x^((n−1)−1) e^(−λx)] / ((n−1)−1)! dx
   = I_(n−1). ✓

(차수 n → n−1로 정확히 같은 형태가 되었다.)

**결론 (귀납):**
I_n = I_(n−1) = I_(n−2) = ... = I_1 = 1.
따라서 모든 n ≥ 1에 대해 Erlang(n, λ) PDF는 1로 적분된다. ∎

## 5. 검산·직관
- 경계항이 0이 되는 핵심은 "지수가 다항을 이긴다" — x = 0에서도 (n ≥ 2이면) x^(n−1) = 0이라 양 끝 모두 소멸. n = 1만 따로 직접 확인한 이유가 여기 있다(그때는 x^0 = 1이라 경계 논리가 다름).
- 팩토리얼 소거 (n−1)/(n−1)! = 1/(n−2)! 가 차수와 정확히 아귀가 맞아 피적분함수가 한 단계 낮은 Erlang으로 떨어지는 것이 이 점화식의 묘미.

## 6. 한 줄 요약
> Erlang PDF의 정규화는 부분적분으로 차수를 한 칸 내려 I_n = I_(n−1)을 만들고, 바닥 I_1 = 1(=Exponential)에서 귀납으로 마무리한다.
