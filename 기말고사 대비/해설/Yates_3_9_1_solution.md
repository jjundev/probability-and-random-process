## 문제
X를 이항(binomial) (100, 1/2) 확률변수라 하자. E_2 를 X가 완전제곱수(perfect square)인 사건이라 하자. P[E_2] 를 계산하라.

## 1. 문제 정리
- 주어진 것: X ~ Binomial(n = 100, p = 1/2). E_2 = {X가 완전제곱수}.
- 구할 것: P[E_2] = P(X ∈ {0, 1, 4, 9, 16, 25, 36, 49, 64, 81, 100}).

## 2. 무엇을 묻고 왜 이 도구인가
이항 PMF P(X = k) = C(100, k)·(1/2)^k·(1/2)^(100−k) = C(100, k)/2^100 (Lecture03, p.11). 사건 E_2는 서로소인 {X = k} (k가 완전제곱수)들의 합집합이므로, 해당 k들의 PMF를 단순히 더하면 된다 (PMF 합산, Lecture03, p.6). 0 ≤ X ≤ 100 범위 안의 완전제곱수만 열거하는 게 관건.

## 3. 핵심 통찰
p = 1/2라 모든 항이 공통분모 2^100을 갖고 분자는 이항계수 C(100, k)뿐 — 완전제곱수 k의 C(100,k)를 더해 2^100으로 나누면 끝. 그리고 분포가 평균 50에 집중되므로 50에 가까운 완전제곱수 49, 64가 사실상 답을 지배한다.

## 4. 풀이

**Step 1 — 범위 내 완전제곱수 열거:** 0 ≤ k ≤ 100 인 m^2:
0(=0^2), 1(=1^2), 4(=2^2), 9(=3^2), 16(=4^2), 25(=5^2), 36(=6^2), 49(=7^2), 64(=8^2), 81(=9^2), 100(=10^2).
→ 11개.

**Step 2 — 이항 PMF:** (Lecture03, p.11)
P(X = k) = C(100, k)·(1/2)^100,   k = 0, 1, …, 100
P(X = k) = 0,   otherwise

**Step 3 — 합산:** 서로소 사건이므로 (Lecture03, p.6)
P[E_2] = Σ_(m=0)^10 P(X = m^2) = (1/2^100) · Σ_(m=0)^10 C(100, m^2).

각 항의 크기 (분포가 k = 50 근처에 집중, Lecture03 p.11):
P(X=49) ≈ 0.078029   ← 지배 항 (49가 평균 50에 가장 가까움)
P(X=64) ≈ 0.0015597
P(X=36) ≈ 0.0015597
P(X=25) ≈ 1.913×10^(−7)
P(X=81) ≈ 1.044×10^(−10)
P(X=16) ≈ 1.06×10^(−12)
나머지(0,1,4,9,100) ≈ 0 (10^(−18) 이하로 무시 가능)

합:
P[E_2] = 0.078029 + 0.0015597 + 0.0015597 + 1.91×10^(−7) + … ≈ 0.08115.

**결론:** P[E_2] ≈ 0.0811 (약 8.1%).
(정확값: Σ C(100, m^2) / 2^100, 분자 = 102867734705055054467692090431, 2^100 = 1267650600228229401496703205376.)

## 5. 검산·직관
- 우세 항 k = 49가 전체의 96%(0.078/0.0811)를 차지 — 이항(100,1/2)은 평균 50, 표준편차 √(100·1/4) = 5 (Lecture03, p.18의 Var = np(1−p))라, 50에서 ±1σ 안의 완전제곱수 49, 64, 36이 사실상 전부.
- 꼬리의 완전제곱수(0, 1, 100 등)는 P ≈ (1/2)^100 ≈ 10^(−30)로 완전히 무시 가능 ✓.
- 0 < 0.0811 < 1 ✓, 합산이 단조 누적이라 확률 공리 만족.

## 6. 한 줄 요약
> 이항(100,1/2)에서 P[X 완전제곱]은 범위 내 제곱수 11개의 PMF 합 = Σ C(100,m^2)/2^100 ≈ 0.081, 평균 50에 가까운 k=49가 사실상 지배한다.
