# Yates 5.4.1 풀이
> 출처: Probability and Stochastic Processes (Yates & Goodman), L06_PSP_5.4.1.png, 1순위 근거 Lecture06, 풀이일 2026-06-08
> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
확률변수 X 와 Y 의 결합 PDF 가 다음과 같다.

f_(X,Y)(x,y) = c,  x ≥ 0, y ≥ 0, x + y ≤ 1
f_(X,Y)(x,y) = 0,  otherwise

(a) 상수 c 의 값은?
(b) P[X ≤ Y] 는?
(c) P[X + Y ≤ 1/2] 는?

## 1. 문제 정리
주어진 것: 지지 S = {(x,y) : x ≥ 0, y ≥ 0, x + y ≤ 1} (꼭짓점 (0,0),(1,0),(0,1) 의 직각삼각형) 위에서 결합 PDF 가 상수 c.
구할 것: (a) 정규화 상수 c, (b) X ≤ Y 영역 확률, (c) X+Y ≤ 1/2 영역 확률.

상수 PDF → 확률 = (부분 넓이)/(전체 넓이) 의 기하 문제. Lecture06, p.11: P((X,Y)∈B) = ∬_B f_(X,Y) dx dy, 정규화 ∬_S f = 1.

## 2. 무엇을 묻고 왜 이 도구인가
균등(uniform) 결합분포 문제. PDF 가 지지 위에서 상수이므로 정규화 ∬_S c dxdy = c·Area(S) = 1 → c = 1/Area(S) (Lecture06, p.11). 이후 모든 확률은 c × (해당 부분영역의 넓이) = (부분넓이)/(전체넓이) — 적분 대신 삼각형 넓이로 끝난다.

## 3. 핵심 통찰
"상수 밀도면 확률 = 넓이 비(比)" — c 를 구한 뒤 (b)(c) 는 삼각형 안에서 직선으로 잘린 부분영역의 넓이만 재면 된다. 적분을 풀 필요조차 없다.

## 4. 풀이

(a) 상수 c
지지 삼각형의 넓이 Area(S) = (1/2)·밑변·높이 = (1/2)·1·1 = 1/2.
정규화: c·Area(S) = 1 → c·(1/2) = 1 → c = 2.
완전한 결합 PDF:
f_(X,Y)(x,y) = 2,  x ≥ 0, y ≥ 0, x + y ≤ 1
f_(X,Y)(x,y) = 0,  otherwise
(검산: ∬_S 2 dxdy = 2·(1/2) = 1 ✓.)

(b) P[X ≤ Y]
영역 B = S ∩ {x ≤ y}. 삼각형 S 를 직선 y = x 로 자른다. y = x 는 S 를 대칭으로 이등분한다(꼭짓점 (1,0) 과 (0,1) 이 직선 y=x 에 대해 거울상; 변 x+y=1 과 y=x 의 교점은 (1/2,1/2)). 따라서 {x ≤ y} 쪽 = 절반.
Area(B) = (1/2)·Area(S) = (1/2)·(1/2) = 1/4.
P[X ≤ Y] = c·Area(B) = 2·(1/4) = 1/2.
(대칭성 직답: f 가 x↔y 교환에 불변이므로 P[X≤Y] = P[Y≤X], 합이 1(경계 X=Y 는 확률 0)이라 각 1/2.)

(c) P[X + Y ≤ 1/2]
영역 C = S ∩ {x+y ≤ 1/2} = {x ≥ 0, y ≥ 0, x+y ≤ 1/2}. 이것은 꼭짓점 (0,0),(1/2,0),(0,1/2) 의 작은 직각삼각형(원삼각형을 1/2 로 축소).
Area(C) = (1/2)·(1/2)·(1/2) = 1/8.
P[X+Y ≤ 1/2] = c·Area(C) = 2·(1/8) = 1/4.
(닮음비 1/2 → 넓이비 (1/2)^2 = 1/4, 그래서 P = 1/4 로도 즉시 확인.)

## 5. 검산·직관
- c = 2 > 1: PDF 는 밀도라 1 을 넘을 수 있다 (Lecture05, p.5 의 "density 가 1 보다 커질 수 있다") — 정상.
- (b) 대칭성 두 방법(넓이 이등분 / 교환 불변)이 모두 1/2 로 일치 ✓.
- (c) 닮음비 1/2 → 넓이 1/4 배 → 확률 1/4. 전체의 1/4 영역에 균등 질량이니 직관적.

## 6. 한 줄 요약
> 지지 위에서 상수인 결합 PDF 는 c = 1/Area 로 정규화되고(여기서 c=2), 모든 확률은 적분 없이 "부분영역 넓이 / 전체 넓이" 비로 구한다.
