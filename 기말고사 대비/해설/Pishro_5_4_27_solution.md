## 문제
X와 Y가 독립인 Uniform(0,1) 확률변수이고 Z = X/Y 라 하자. Z의 CDF와 PDF를 구하라.

## 1. 문제 정리
- 주어진 것: X, Y ~ Uniform(0,1) 독립. 결합밀도 f_(X,Y)(x,y) = 1 on 단위정사각형 (0,1)×(0,1), 0 otherwise.
- 구할 것: F_Z(z), f_Z(z), where Z = X/Y.
- support: X, Y ∈ (0,1) 이고 둘 다 양수이므로 Z = X/Y > 0. Z는 (0, ∞) 전체를 취한다 (Y가 작으면 Z 매우 큼).

## 2. 무엇을 묻고 왜 이 도구인가
Z = g(X,Y) 형 파생분포. 원칙대로 **CDF부터** 구하고 미분한다. (Lecture07, p.10) 정리노트의 예시 1은 Z = Y/X 의 PDF를 단위정사각형 위 면적으로 푼다 (Lecture07, p.12). 본 문제는 Z = X/Y 로 분자·분모만 바뀌었는데, X와 Y가 **동일분포·독립**이라 역할이 대칭이므로 분포는 정확히 같다. 같은 면적-적분 기법을 쓴다.

## 3. 핵심 통찰
"X/Y ≤ z" ⇔ "X ≤ zY" 로 정리하면, 단위정사각형에서 원점을 지나는 직선 x = zy (기울기 측면에서 y = x/z) **아래/위 영역의 면적**을 재는 문제가 된다. 균등분포라 확률 = 면적. 직선이 정사각형을 자르는 방식이 z=1을 기준으로 바뀌므로 두 경우로 나눈다. (Lecture07, p.12; STT 260518, "원점에서 그 점을 잇는 직선의 기울기")

## 4. 풀이

밀도가 단위정사각형에서 1이므로 F_Z(z) = P(X ≤ zY) = (영역 {(x,y) ∈ (0,1)² : x ≤ z y} 의 면적). z > 0 에서 두 경우.

경계선은 x = z y. (x, y) 평면에서 보면 기울기 1/z 인 직선 y = x/z, 또는 동등하게 x = zy.

case 1) 0 < z ≤ 1:
직선 x = zy 는 (x,y)=(0,0) 에서 (z,1) 로 간다 (y=1 일 때 x=z ≤ 1, 즉 정사각형 오른쪽 변에 닿기 전 위쪽 변에서 빠져나감). 부등식 x ≤ zy 를 만족하는 영역은 직선의 **왼쪽(위쪽)** 삼각형. 꼭짓점 (0,0), (0,1), (z,1). 면적 = (1/2)·밑변·높이 = (1/2)·z·1 = z/2.

F_Z(z) = z/2,   0 < z ≤ 1.

case 2) z > 1:
이번엔 x ≤ zy 를 만족하지 **않는** 영역(x > zy)이 작은 삼각형이 되어 그것을 1에서 빼는 게 쉽다. x > zy ⇔ y < x/z. 직선 y = x/z 는 (0,0)에서 (1, 1/z) 로 감 (x=1 일 때 y=1/z < 1). 그 아래 삼각형 꼭짓점 (0,0), (1,0), (1, 1/z), 면적 = (1/2)·1·(1/z) = 1/(2z).

F_Z(z) = 1 − 1/(2z),   z > 1.

정리하면

F_Z(z) = 0,   z ≤ 0
F_Z(z) = z/2,   0 < z ≤ 1
F_Z(z) = 1 − 1/(2z),   z > 1

(연속성 점검: z=1 에서 좌측 z/2 = 1/2, 우측 1 − 1/2 = 1/2 ✓. z→∞ 에서 → 1 ✓.)

PDF는 미분.
- 0 < z < 1: d/dz (z/2) = 1/2.
- z > 1: d/dz (1 − 1/(2z)) = 1/(2z²).

f_Z(z) = 1/2,   0 < z ≤ 1
f_Z(z) = 1/(2z²),   z > 1
f_Z(z) = 0,   otherwise

## 5. 검산·직관
- 정규화: ∫_0^1 (1/2) dz + ∫_1^∞ 1/(2z²) dz = 1/2 + (1/2)[−1/z]_1^∞ = 1/2 + (1/2)(0 − (−1)) = 1/2 + 1/2 = 1 ✓.
- 대칭성 점검: X, Y 가 i.i.d. 이므로 Z = X/Y 와 1/Z = Y/X 는 같은 분포여야 한다. P(Z ≤ 1) = F_Z(1) = 1/2 ✓ (X<Y와 X>Y가 대칭이라 정확히 1/2). 정리노트 예시(Z=Y/X)와 CDF·PDF가 완전히 동일 (Lecture07, p.12) ✓.
- 직관: 0<z<1 구간은 밀도가 1/2로 평평 — Y가 X보다 큰 "보통" 비율들. z가 커질수록 1/(2z²)로 빠르게 감소 — 분모 Y가 0에 가까워야 큰 비율이 나오는데 그럴 확률이 작음. E[Z] = ∫_0^1 z·(1/2) dz + ∫_1^∞ z·1/(2z²) dz = 1/4 + (1/2)∫_1^∞ (1/z) dz = ∞ — 평균이 발산함은 유명한 사실(작은 분모의 꼬리).

## 6. 한 줄 요약
> 비율 Z = X/Y 의 분포는 "X ≤ zY"를 단위정사각형 위 직선이 자르는 면적으로 읽어 CDF를 만들며(z=1 기준 두 경우), i.i.d. 대칭 덕에 X/Y와 Y/X가 동일 분포다.
