## 문제
확률변수 X, Y 의 결합 PMF 가 다음 격자점에서 주어진다 (그 외 = 0):

P_{X,Y}(1,1)=1/4
P_{X,Y}(2,1)=1/8,  P_{X,Y}(2,2)=1/8
P_{X,Y}(3,1)=1/12, P_{X,Y}(3,2)=1/12, P_{X,Y}(3,3)=1/12
P_{X,Y}(4,1)=1/16, P_{X,Y}(4,2)=1/16, P_{X,Y}(4,3)=1/16, P_{X,Y}(4,4)=1/16

다음을 구하라.
(a) 기댓값 E[X], E[Y]
(b) 분산 Var[X], Var[Y]
(c) 상관 r_{X,Y} = E[XY]
(d) 공분산 Cov[X, Y]
(e) 상관계수 ρ_{X,Y}

## 1. 문제 정리
| 구할 것 | 도구 |
|---|---|
| (a) E[X], E[Y] | 주변 PMF → 가중합 |
| (b) Var[X], Var[Y] | E[·²] − (E[·])² |
| (c) E[XY] | Σ xy P(x,y) |
| (d) Cov | E[XY] − E[X]E[Y] |
| (e) ρ | Cov / √(VarX · VarY) |

## 2. 무엇을 묻고 왜 이 도구인가
먼저 주변 PMF 를 marginalize 로 뽑는다 (Lecture03, p.20). 그다음 Cov = E[XY] − E[X]E[Y] (Lecture07, p.23) 와 ρ = Cov/√(VarX·VarY) (Lecture08, p.6). 결합 PMF 가 격자라 모든 합은 유한합이다.

## 3. 핵심 통찰
열(x 고정)별로 더하면 p_X(x) = 1/4 (x=1,2,3,4) — **X 가 균등분포**로 깔끔히 떨어진다. 덕분에 E[X], Var[X] 는 균등분포 공식으로 즉답되고, 나머지는 행(y)별 가중합과 E[XY] 격자합으로 마무리.

## 4. 풀이

### 주변 PMF
열 합 (각 x 의 세로줄):
p_X(1)=1/4, p_X(2)=1/8+1/8=1/4, p_X(3)=3·(1/12)=1/4, p_X(4)=4·(1/16)=1/4.
⇒ X ~ Uniform{1,2,3,4}, p_X(x)=1/4 (x=1,2,3,4), otherwise = 0.

행 합 (공통분모 48):
p_Y(1)=1/4+1/8+1/12+1/16 = 25/48
p_Y(2)=1/8+1/12+1/16 = 13/48
p_Y(3)=1/12+1/16 = 7/48
p_Y(4)=1/16 = 3/48
otherwise = 0.  (합 = 48/48 = 1 ✓)

### (a) 기댓값
E[X] = (1+2+3+4)/4 = 2.5.
E[Y] = (1·25 + 2·13 + 3·7 + 4·3)/48 = (25+26+21+12)/48 = 84/48 = 7/4 = 1.75.

### (b) 분산
E[X²] = (1+4+9+16)/4 = 30/4 = 7.5 → Var[X] = 7.5 − 2.5² = 7.5 − 6.25 = 1.25 = 5/4.
E[Y²] = (1·25 + 4·13 + 9·7 + 16·3)/48 = (25+52+63+48)/48 = 188/48 = 47/12.
Var[Y] = 47/12 − (7/4)² = 188/48 − 147/48 = 41/48 ≈ 0.8542.

### (c) 상관 r_{X,Y} = E[XY]
행별로 합산 (xy·P 를 더함):
y=1: (1·1/4)+(2·1/8)+(3·1/12)+(4·1/16) = 1/4+1/4+1/4+1/4 = 1
y=2: (4·1/8)+(6·1/12)+(8·1/16) = 1/2+1/2+1/2 = 3/2
y=3: (9·1/12)+(12·1/16) = 3/4+3/4 = 3/2
y=4: (16·1/16) = 1
E[XY] = 1 + 3/2 + 3/2 + 1 = 5.

### (d) 공분산
Cov[X,Y] = E[XY] − E[X]E[Y] = 5 − (2.5)(1.75) = 5 − 4.375 = 0.625 = 5/8.

### (e) 상관계수
ρ_{X,Y} = Cov / √(Var[X]·Var[Y]) = (5/8) / √((5/4)(41/48))
  = (5/8) / √(205/192) = 5√3 / √205 ≈ 0.605.
(ρ² = 75/205 = 15/41 = 0.3659 → ρ ≈ 0.6048.)

## 5. 검산·직관
- 정규화: 열 합·행 합 모두 1 ✓.
- ρ 범위: 0 < ρ ≈ 0.605 < 1 ✓ (Lecture08, p.6 의 −1 ≤ ρ ≤ 1). 양수 → X 가 크면 Y 도 큰 경향: 격자가 우상향 삼각형(큰 x 일수록 y 가 높이 닿음)이라 양의 상관이 직관과 일치.
- Cov 부호: E[XY]=5 > E[X]E[Y]=4.375 → 양수 ✓.

## 6. 한 줄 요약
> 결합 PMF 의 열 합이 균등분포(X)로 떨어지면 X 의 모멘트는 공식으로 즉답되고, 나머지는 E[XY] 격자합 → Cov → ρ 의 표준 사슬로 마무리한다.
