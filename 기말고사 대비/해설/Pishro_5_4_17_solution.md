> 수식 표기: PDF용 유니코드 평문 (solve-problem 기본 $$ 대비 의도적 평문화)

## 문제
X, Y 가 결합연속(jointly continuous) 확률변수이고 결합 PDF 가 다음과 같다.

f_(X,Y)(x,y) = e^(−xy),  1 ≤ x ≤ e, y > 0
f_(X,Y)(x,y) = 0,        otherwise

(a) 주변(marginal) PDF f_X(x) 와 f_Y(y) 를 구하라.
(b) P(0 ≤ Y ≤ 1, 1 ≤ X ≤ √e) 를 계산하는 적분을 써라.

## 1. 문제 정리
주어진 것: 지지(support) S = {(x,y) : 1 ≤ x ≤ e, y > 0} 위에서 f_(X,Y)(x,y) = e^(−xy), 그 밖은 0.
구할 것: (a) 두 주변 PDF, (b) 직사각형 영역 확률의 적분식.

주변화 공식(Lecture06, p.11): f_X(x) = ∫ f_(X,Y)(x,y) dy, f_Y(y) = ∫ f_(X,Y)(x,y) dx. 핵심은 e^(−xy) 가 변수분리가 안 되므로(곱 형태 아님) 독립이 아니고, 적분 한계를 지지에서 정확히 읽는 것.

## 2. 무엇을 묻고 왜 이 도구인가
결합 PDF → 주변 PDF 추출 문제. 한 변수를 적분으로 "소거(marginalize)" 한다 (Lecture06, p.11). 적분 한계는 지지 영역에서 결정: f_X(x) 는 고정 x 에 대해 y 를 그 단면 범위에서 적분, f_Y(y) 는 고정 y 에 대해 x 를 적분. ∫ e^(−xy) dy = (1/x)(1 − e^(−xy 경계)) 같은 지수적분만 쓰면 된다.

## 3. 핵심 통찰
"고정한 변수의 단면(slice) 위에서 나머지를 적분" — x 를 고정하면 y ∈ (0, ∞) 전체, y 를 고정하면 x ∈ [1, e]. e^(−xy) 는 분리 안 되니 독립 아님이 즉시 보이고, 주변 PDF 는 서로 다른 모양이 된다.

## 4. 풀이

(a) 주변 PDF

f_X(x): x ∈ [1, e] 고정, y 를 지지 단면 (0, ∞) 에서 적분 (Lecture06, p.11).
f_X(x) = ∫_0^∞ e^(−xy) dy = [ −(1/x) e^(−xy) ]_(y=0)^(y=∞) = 0 − ( −(1/x)·1 ) = 1/x.
(x ≥ 1 > 0 이므로 e^(−xy) → 0, 적분 수렴.)
정리:
f_X(x) = 1/x,  1 ≤ x ≤ e
f_X(x) = 0,    otherwise
(검산: ∫_1^e (1/x) dx = [ln x]_1^e = 1 − 0 = 1 ✓ — 올바른 주변 PDF.)

f_Y(y): y > 0 고정, x 를 지지 단면 [1, e] 에서 적분.
f_Y(y) = ∫_1^e e^(−xy) dx = [ −(1/y) e^(−xy) ]_(x=1)^(x=e) = −(1/y) e^(−ey) + (1/y) e^(−y)
       = ( e^(−y) − e^(−ey) ) / y.
정리:
f_Y(y) = ( e^(−y) − e^(−ey) ) / y,  y > 0
f_Y(y) = 0,                          otherwise
(y → 0^+ 에서 분자 ≈ (1− y) − (1−ey) = (e−1)y, 따라서 f_Y → e−1 으로 유한 ✓; 0/0 발산 아님.)

독립 여부 점검: f_X(x)·f_Y(y) = (1/x)·(e^(−y)−e^(−ey))/y ≠ e^(−xy) = f_(X,Y). 따라서 X, Y 는 독립이 아니다 (Lecture06, p.18 의 독립 판정 f_(X,Y) = f_X f_Y 불성립).

(b) P(0 ≤ Y ≤ 1, 1 ≤ X ≤ √e) 의 적분식
결합 PDF 를 해당 직사각형 위에서 이중적분 (Lecture06, p.11 의 P((X,Y)∈B) = ∬_B f_(X,Y) dx dy). 요청은 적분을 "쓰는" 것이므로:
P(0 ≤ Y ≤ 1, 1 ≤ X ≤ √e) = ∫_(x=1)^(√e) ∫_(y=0)^(1) e^(−xy) dy dx.
(지지와 교집합: 1 ≤ √e ≤ e 이고 0 ≤ y ≤ 1 ⊂ (0,∞) 이므로 영역 전체가 지지 안. 적분 순서는 바꿔도 동일:
 = ∫_(y=0)^(1) ∫_(x=1)^(√e) e^(−xy) dx dy.)

(참고로 안쪽 적분을 수행하면 ∫_0^1 [ (e^(−y) − e^(−√e·y))/y ] dy 형태가 되어 닫힌형 초등함수로는 안 떨어진다 — 그래서 문제는 "적분을 쓰라"까지만 요구.)

## 5. 검산·직관
- f_X 적분 = 1 ✓, f_Y → e−1 (유한) ✓ — 둘 다 정상적 PDF.
- f_X(x) = 1/x 는 y 의 지수꼬리를 전부 적분한 결과로 x 에만 의존(깔끔), f_Y 는 x 의 유한 구간 적분이라 y 의 복잡한 함수 — 비대칭 지지(x 유한, y 무한)가 그대로 반영.
- e^(−xy) 는 곱으로 안 쪼개지므로 독립 아님 — 결합이 marginal 곱과 다른 전형적 사례.

## 6. 한 줄 요약
> 주변 PDF 는 "고정 변수의 지지 단면 위에서 나머지를 적분"으로 얻고(f_X=1/x, f_Y=(e^(−y)−e^(−ey))/y), 직사각형 확률은 결합 PDF 의 이중적분으로 세운다.
