## 문제
뉴저지의 어느 7월 날 측정한 최고기온 T(화씨, °F)는 Gaussian(85, 10) 확률변수다. P[T > 100], P[T < 60], P[70 ≤ T ≤ 100] 을 구하라.

(표기 주의: Yates의 Gaussian(μ, σ) 은 (평균, 표준편차) 순서다. 즉 μ = 85, σ = 10.)

## 1. 문제 정리
- 주어진 것: T ~ N(평균 μ = 85, 표준편차 σ = 10), 즉 분산 σ² = 100.
- 구할 것: P[T > 100], P[T < 60], P[70 ≤ T ≤ 100].

## 2. 무엇을 묻고 왜 이 도구인가
정규분포 확률 계산. 일반 정규는 표준화(standardization)로 표준정규 Z ~ N(0,1) 의 CDF Φ로 환원한다. 근거: 정규의 선형변환은 정규를 보존하며 Z = (T − μ)/σ ~ N(0,1) (Lecture07, p.9, "X~N(μ,σ²)이면 aX+b~N(aμ+b, a²σ²)"; 여기 a=1/σ, b=−μ/σ). Φ의 수치값은 표준정규표에서 읽는다(표 자체는 강의 밖 일반 지식).

## 3. 핵심 통찰
모든 정규 확률은 Z = (T − μ)/σ 로 옮겨 Φ 하나로 표현된다. "T가 100" → z = (100−85)/10 = 1.5 처럼 "평균에서 몇 σ 떨어졌나"로 바꾸는 것이 핵심.

## 4. 풀이

표준화: Z = (T − 85)/10 ~ N(0,1). Φ(z) = P[Z ≤ z]. 대칭성 Φ(−z) = 1 − Φ(z).

표 값(소수 넷째 자리): Φ(1.5) = 0.9332, Φ(2.5) = 0.9938, Φ(1.0) = 0.8413.

(1) P[T > 100]:
z = (100 − 85)/10 = 1.5.
P[T > 100] = P[Z > 1.5] = 1 − Φ(1.5) = 1 − 0.9332 = 0.0668.

(2) P[T < 60]:
z = (60 − 85)/10 = −2.5.
P[T < 60] = P[Z < −2.5] = Φ(−2.5) = 1 − Φ(2.5) = 1 − 0.9938 = 0.0062.

(3) P[70 ≤ T ≤ 100]:
하한 z₁ = (70 − 85)/10 = −1.5, 상한 z₂ = (100 − 85)/10 = 1.5.
P[70 ≤ T ≤ 100] = Φ(1.5) − Φ(−1.5) = Φ(1.5) − (1 − Φ(1.5)) = 2Φ(1.5) − 1 = 2(0.9332) − 1 = 0.8664.

## 5. 검산·직관
- 세 확률 모두 [0,1] 안 ✓. P[T<60] = 0.0062 가 가장 작음 — 평균에서 2.5σ나 떨어진 쪽이라 직관과 일치.
- 대칭 점검: 70과 100은 평균 85에서 각각 ±15(=±1.5σ) — 대칭 구간이라 답이 2Φ(1.5)−1 깔끔한 꼴 ✓. 이는 "평균 중심 ±1.5σ 안에 들 확률 ≈ 86.6%"로 68–95–99.7 경험법칙(±1σ 68%, ±2σ 95%)과 부합.
- P[T>100] = 0.0668 ≈ 6.7% — 100°F 넘는 폭염일이 드물게 일어남, 모델로 그럴듯.

## 6. 한 줄 요약
> 일반 정규 확률은 z = (값 − μ)/σ 로 표준화해 Φ로 환산하며, 평균 중심 대칭구간은 2Φ(z) − 1 로 떨어진다.
