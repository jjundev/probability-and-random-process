## 문제
표준정규 확률변수 Z의 PDF는 무엇인가?

## 1. 문제 정리
- 구할 것: Z ~ N(0, 1) (평균 0, 분산 1)의 확률밀도함수 f_Z(z).

## 2. 무엇을 묻고 왜 이 도구인가
정의 확인 문제. 일반 정규 N(μ, σ²)의 PDF에서 μ = 0, σ = 1 을 대입하면 표준정규 PDF가 나온다. 정규 PDF의 일반형은 강의의 선형보존 논의의 출발점이다 (Lecture07, p.9). 표준정규는 모든 정규를 표준화 Z = (X−μ)/σ 했을 때 도달하는 기준 분포.

## 3. 핵심 통찰
"표준"이란 μ = 0, σ = 1 을 뜻한다. 일반형 (1/√(2πσ²)) e^(−(x−μ)²/(2σ²)) 에 그 둘을 넣으면 지수가 −z²/2 로, 정규화 상수가 1/√(2π) 로 단순화된다.

## 4. 풀이

일반 정규 N(μ, σ²)의 PDF:

f_X(x) = (1/√(2πσ²)) · e^(−(x−μ)²/(2σ²)),   모든 실수 x

여기에 μ = 0, σ² = 1 대입:

f_Z(z) = (1/√(2π)) · e^(−z²/2),   모든 실수 z (otherwise = 0 해당 없음 — 전 실수에서 양수)

## 5. 검산·직관
- 정규화: ∫_(−∞)^(∞) (1/√(2π)) e^(−z²/2) dz = 1 ✓ (가우시안 적분 ∫ e^(−z²/2) dz = √(2π)).
- 대칭: f_Z(−z) = f_Z(z) — 0을 중심으로 좌우 대칭(짝함수). 따라서 E[Z] = 0.
- 최댓값: z = 0 에서 1/√(2π) ≈ 0.3989.

## 6. 한 줄 요약
> 표준정규 PDF는 f_Z(z) = (1/√(2π)) e^(−z²/2) (전 실수, otherwise 없음) — 일반 정규에 μ=0, σ=1 을 넣은 0 중심 대칭종형이다.
