## 문제
X와 Y는 결합 정규(jointly normal) 확률변수이고 파라미터는 μ_X = 2, σ_X^2 = 4, μ_Y = 1, σ_Y^2 = 9, ρ = −1/2이다.

(a) E[Y | X = 3]을 구하라.
(b) Var(Y | X = 2)를 구하라.
(c) P(X + 2Y ≤ 5 | X + Y = 3)을 구하라.

## 1. 문제 정리
주어진 것: (X,Y) 결합 정규. μ_X=2, σ_X^2=4 (σ_X=2), μ_Y=1, σ_Y^2=9 (σ_Y=3), ρ=−1/2.
구할 것: (a) 조건부 평균, (b) 조건부 분산, (c) 한 선형결합을 다른 선형결합에 조건화한 확률.

⚠ 강의 밖 보강: 정리노트 Lecture06은 단변량 Gaussian·조건부 PDF 정의를, Lecture08은 조건부 기댓값/분산을 RV로 다루지만, **결합정규의 명시적 조건부 공식**(아래 (★)(★★))은 정리노트에 없어 일반 교과서 지식으로 보강한다. 결합정규의 선형결합도 정규라는 사실(Lecture07, p.18의 정규합 보존을 다변량으로 일반화)을 사용한다.

## 2. 무엇을 묻고 왜 이 도구인가
결합정규에는 두 가지 마법 같은 성질이 있다.
- 조건부 분포가 다시 정규이고, 조건부 평균은 조건 변수의 **선형함수**, 조건부 분산은 **조건 값과 무관한 상수**.
- 임의의 선형결합 (U,V)도 결합정규 → V|U도 정규. (c)는 이 둘을 합쳐 푼다.

이 문제의 도구는 결합정규 조건부 공식 두 개:
(★)  E[Y | X=x] = μ_Y + ρ·(σ_Y/σ_X)·(x − μ_X)
(★★) Var(Y | X=x) = σ_Y^2·(1 − ρ^2)   (x에 무관)

## 3. 핵심 통찰
(c)에서 조건 X+Y=3은 정확히 E[X+Y]=μ_X+μ_Y=3과 같다 — 평균값에 조건화하므로 조건부 평균의 보정항이 0이 되어 E[X+2Y | X+Y=3] = E[X+2Y]로 떨어진다. 계산이 크게 단순해진다.

## 4. 풀이

먼저 공분산: Cov(X,Y) = ρ·σ_X·σ_Y = (−1/2)·2·3 = −3.

**(a) E[Y | X=3]** — (★) 적용, σ_Y/σ_X = 3/2:

E[Y | X=3] = μ_Y + ρ·(σ_Y/σ_X)·(3 − μ_X) = 1 + (−1/2)·(3/2)·(3 − 2) = 1 − 3/4 = 1/4

**(b) Var(Y | X=2)** — (★★) 적용 (조건 값 2는 결과에 영향 없음):

Var(Y | X=2) = σ_Y^2·(1 − ρ^2) = 9·(1 − (−1/2)^2) = 9·(1 − 1/4) = 9·(3/4) = 27/4 = 6.75

**(c) P(X + 2Y ≤ 5 | X + Y = 3)**

U = X+Y, V = X+2Y로 두면 (U,V)는 결합정규, 따라서 V|U도 정규. 필요한 모멘트:

E[U] = μ_X+μ_Y = 3,   E[V] = μ_X+2μ_Y = 2+2 = 4

Var(U) = Var(X)+Var(Y)+2Cov(X,Y) = 4+9+2(−3) = 7
Var(V) = Var(X)+4Var(Y)+4Cov(X,Y) = 4+36+4(−3) = 28
Cov(U,V) = Cov(X+Y, X+2Y) = Var(X) + 3Cov(X,Y) + 2Var(Y) = 4 + 3(−3) + 18 = 13

조건부 평균(선형회귀식):
E[V | U=u] = E[V] + (Cov(U,V)/Var(U))·(u − E[U]) = 4 + (13/7)(3 − 3) = 4
→ u=3=E[U]이라 보정항 0.

조건부 분산:
Var(V | U) = Var(V) − Cov(U,V)^2/Var(U) = 28 − 13^2/7 = 28 − 169/7 = (196 − 169)/7 = 27/7

따라서 V | (U=3) ~ Normal(평균 4, 분산 27/7), 표준편차 √(27/7) ≈ 1.964.

P(V ≤ 5 | U=3) = Φ( (5 − 4)/√(27/7) ) = Φ( √(7/27) ) = Φ(0.5092) ≈ 0.695

## 5. 검산·직관
- (a) ρ<0이라 X가 평균 2보다 큰 3으로 관측되면 Y의 조건부 평균은 무조건부 평균 1보다 아래(1/4)로 끌려감 — 음의 상관과 일치 ✓.
- (b) 조건부 분산 27/4 < 무조건부 분산 9: X를 알면 Y의 불확실성이 줄어듦. 인자 (1−ρ^2)=3/4 ✓.
- (c) Var(V|U)=27/7 < Var(V)=28: 조건화로 분산 감소 ✓. 조건 값이 평균과 같아 중심이 그대로 4, 5는 평균보다 약간 위 → 확률이 1/2를 살짝 넘는 0.695 ✓.
- 공분산 행렬 양정치성 점검: Var(U)Var(V) − Cov(U,V)^2 = 7·28 − 169 = 196 − 169 = 27 > 0 ✓.

## 6. 한 줄 요약
> 결합정규는 조건부도 정규라 평균은 선형회귀식 μ_Y+ρ(σ_Y/σ_X)(x−μ_X), 분산은 σ_Y^2(1−ρ^2)이며, 선형결합끼리의 조건화 (c)도 (U,V)를 결합정규로 보고 같은 공식을 쓰면 된다.
