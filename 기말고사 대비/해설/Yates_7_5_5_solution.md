## 문제
확률변수 N과 K가 결합 PMF
P_N,K(n,k) = 100^n · e^(−100) / (n+1)!,   n = 0,1,2,... ; k = 0,1,...,n
P_N,K(n,k) = 0,   otherwise
를 가진다.

(a) 주변 PMF P_N(n) 과 조건부 PMF P_K|N(k|n) 을 구하라.
(b) 조건부 기댓값 E[K|N=n] 을 구하라.
(c) 확률변수 E[K|N] 을 N의 함수로 표현하고, 반복기댓값(iterated expectation)을 이용해 E[K] 를 구하라.

## 1. 문제 정리
주어진 것:
- P_N,K(n,k) = 100^n e^(−100) / (n+1)!, 정의역 n≥0, 0≤k≤n (정수), 그 외 0.
- 주목: 이 결합 PMF는 k에 의존하지 않는다 (같은 n이면 모든 k=0,...,n에 동일값).

구할 것: (a) P_N(n), P_K|N(k|n)  (b) E[K|N=n]  (c) E[K|N]을 N의 함수로, 그리고 E[K].

## 2. 무엇을 묻고 왜 이 도구인가
주변은 다른 변수를 합쳐서(marginalize) 얻는다: P_N(n) = Σ_k P_N,K(n,k) (Lecture03, p.20). 조건부는 P_K|N = P_N,K/P_N (Lecture04, slide 8). 결합이 k에 무관하다는 점이 결정타 — k에 대한 합이 단순히 "항의 개수 (n+1)"를 곱하는 것이 된다. (c)는 E[K|N]을 N의 함수 RV로 보고 (Lecture08, p.16) 반복기댓값 E[E[K|N]] = E[K] (Lecture08, p.17)로 마무리.

## 3. 핵심 통찰
결합이 k에 무관 ⟹ K|N=n은 {0,1,...,n} 위 균등분포. 그러면 주변 P_N은 Poisson(100)으로 깔끔히 떨어지고, E[K|N=n]은 균등분포 평균 n/2, 따라서 E[K] = E[N]/2 = 50.

## 4. 풀이

### (a) 주변 PMF P_N(n)
n을 고정하면 k는 0,1,...,n 의 (n+1)개 값을 갖고 각 항은 동일(100^n e^(−100)/(n+1)!):
P_N(n) = Σ_{k=0}^{n} 100^n e^(−100)/(n+1)! = (n+1) · 100^n e^(−100)/(n+1)!
       = 100^n e^(−100)/n!,   n = 0,1,2,...
P_N(n) = 0,   otherwise.

이는 평균 100의 Poisson(100) PMF다.

### (a) 조건부 PMF P_K|N(k|n)
n ≥ 0 (P_N(n) > 0) 에 대해:
P_K|N(k|n) = P_N,K(n,k)/P_N(n) = [100^n e^(−100)/(n+1)!] / [100^n e^(−100)/n!]
          = n!/(n+1)! = 1/(n+1),   k = 0,1,...,n
P_K|N(k|n) = 0,   otherwise.

즉 N=n이 주어지면 K는 {0,1,...,n} 위 이산 균등분포 Uniform{0,...,n}.

### (b) 조건부 기댓값 E[K|N=n]
{0,1,...,n} 균등분포의 평균은 양 끝의 중점:
E[K|N=n] = Σ_{k=0}^{n} k · 1/(n+1) = 1/(n+1) · [n(n+1)/2] = n/2.

(여기서 Σ_{k=0}^{n} k = n(n+1)/2 사용.)

### (c) E[K|N]을 N의 함수로 + E[K]
E[K|N] 은 N=n일 때 값 n/2 를 갖는 RV이므로:
E[K|N] = N/2.

반복기댓값 법칙 (Lecture08, p.17: E[E[K|N]] = E[K]):
E[K] = E[E[K|N]] = E[N/2] = (1/2)·E[N].
N ~ Poisson(100) 이므로 E[N] = 100.
따라서 E[K] = (1/2)·100 = 50.

## 5. 검산·직관
- P_N(n)이 Poisson(100)으로 정상화되는지: Σ_{n≥0} 100^n e^(−100)/n! = e^(−100)·e^(100) = 1 ✓.
- 조건부 정상성: Σ_{k=0}^{n} 1/(n+1) = (n+1)/(n+1) = 1 ✓.
- (b) K∈{0,...,n}이라 E[K|N=n]=n/2 ∈ [0,n] ✓ (균등분포 중점).
- (c) E[K]=50 은 E[N]=100의 절반 — K가 0~N 사이를 균등하게 도므로 "평균적으로 N의 절반"이 직관과 일치 ✓.

## 6. 한 줄 요약
> 결합 PMF가 k에 무관하면 K|N은 {0,...,N} 균등분포가 되고, 주변은 Poisson(100)으로 떨어져 E[K]=E[N]/2=50을 반복기댓값으로 얻는다.
