## 문제
X가 가우시안(정규)일 때, Equation (10.24)을 검증하라. 이 식은 "표본평균(sample mean)이 기댓값의 1 표준오차(standard error) 이내에 있을 확률이 0.68"임을 말한다.

(Equation 10.24의 내용: P[ |M_n(X) − μ| ≤ σ_{M_n} ] = 0.6827, 단 σ_{M_n} = σ/√n = 표준오차.)

## 1. 문제 정리
- 주어진 것: X ~ Gaussian(μ, σ²), n개 표본 X₁,…,X_n i.i.d., 표본평균 M_n = (X₁+⋯+X_n)/n.
- 표준오차(standard error): σ_{M_n} = √(Var[M_n]) = σ/√n.
- 보일 것: P( |M_n − μ| ≤ σ/√n ) = 0.6827 ≈ 0.68.

## 2. 무엇을 묻고 왜 이 도구인가
표본평균 M_n도 확률변수이며, E[M_n] = μ, Var[M_n] = σ²/n (Lecture09, slide 6). 일반적으로 M_n의 정확한 분포는 모르지만(그래서 보통 CLT 근사), **X가 가우시안이면 가우시안의 선형결합도 가우시안**이라 M_n이 정확히 정규분포다 (Lecture07, p.18: 독립 정규합 = 정규). 그러면 표준화 한 번으로 확률을 표준정규표에서 정확히 읽을 수 있다 — 근사가 아니라 등식.

## 3. 핵심 통찰
"1 표준오차 이내"는 곧 "표준화하면 |Z| ≤ 1"이다. M_n을 자기 평균 μ와 표준오차 σ/√n으로 표준화하면 Z = (M_n − μ)/(σ/√n) ~ N(0,1)이 되고, 묻는 확률은 P(|Z| ≤ 1) — 정규분포의 "68% 규칙"의 정의 그 자체다.

## 4. 풀이

(1) M_n의 분포. X_i ~ N(μ, σ²) 독립이므로 합 ΣX_i ~ N(nμ, nσ²) (독립 정규합 = 정규, Lecture07, p.18). M_n = (1/n)ΣX_i 는 정규의 선형변환이라 역시 정규 (Lecture07, p.9: Y = aX+b, X 정규 ⇒ Y 정규):
E[M_n] = μ,   Var[M_n] = σ²/n   (Lecture09, slide 6)
즉 M_n ~ N(μ, σ²/n),   표준오차 σ_{M_n} = σ/√n.

(2) 표준화 (Lecture09, slide 23: Y = (X−μ)/σ ⇒ 평균 0, 분산 1):
Z = (M_n − μ) / (σ/√n) ~ N(0,1)

(3) 확률 계산:
P( |M_n − μ| ≤ σ/√n ) = P( |Z| ≤ 1 )
 = Φ(1) − Φ(−1)
 = 2Φ(1) − 1
 = 2(0.8413) − 1
 = 0.6827 ≈ 0.68   ∎

검증 완료: X(따라서 M_n)가 가우시안일 때 표본평균이 기댓값의 1 표준오차 이내일 확률은 정확히 0.6827이다.

## 5. 검산·직관
- 표준정규의 68–95–99.7 규칙: |Z| ≤ 1 → 68.27%, |Z| ≤ 2 → 95.45%, |Z| ≤ 3 → 99.73%. 첫 값과 일치 ✓.
- n에 무관: 표준오차 자체가 σ/√n로 스케일되므로, "몇 표준오차 이내"로 물으면 n이 약분되어 확률은 n과 무관 — 가우시안의 자기닮음 ✓.
- 비가우시안이면? CLT에 의해 n이 클 때 근사적으로 0.68 (Lecture09, slide 9). 가우시안일 때만 모든 n에서 **정확히** 0.6827.
- 표준오차 = σ/√n: WLLN처럼 n↑이면 √(Var[M_n]) = σ/√n → 0, M_n이 μ로 집중 (Lecture09, slide 7)과 정합.

## 6. 한 줄 요약
> X가 정규면 M_n ~ N(μ, σ²/n)도 정규 → "1 표준오차 이내" = P(|Z| ≤ 1) = 2Φ(1) − 1 = 0.6827, n에 무관하게 정확.
