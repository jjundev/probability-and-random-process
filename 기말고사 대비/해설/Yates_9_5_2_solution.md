## 문제
인터넷 패킷은 비디오(V) 또는 일반 데이터(D)로 분류된다. 인터넷 서비스 제공자의 많은 관측에 근거해 다음 확률모델을 얻었다: P[V] = 3/4, P[D] = 1/4. 데이터 패킷과 비디오 패킷은 서로 독립적으로 발생한다. 랜덤변수 K_n 은 n개 패킷 모음에서 비디오 패킷의 개수다.
(a) E[K_100], 즉 100개 패킷 집합에서 비디오 패킷의 기대 개수는?
(b) σ_(K_100) 은?
(c) CLT를 사용해 P[K_100 ≥ 18] 을 추정하라.
(d) CLT를 사용해 P[16 ≤ K_100 ≤ 24] 를 추정하라.

## 1. 문제 정리
| 항목 | 내용 |
|---|---|
| 각 패킷 | 비디오일 확률 p = 3/4, 독립 |
| K_100 | 100개 중 비디오 수 ~ Binomial(100, 3/4) |
| (a) | E[K_100] |
| (b) | σ_(K_100) |
| (c) | P[K_100 ≥ 18] (CLT) |
| (d) | P[16 ≤ K_100 ≤ 24] (CLT) |

K_n 의 PMF (이항):

  P(K_n = k) = C(n,k) (3/4)^k (1/4)^(n−k),   k = 0,1,…,n
  P(K_n = k) = 0,   otherwise

## 2. 무엇을 묻고 왜 이 도구인가
K_100 은 i.i.d. 베르누이(비디오=1) 100개의 합 = 이항. (a)(b)는 이항의 평균 np·표준편차 √(np(1−p)), (c)(d)는 그 꼬리·구간 확률을 CLT로 근사. 이항 100개의 정확한 합/구간을 손계산하긴 번거롭고, 강의노트가 권하는 표준화-가우시안 근사가 자연스럽다. (Lecture09, slide 9; 이항 PMF는 slide 4의 출석학생 예시와 동형)

## 3. 핵심 통찰
K_100 = ΣX_i (X_i ~ Bernoulli(3/4))는 평균 np·분산 np(1−p)을 가지며 (K − np)/√(np(1−p)) 이 N(0,1)로 수렴. 구간 확률은 양 끝을 각각 표준화해 Φ의 차로 구한다. (Lecture09, slide 9)

## 4. 풀이

(a) E[K_100]. 이항 평균 np — (Lecture09, slide 4의 이항 예시):

  E[K_100] = np = 100 × 3/4 = 75

(b) σ_(K_100). 이항 분산 np(1−p):

  Var(K_100) = np(1−p) = 100 × (3/4) × (1/4) = 18.75
  σ_(K_100) = √18.75 ≈ 4.3301

(c) CLT로 P[K_100 ≥ 18] (Lecture09, slide 9):

  Z = (K_100 − 75)/4.3301  ≈  N(0,1)

  P[K_100 ≥ 18] = P( Z ≥ (18 − 75)/4.3301 )
                = P( Z ≥ −57/4.3301 )
                = P( Z ≥ −13.16 )
                ≈ 1.0000

(−13.16은 평균에서 13σ 아래라 사실상 확실. 즉 100개 중 비디오가 평균 75개인데 18개 이상일 확률은 ≈ 1.)

(d) CLT로 P[16 ≤ K_100 ≤ 24]. 두 끝을 표준화:

  하단: (16 − 75)/4.3301 = −59/4.3301 ≈ −13.63
  상단: (24 − 75)/4.3301 = −51/4.3301 ≈ −11.78

  P[16 ≤ K_100 ≤ 24] = Φ(−11.78) − Φ(−13.63) ≈ 0 − 0 = 0 (≈ 0)

(16~24 구간은 평균 75에서 약 12~14σ 아래 — 정규근사로 사실상 0.)

## 5. 검산·직관
- (c) vs (d) 합리성: 평균이 75인데 (c)는 "18 이상"이라 거의 전부 포함(≈1), (d)는 "16~24"라는 극히 낮은 구간이라 거의 0 — 두 결과가 평균 75 위치와 정합.
- 주의: 이 문제의 임계값(18, 16~24)들은 평균 75에서 10σ 이상 떨어져 있어 정규근사 답이 극단(0 또는 1)으로 나온다. (문헌·해답지에 따라 숫자가 75 부근(예: 78, 72~78)으로 의도됐을 가능성도 있으나, PNG에 적힌 18·16·24를 ground truth로 그대로 사용했다.)
- (b) 단위: 패킷 수(정수)의 표준편차 → 무차원 개수. ✓

## 6. 한 줄 요약
> 이항(100, 3/4) 비디오 수는 평균 np=75·표준편차 √(np(1−p))≈4.33이며, 18·16·24는 모두 평균에서 10σ 이상 떨어져 CLT 근사로 P[K≥18]≈1, P[16≤K≤24]≈0.
