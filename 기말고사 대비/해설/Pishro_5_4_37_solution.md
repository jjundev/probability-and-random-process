## 문제
X와 Y는 결합정규(jointly normal) 확률변수로 파라미터가 μ_X = 1, σ_X^2 = 4, μ_Y = 1, σ_Y^2 = 1, ρ = 0 이다.

a. P(X + 2Y > 4) 를 구하라.
b. E[X^2 Y^2] 을 구하라.

## 1. 문제 정리
주어진 것:
- μ_X = 1, σ_X^2 = 4 (σ_X = 2)
- μ_Y = 1, σ_Y^2 = 1 (σ_Y = 1)
- ρ = 0 → Cov(X,Y) = ρσ_Xσ_Y = 0

구할 것:
- (a) P(X + 2Y > 4)
- (b) E[X^2 Y^2]

## 2. 무엇을 묻고 왜 이 도구인가
(a) 5.4.36과 같은 도구: 결합정규의 선형결합 W = X+2Y 는 정규분포이므로 평균·분산 → 표준화 Φ.
(b) 여기서 결합정규의 결정적 성질이 등장한다 — "ρ = 0 인 결합정규는 곧 독립(independent)"이다. 일반 RV는 uncorrelated여도 독립이 아니지만(Lecture08, p.6 "uncorrelated ≠ 독립"), 결합정규에서는 ρ=0 ⟺ 독립이다. 독립이면 E[X^2 Y^2] = E[X^2]·E[Y^2] 로 분리된다 (Lecture04, slide 22: X⊥Y일 때 E[g(X)h(Y)] = E[g(X)]E[h(Y)]).

상관계수↔공분산·곱의 기댓값 분리는 Lecture08/Lecture04 범위. "ρ=0 결합정규 ⟹ 독립"은 결합정규 고유 성질 → ⚠ 강의 밖, 일반 교과서 지식.

## 3. 핵심 통찰
ρ=0 결합정규 ⟹ X와 Y는 독립. 따라서 (a) 분산의 교차항이 사라지고, (b) E[X^2Y^2]가 E[X^2]·E[Y^2] 로 깔끔히 분리된다. 각 2차 모멘트는 E[Z^2] = var[Z] + (E[Z])^2 로 구한다.

## 4. 풀이

### (a) P(X + 2Y > 4)
W = X + 2Y ~ Normal(μ_W, σ_W^2) (결합정규의 선형결합, ⚠ 강의 밖).

평균:
μ_W = μ_X + 2μ_Y = 1 + 2·1 = 3.

분산 (Cov(X,Y)=0 이므로 교차항 소멸):
σ_W^2 = 1^2·σ_X^2 + 2^2·σ_Y^2 + 2·1·2·Cov(X,Y)
      = 4 + 4·1 + 0 = 8.
σ_W = √8 = 2√2 ≈ 2.8284.

표준화 (우측 꼬리):
P(W > 4) = 1 − Φ((4 − μ_W)/σ_W) = 1 − Φ((4 − 3)/(2√2)) = 1 − Φ(1/(2√2)).

1/(2√2) = 1/2.8284 ≈ 0.3536.
Φ(0.3536) ≈ 0.6382.

따라서 P(X + 2Y > 4) = 1 − Φ(0.3536) ≈ 1 − 0.6382 = 0.3618.

### (b) E[X^2 Y^2]
ρ = 0 인 결합정규이므로 X ⊥ Y (⚠ 강의 밖: 결합정규에서 uncorrelated ⟺ independent).
독립이면 (Lecture04, slide 22):
E[X^2 Y^2] = E[X^2]·E[Y^2].

각 2차 모멘트는 분산 계산공식 var[Z] = E[Z^2] − (E[Z])^2 ⟹ E[Z^2] = var[Z] + (E[Z])^2 (Lecture03, p.18):
E[X^2] = var[X] + (μ_X)^2 = 4 + 1^2 = 5.
E[Y^2] = var[Y] + (μ_Y)^2 = 1 + 1^2 = 2.

따라서:
E[X^2 Y^2] = 5 · 2 = 10.

(새로 정의되는 PMF/PDF 없음. W ~ Normal(3, 8)은 전 실수에서 양의 밀도를 갖는 정규 PDF라 "otherwise = 0" 구간이 없다.)

## 5. 검산·직관
- (a) σ_W^2 = 8 > 0 ✓. ρ=0 이라 교차항 0 → 5.4.36보다 분산이 큼(4 vs 8)인데, 거기선 음의 상관이 분산을 깎았던 것과 대비됨 — 일관적.
- (a) 임계값 4가 평균 3보다 약간 위(+0.35σ)이므로 P(W>4)는 0.5보다 약간 작아야 한다. 0.3618 < 0.5 ✓.
- (b) 독립 가정 없이 무턱대고 E[X^2Y^2]=E[X^2]E[Y^2]를 쓰면 안 되지만, 여기선 결합정규+ρ=0이 독립을 보장하므로 정당 ✓. 만약 ρ≠0이었다면 이 분리는 틀린다.

## 6. 한 줄 요약
> 결합정규에서 ρ=0은 곧 독립이므로 합의 분산 교차항이 사라지고 E[X^2Y^2]가 E[X^2]·E[Y^2]로 분리된다 — 일반 RV에서는 성립하지 않는 결합정규 특권.
