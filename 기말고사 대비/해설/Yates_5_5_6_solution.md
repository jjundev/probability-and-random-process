## 문제
원판 X² + Y² ≤ r² 위에서 확률변수 X, Y 가 다음 PDF 를 갖는다.

f_{X,Y}(x, y) = 2|xy| / r⁴,  x² + y² ≤ r²
f_{X,Y}(x, y) = 0,           otherwise

(a) 주변 PDF f_X(x) 는?
(b) 주변 PDF f_Y(y) 는?

## 1. 문제 정리
- 주어진 것: 원판 정의역 위 결합 PDF f_{X,Y}(x,y) = 2|xy|/r⁴.
- 구할 것: 주변 PDF f_X(x), f_Y(y).
- 핵심 함정: 정의역이 사각형이 아니라 **원판**이라, y 적분 한계가 x 에 의존한다.

## 2. 무엇을 묻고 왜 이 도구인가
주변 PDF 는 결합 PDF 를 나머지 변수로 적분: f_X(x) = ∫ f_{X,Y}(x,y) dy (Lecture06, p.11). 도구 자체는 단순하나, 적분 한계가 원판 제약 x² + y² ≤ r² 에서 나온다 — 고정된 x 에 대해 y ∈ [−√(r²−x²), √(r²−x²)]. 정의역을 사각형으로 착각해 ±r 로 적분하면 틀린다.

## 3. 핵심 통찰
고정된 x (|x| ≤ r) 에서 y 의 허용범위는 길이 a = √(r²−x²) 의 대칭구간. 피적분함수 |y| 는 우함수라 ∫_{−a}^a |y| dy = a² = r² − x² 로 깔끔히 떨어진다. PDF 가 x↔y 대칭이므로 f_Y 는 f_X 에서 x→y 치환만 하면 끝.

## 4. 풀이

### (a) f_X(x)
|x| ≤ r 일 때, y 범위는 −√(r²−x²) ≤ y ≤ √(r²−x²). a = √(r²−x²) 로 두면

f_X(x) = ∫_{−a}^{a} (2|x||y| / r⁴) dy = (2|x| / r⁴) ∫_{−a}^{a} |y| dy.

|y| 는 우함수: ∫_{−a}^{a} |y| dy = 2∫_0^a y dy = 2·(a²/2) = a² = r² − x².

따라서

f_X(x) = 2|x|(r² − x²) / r⁴,  −r ≤ x ≤ r
f_X(x) = 0,                    otherwise

### (b) f_Y(y)
결합 PDF 가 x 와 y 에 대해 완전 대칭(2|xy|/r⁴, 정의역도 대칭)이므로 같은 계산을 y 에 대해 반복하면 x→y 치환형:

f_Y(y) = 2|y|(r² − y²) / r⁴,  −r ≤ y ≤ r
f_Y(y) = 0,                    otherwise

## 5. 검산·직관
- 정규화: ∫_{−r}^r 2|x|(r²−x²)/r⁴ dx = (4/r⁴)∫_0^r x(r²−x²) dx = (4/r⁴)[r²x²/2 − x⁴/4]_0^r = (4/r⁴)(r⁴/4) = 1 ✓.
- 비음성: −r ≤ x ≤ r 에서 |x| ≥ 0, (r²−x²) ≥ 0 → f_X ≥ 0 ✓.
- 경계값: x = 0 에서 f_X = 0 (|x| 인자), x = ±r 에서도 f_X = 0 (r²−x² 인자). 중간 어딘가(미분하면 x = r/√3)에서 최대 — |xy| 가중이라 축 근처와 경계에서 밀도가 죽는 것이 직관적.
- ⚠ 주의: f_X, f_Y 가 각각 잘 정의돼도 X, Y 는 **독립이 아니다**. f_X(x)f_Y(y) 의 정의역은 사각형인데 실제 정의역은 원판이라 f_{X,Y} ≠ f_X f_Y (Lecture06, p.18).

## 6. 한 줄 요약
> 원판 정의역의 주변 PDF 는 적분 한계 자체가 ±√(r²−x²) 로 x 에 의존하며, 우함수 |y| 적분이 r²−x² 로 떨어져 f_X(x) = 2|x|(r²−x²)/r⁴ 를 준다.
