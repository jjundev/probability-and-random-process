## 문제
확률변수 X 의 CDF 가 다음과 같다.

F_X(x) = 0,        x < 0
F_X(x) = x,        0 ≤ x < 1/4
F_X(x) = x + 1/2,  1/4 ≤ x < 1/2
F_X(x) = 1,        x ≥ 1/2

(a) F_X(x) 를 그리고, X 가 혼합(mixed) 확률변수인 이유를 설명하라.
(b) P(X ≤ 1/3) 을 구하라.
(c) P(X ≥ 1/4) 을 구하라.
(d) X 의 CDF 를 F_X(x) = C(x) + D(x) 형태로 써라. 여기서 C(x) 는 연속함수, D(x) 는 계단함수
    D(x) = Σ_k a_k · u(x − x_k) 형태(u 는 단위계단함수).
(e) c(x) = (d/dx) C(x) 를 구하라.
(f) EX = ∫_(−∞)^∞ x·c(x) dx + Σ_k x_k·a_k 로 EX 를 구하라.

## 1. 문제 정리
주어진 것: 구간별로 정의된 CDF F_X. 0 ≤ x < 1/4 와 1/4 ≤ x < 1/2 에서 기울기 1 의 연속 증가 구간을 가지되, x = 1/4 에서 값이 1/4 → 1/4 + 1/2 = 3/4 로 1/2 만큼 **점프**한다.
구할 것: (a) 그래프·혼합 판정, (b)(c) 두 확률, (d) 연속부 C + 점프부 D 분해, (e) 연속부의 밀도 c, (f) 기댓값.

검증(필수): CDF 는 우연속·비감소·F(−∞)=0·F(∞)=1 이어야 한다 (Lecture05, p.10).
- x = 1/4 좌극한 = 1/4, 우값 = 1/4 + 1/2 = 3/4. 비감소 ✓ (점프 +1/2).
- x = 1/2 좌극한 = 1/2 + 1/2 = 1, 우값 = 1. 연속 ✓.
→ 총 "확률 질량"은 연속부 적분 + 점프 합 = 1 이 되어야 한다(아래 검산).

## 2. 무엇을 묻고 왜 이 도구인가
혼합 RV 문제. CDF 가 (i) 연속적으로 증가하는 부분과 (ii) 불연속 점프를 동시에 가지면, 그 RV 는 연속도 이산도 아닌 **혼합형**이다. 연속부는 PDF(밀도 c(x))로, 점프부는 PMF(점프 크기 a_k)로 따로 기술한다.

CDF 의 점프 크기 = 그 점의 점질량(point mass): P(X = x_0) = F_X(x_0) − F_X(x_0^−) (Lecture05, p.10 의 "이산: p_X(k) = F_X(k) − F_X(k−1)" 을 일반점으로 확장). 연속 구간에서는 P(X = a) = 0 (Lecture05, p.5).

## 3. 핵심 통찰
"CDF 의 기울기는 연속 밀도 c(x), CDF 의 수직 점프는 이산 질량 a_k" — 둘을 분리하면 혼합 RV 가 곧바로 풀린다. 점프 위치 x = 1/4 에 질량 1/2 가 박혀 있다.

## 4. 풀이

(a) 그래프·혼합 판정
그래프 개형:
- x < 0: 0 (수평).
- 0 ≤ x < 1/4: 원점에서 시작해 기울기 1 직선, x=1/4^− 에서 1/4 도달.
- x = 1/4: 1/4 에서 3/4 로 수직 점프(크기 1/2).
- 1/4 ≤ x < 1/2: 3/4 에서 시작해 기울기 1 직선, x=1/2^− 에서 1 도달.
- x ≥ 1/2: 1 (수평).

혼합인 이유: F_X 는 (0,1/4) 와 (1/4,1/2) 에서 **연속적으로 증가**(연속 성분 존재) 하면서, x = 1/4 에서 **점프 불연속**(P(X=1/4) = 1/2 > 0, 이산 성분 존재)을 가진다. 순수 연속이면 점프가 없고, 순수 이산이면 증가 구간 없이 계단만 있어야 하는데 둘이 공존하므로 혼합형이다 (Lecture05, p.10).

(b) P(X ≤ 1/3)
정의상 P(X ≤ x) = F_X(x) (Lecture05, p.9). 1/3 은 1/4 ≤ 1/3 < 1/2 구간이므로
P(X ≤ 1/3) = F_X(1/3) = 1/3 + 1/2 = 5/6.

(c) P(X ≥ 1/4)
여확률을 쓰되, x = 1/4 에 점질량이 있으므로 경계 포함에 주의한다.
P(X ≥ 1/4) = 1 − P(X < 1/4) = 1 − F_X(1/4^−).
F_X(1/4^−) = 1/4 (좌극한, 점프 전 값). 따라서
P(X ≥ 1/4) = 1 − 1/4 = 3/4.
(점검: P(X ≥ 1/4) = P(X = 1/4) + P(X > 1/4) = 1/2 + [1 − F_X(1/4)] = 1/2 + (1 − 3/4) = 1/2 + 1/4 = 3/4 ✓.)

(d) C(x) + D(x) 분해
점프는 x_1 = 1/4 한 곳, 크기 a_1 = F_X(1/4) − F_X(1/4^−) = 3/4 − 1/4 = 1/2.
이산(계단) 성분:
D(x) = (1/2)·u(x − 1/4).
연속 성분 C(x) = F_X(x) − D(x) 는 점프를 제거한, 기울기 1 로 매끄럽게 이어지는 함수:
C(x) = 0,    x < 0
C(x) = x,    0 ≤ x < 1/2
C(x) = 1/2,  x ≥ 1/2
(확인: 1/4 ≤ x < 1/2 에서 F_X = x + 1/2 = C(x) + D(x) = x + 1/2 ✓; x ≥ 1/2 에서 1 = 1/2 + 1/2 ✓.)

(e) c(x) = (d/dx) C(x)
C(x) 는 (0, 1/2) 에서 기울기 1, 그 밖은 수평이므로
c(x) = 1,  0 ≤ x ≤ 1/2
c(x) = 0,  otherwise
(c 는 "연속부의 밀도"로 적분하면 1/2 — 나머지 질량 1/2 는 점질량이 가져간다.)

(f) EX
공식 EX = ∫_(−∞)^∞ x·c(x) dx + Σ_k x_k·a_k 에 대입.
연속부: ∫_0^(1/2) x·1 dx = [x^2/2]_0^(1/2) = (1/4)/2 = 1/8.
이산부: x_1·a_1 = (1/4)·(1/2) = 1/8.
EX = 1/8 + 1/8 = 1/4.

## 5. 검산·직관
- 전체 질량 = ∫_0^(1/2) c dx + Σ a_k = 1/2 + 1/2 = 1 ✓ (CDF 가 1 에 도달).
- c(x) 는 [0,1/2] 균등 밀도(높이 1) + 1/4 에 질량 1/2. 직관적 평균은 균등부 평균 1/4 과 점질량 위치 1/4 의 가중평균인데, 둘 다 1/4 근처라 EX = 1/4 가 자연스럽다.
- (c) 에서 ≥ 와 > 를 혼동하면 1/2 만큼 틀린다 — 점질량 있는 경계는 항상 좌극한으로 분리.

## 6. 한 줄 요약
> 혼합 RV 는 "CDF 기울기 = 연속 밀도 c, CDF 점프 = 이산 질량 a_k" 로 쪼개고, EX 는 ∫x·c dx 와 Σ x_k·a_k 를 더해 구한다.
