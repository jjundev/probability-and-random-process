## 문제
X 와 Y 는 독립인 N(0, 1) 확률변수이고
Z = 11 − X + X²Y,
W = 3 − Y.
Cov(Z, W) 를 구하라.

## 1. 문제 정리
- 주어진 것: X, Y ~ N(0,1) 독립. Z = 11 − X + X²Y, W = 3 − Y.
- 구할 것: Cov(Z, W).
- 필요한 모멘트: E[X] = E[Y] = 0, Var = 1 ⇒ E[X²] = E[Y²] = 1 (Lecture06, p.5).

## 2. 무엇을 묻고 왜 이 도구인가
공분산은 정의 Cov(Z,W) = E[ZW] − E[Z]E[W] (Lecture07, p.23) 와 **이중선형성**(Lecture07, p.25): 상수는 무시되고 Cov(aX+b, Y) = a·Cov(X,Y), Cov(X, Y+U) = Cov(X,Y) + Cov(X,U). 독립이면 Cov = 0 (Lecture07, p.23). X²Y 같은 비선형 항이 끼므로 독립으로 떨어지는 항과 남는 항을 가른다.

## 3. 핵심 통찰
상수 11, 3 은 공분산에서 즉시 사라지고, Cov(−X, −Y) 는 독립이라 0. 살아남는 단 하나의 항은 Cov(X²Y, −Y) 인데, 이는 E[X²Y²] = E[X²]E[Y²] = 1 때문에 −1 이 된다. X²Y 는 X 와 안 엮이는 듯 보여도 Y 와는 Y² 를 통해 엮인다.

## 4. 풀이

### 방법 A — 이중선형성으로 항 분해
상수 11, 3 제거 (Cov(상수, ·) = 0):
Cov(Z, W) = Cov(−X + X²Y, −Y)
  = Cov(−X, −Y) + Cov(X²Y, −Y).

항 1: Cov(−X, −Y) = (−1)(−1)Cov(X, Y) = Cov(X, Y) = 0  (X ⊥ Y).

항 2: Cov(X²Y, −Y) = (−1)Cov(X²Y, Y) = −[E[X²Y·Y] − E[X²Y]·E[Y]].
- E[X²Y²] = E[X²]E[Y²] = 1·1 = 1  (독립 ⇒ E[g(X)h(Y)] = E[g(X)]E[h(Y)], Lecture04 slide 21).
- E[X²Y] = E[X²]E[Y] = 1·0 = 0;  E[Y] = 0.
- ⇒ Cov(X²Y, Y) = 1 − 0·0 = 1, 따라서 항 2 = −1.

Cov(Z, W) = 0 + (−1) = −1.

### 방법 B — 정의로 직접 검산 (Cov = E[ZW] − E[Z]E[W])
E[Z] = 11 − E[X] + E[X²]E[Y] = 11 − 0 + 1·0 = 11.
E[W] = 3 − E[Y] = 3.
ZW 전개: (11 − X + X²Y)(3 − Y) = 33 − 11Y − 3X + XY + 3X²Y − X²Y².
E[ZW] = 33 − 11·0 − 3·0 + E[X]E[Y] + 3E[X²]E[Y] − E[X²]E[Y²]
  = 33 + 0 + 0 + 0 − 1 = 32.
Cov(Z, W) = 32 − (11)(3) = 32 − 33 = −1.  (방법 A 와 일치 ✓)

## 5. 검산·직관
- 두 방법이 −1 로 일치 ✓.
- 부호 해석: W = 3 − Y 는 Y 에 감소하고, Z 안의 X²Y 항은 Y 에 증가(평균적으로 X² > 0) → Z 와 W 가 반대로 움직여 음의 공분산이 나오는 것이 직관적.
- 홀수차 모멘트 0 활용: N(0,1) 은 대칭이라 E[X] = E[Y] = E[X²Y] = 0 등 홀수 결합항이 다 죽고, 살아남는 건 짝수 모멘트 E[X²Y²] = 1 뿐.

## 6. 한 줄 요약
> 공분산은 상수·독립항을 즉시 버리는 이중선형 연산이며, 비선형 항 X²Y 도 Y² 를 통해 Y 와 엮여 Cov(Z,W) = −1 을 남긴다.
