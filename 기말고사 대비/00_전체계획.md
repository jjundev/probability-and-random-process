# 확률·랜덤프로세스 기말 대비 — 10일 복습계획 (증량판, 6/7~6/16)

> **시험범위**: L01~09 (중간범위 포함). 신규 L05~09 중점 + 중간범위 인터리브.
> ⚠️ **시험일 확인**: 6/16 직후 가정. 6/16이 시험일이면 D10 모의를 6/15로 당기고 6/16은 오답·치트시트만.

## 이 계획의 핵심 = "훑기 금지, 복습(반복) 우선"
**3패스 구조**로 같은 주제를 최소 3회 접촉한다.
- **1차(D1~D5)**: 강의별 학습 — 하루 신규 8~10 + 어제 오답.
- **2차(D6~D8)**: 인터리브 복습 — 주제 섞어 풀며 "공식 고르는" 연습 + 누적 오답.
- **실전(D9~D10)**: 모의고사 + 오답 분석.

## 절대 규칙 (문제량을 늘려도 복습이 1순위)
1. **오답 먼저**: 매일 오답 재풀이 블록을 **맨 먼저** 한다. 절대 생략 금지.
2. **신규는 상한(ceiling)**: 그날 신규 목록은 "최대치". 시간 부족하면 신규를 다음날로 이월하되 **오답은 미룬다 ❌**.
3. **2회 연속 정답 = 클리어**: 한 번 맞아도 한 번 더 맞혀야 끝. (→ `오답노트.md`)
4. **≥3회 접촉 유지**: 학습 → 오답 → 인터리브 → 모의.

## 하루 시간 예산 (권장)
`오답 재풀이 30~40분` → `신규 문제(상한) 90~120분` → `채점·오답등록 20분`.
모의일(D9·D10)은 `풀이 90분` → (휴식) → `오답 분석·재풀이 별도`.

## 오답 루프
틀린 문제 = `오답노트.md`에 등록(번호·강의·틀린 이유) → 다음 복습 블록에서 재풀이.
1회 정답=노랑, **2회 연속 정답=클리어(삭제)**, 또 틀림=빨강. 모의 전 **빨강 0개**가 목표.
